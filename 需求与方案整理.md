# 需求与方案整理

## 战斗表现

| 需求 | 方案 |
|---|---|
| 角色3C（locomotion）资源 | 接入动画 Clip |
| 死亡 | Timeline 配合相机表现，支持表达分步/阶段表现 |
| 角色3C locomotion 动画 | 检查相关 RootMotion、地面材质、标记角色行为（移动/奔跑/下蹲），每套配置需求待确认 |
| 角色3C 受击效果 | 合并在 locomotion 表内，不区分受击，只检测材质 |
| 角色3C 演出效果 | 预制标准角色放在 3C 的 locomotion 中；预制标准骨骼接入 AnimClip |
| 脚步/扫尘/踏水/冰冻 | 欧卡已配置，相关方案待确认 |
| 流水水花 | 在流水区域打 Tag，Tag 跟随触发 Timeline |
| 喘息 | 在体力栏等位置打 Tag，Tag 跟随触发 Timeline；之后再配动作 |
| 武器冲击音效 | 提取攻击音效，索引武器材质、攻击类型，在 Timeline 内配置 |
| 武器命中特效 | 扩展命中特效索引：目标类型、攻击材质、命中部位、命中材质；伴随表现：命中特效、命中音效、持续命中特效、持续环境影响作用 |
| 击杀音效（HUD） | 配战斗感；旁边一下仅对生效，待确认 |
| 特技配音音效 | 独立增加一套表，检索受击 ID；使用玩家交互时的怪物/招式命中音效，内置同步 |

## 武器表现

| 需求 | 方案 |
|---|---|
| 武器模型 | 特效跟随走动画，音效统一跟随走 |
| 武器特效挂点数据 | 配置在相应的 Prefab 上 |

## 异常状态

| 需求 | 方案 |
|---|---|
| 异常状态音效 | 配置任意 Timeline |
