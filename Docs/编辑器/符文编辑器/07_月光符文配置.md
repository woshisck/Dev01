# 月光符文配置指南（含连携效果）

---

## 一、定位

月光符文是**主动战斗卡**，打出时发射投射物。  
连携系统支持两种正向连携（月焰/蚀月）和两种反向连携（月照/月毒）。

| 触发方式 | `CombatCard`（战斗卡主动使用） |
| 流程资产 | `FA_Rune_Moonlight` |

## 五、流程图配置

月光共涉及 **4 个 FlowAsset**，均放置在 `/Game/YogRuneEditor/Flows/`：

### FA 1：FA_Rune_Moonlight（主流程）

```
Entry ──→ 远程弹幕(SpawnRangedProjectiles)
              bPreferCombatCardProjectileClass = ✅
              bPreferCombatCardDamageEffectClass = ✅
              bUseCombatCardAttackDamage = ✅
              bShareAttackInstanceGuid = ✅
              bRequireRangedWeaponTag = ✅
              ProjectileCount = 1
              HitGameplayEventTag = Event.Moonlight.Hit
              bSendHitGameplayEventToSourceASC = ✅
              bUseDamageAsHitGameplayEventMagnitude = ✅
       └──→ 等待事件(WaitGameplayEvent)
                EventTag = Event.Moonlight.Hit
                Target   = BuffOwner
                Out → 播放特效(命中 VFX, Target=LastDamageTarget)
                        Out → 结束符文
```

> `远程弹幕` 的 Out 引脚**不连接**；FA 结束时机由 `等待事件 → 结束符文` 控制。

### FA 2：FA_Rune_MoonlightBonus（备用公共多发，无额外效果）

```
Entry → 远程弹幕(ProjectileCount=2, ConeAngle=20, bRequireRangedWeaponTag=❌)
       → 结束符文
```

### FA 3：FA_Rune_Moonlight_FwdBurn（月焰连携 LinkFlow）

```
Entry ──→ 远程弹幕(ProjectileCount=2, ConeAngle=15, bRequireRangedWeaponTag=❌)
              Out → 结束符文(C)
       └──→ 播放特效(月焰连携激活 VFX)  [Out 不连]
```

### FA 4：FA_Rune_Moonlight_FwdPoison（蚀月连携 LinkFlow）

```
Entry ──→ 远程弹幕(ProjectileCount=2, HitGameplayEventTag=Event.Moonlight.FwdPoison.Hit)
              Out [不连]
       └──→ 等待事件(Event.Moonlight.FwdPoison.Hit)
                  Out → 施加状态(GE_Moonlight_Poison, Target=LastDamageTarget)
                          Out → 结束符文
```

## 六、连携配方配置

在中间面板 **连携配方** Tab 填写，填完后点击 **保存连携配方**：

| # | 邻居 Tag | 匹配方式 | 正向 Flow | 正向倍率 | 正向说明 | 反向 Flow | 反向倍率 | 反向说明 |
|---|---|---|---|---|---|---|---|---|
| 0 | `RuneTag.Fire` | 效果Tag | `FA_Rune_Moonlight_FwdBurn` | **1.5** | 月焰 | 空 | **1.3** | 月照 |
| 1 | `RuneTag.Poison` | 效果Tag | `FA_Rune_Moonlight_FwdPoison` | **1.0** | 蚀月 | 空 | **1.4** | 月毒 |

> - 正向（Fire/Poison 在前 → 月光）：倍率 × LinkFlow（多发 + 特殊效果）  
> - 反向（月光在前）：只有倍率，无 LinkFlow，**不触发多发**  
> - 点击行末的 **→ 按钮**可在流程图面板直接打开对应的连携 FA

---

## 七、连携激活条件

连携系统读取 `State.Combo.ChainActive` Tag 判断连击链是否激活：
- 连击链激活时该 Tag 自动挂载（由 `ComboRuntimeComponent` 管理）
- 冲刺后续连时 Tag 保留（`DashSavedLinkContext` 机制）
- 不在连击链内打出月光卡：不触发连携，只有基础效果

## 九、注意事项

- 月光主 FA 的 `bRequireRangedWeaponTag = ✅`，连携 FA 里必须关掉（= ❌），否则连携 LinkFlow 会因无武器 Tag 而走 Failed 引脚
- 蚀月 LinkFlow 中 `WaitGameplayEvent` 会在每枚投射物命中时触发一次（共 2 次），如需限制只附加一次中毒可在 GE 层用 `Stack Limit = 1` 或包裹 `只执行一次` 条件节点
- 连携倍率（1.5/1.0/1.3/1.4）作用在**伤害层**，由战斗卡系统自动应用，FA 内不需要手动乘算
