# 角色平衡编辑器（Character Balance）

菜单路径：`DevKit Data → Character Balance`

---

## 一、定位

角色平衡编辑器以**表格形式批量浏览和修改**所有 `UCharacterData` 资产的数值，适合：
- 横向对比多个角色的基础属性（血量/攻击/防御）
- 快速调整单个参数后立即看效果
- 导出 CSV 给策划做版本间数值对比

---

## 二、界面区域

```
┌──────────────────┬──────────────────────────────┐
│  角色列表（左侧）  │  属性表格（右侧）                │
│                  │  基础属性表 / 移动属性表           │
│  [双击] 打开DA   │  [直接点击单元格] 修改浮点值        │
└──────────────────┴──────────────────────────────┘
```

---

## 三、支持编辑的属性分组

### 基础属性表（BaseAttributes）

| 字段 | 说明 |
|---|---|
| MaxHealth | 最大生命值 |
| BaseAttack | 基础攻击力 |
| BaseDefense | 基础防御力 |
| CritRate | 暴击率 |
| CritDamage | 暴击伤害倍率 |

### 移动属性表（MovementAttributes）

| 字段 | 说明 |
|---|---|
| WalkSpeed | 行走速度 |
| DashDistance | 冲刺距离 |
| DashCooldown | 冲刺冷却 |

---

## 四、使用流程

```
1. 打开 DevKit Data → Character Balance
2. 左侧列表显示所有 CharacterData 资产
3. 点击某行 → 右侧表格显示该角色的属性
4. 点击属性单元格 → 直接输入新数值 → 回车确认
5. Ctrl+S 保存
6. 如需批量导出：工具栏 → 导出 CSV
```

---

## 五、验证与导出

- **验证**：检查角色数据完整性（是否有缺失的必填字段）
- **导出 CSV**：包含所有角色的全属性快照，可用 Excel 做版本对比

---

## 六、注意事项

- 直接修改不走 Undo 栈，修改后无法撤销，请谨慎操作
- 如需修改角色的 Flow / GA 配置，请直接打开对应的 CharacterData DA 编辑
- 角色的符文槽位和卡组配置不在此编辑器中，在各自的 DA 里管理
