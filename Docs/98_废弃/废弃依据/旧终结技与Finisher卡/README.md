# 旧终结技与 Finisher 卡废弃说明

> 状态：废弃。旧终结技/Finisher 卡方案已被当前“攻击出牌、满足条件后系统自动爆发”的战斗方向替代，不应作为实现或配置依据。

## 废弃原因

当前战斗方向已经调整为：

- 玩家攻击不再区分轻重攻击。
- 玩家不再维护主动攻击终结技。
- 卡牌通过攻击顺序依次触发。
- 卡牌反应满足条件后，由系统自动打出爆发。
- 战技可承担引爆逻辑，但不再作为旧 Finisher 卡/QTE 体系。

因此，旧文档中关于 `Finisher` 卡、终结技 QTE、连招终结击触发终结技、终结技专用教程弹窗、终结技卡锁定等内容都不应再作为当前依据。

## 已迁入本文档夹的旧依据

- 旧 Combat Finisher 技术文档。
- 旧 Rune Editor 终结技配置说明。
- 旧 UI 终结技教程弹窗配置说明。
- 旧终结技卡牌设计案。
- 旧“连招与终结击”系统需求。

如果后续需要“自动爆发”设计，应在 `01_策划需求与版本方案/` 和 `03_策划配置与制作手册/` 中重新建立当前方案文档，不要复用这里的旧字段和旧流程。
