## 一、目标用户画像

### 1.1 核心玩家群体

| 维度 | 描述 |
|---|---|
| 年龄 | 18-32 岁为主 |
| 平台 | PC（Steam）为主，主机为辅 |
| 玩龄 | 有 Roguelite / 动作游戏经验，不完全是硬核玩家 |
| 游玩习惯 | 单次游玩 30-90 分钟；喜欢局内构筑带来的"当局最强"感 |
| 参考画像 | 《黑帝斯》通关后想要更多构筑深度的玩家；《赛博朋克 2077》+ 《哈迪斯》双开的玩家 |

### 1.2 三类核心用户

| 用户类型 | 核心需求 | 星骸降临对应点 |
|---|---|---|
| **构筑探索者** | 每局尝试不同 Build，找到"破局点"带来的满足感 | 热度阶段 × 符文 Flow 组合带来大量 Build 路线 |
| **动作爱好者** | 流畅的攻击手感、连击反馈、敌人死亡演出 | GAS 底层 + 连击系统 + 死亡消解特效 |
| **悬念驱动者** | "再玩一局"的牵引力，每局都有遗憾和好奇 | 随机关卡骰子 + 分支选关 + 未解锁符文 |

## 二、玩家声音调研（参考相似游戏评论分析）

> 以下基于《黑帝斯》、《Vampire Survivors》、《Risk of Rain 2》、《死亡细胞》等类似 Roguelite 游戏的 Steam 评论、Reddit/Discord 讨论提炼的玩家核心诉求。

### 2.1 玩家最看重的正向体验

| 排名 | 玩家诉求 | 代表评论风格 |
|---|---|---|
| 1 | **手感爽快** | "每一刀打下去都有反馈，敌人的受击动画很到位" |
| 2 | **Build 多样性** | "我玩了 200 小时，每局感觉都不一样" |
| 3 | **死亡不懊悔** | "失败了也觉得是我的决策问题，不是游戏不公平" |
| 4 | **前期引导足够** | "10 分钟内知道自己在做什么" |
| 5 | **进度奖励感** | "每局都能解锁点什么新东西" |

### 2.2 玩家最常抱怨的负向体验

| 排名 | 玩家抱怨 | 对星骸降临的启示 |
|---|---|---|
| 1 | **早期难度过高** | 确保前 3 关有足够容错（结算回血、护盾等）|
| 2 | **Build 同质化** | 确保不同稀有度符文有明显的 Flow 差异，避免"永远同一路线" |
| 3 | **操作不跟手** | 输入缓冲要精准，避免技能卡顿（InputBufferComponent 已有）|
| 4 | **加载时间过长** | UE5 场景切换需要优化，考虑加载中显示 Lore |
| 5 | **背景设定看不懂** | "星骸"世界观需要在游玩中自然引出，避免信息轰炸 |

## 三、细分用户的购买决策路径

### 3.1 Steam 用户从看到到购买的路径

```
Discovery（发现）
  └─ 算法推荐 / 朋友分享 / 直播/视频
       ↓
Interest（兴趣）
  └─ 封面图 / 宣传视频前 30 秒 → 是否有"哇"时刻
       ↓
Consideration（考虑）
  └─ 评论区 / 截图 / EA 价格 → 是否值得"赌一把"
       ↓
Purchase（购买）
  └─ 价格、评价数量（100+ reviews 是重要门槛）
```

**关键漏斗节点：宣传视频前 30 秒**

| 内容 | 目标 |
|---|---|
| 0-10s | 展示最酷的战斗动作 / 爆炸 / 符文激活特效 |
| 10-25s | 展示背包构筑 / 传送门选择 / 不同 Build 视觉差异 |
| 25-30s | 游戏 Title + EA 时间 |

## 四、设计建议（基于玩家调研）

### 4.1 高优先级（影响首周留存）

| 问题 | 设计建议 |
|---|---|
| 新手不知道如何使用热度系统 | 前 2 关加入软引导（敌人密度低，给玩家时间理解升阶逻辑）|
| 符文太多无从下手 | 每局开始只提供 2-3 个预设入门符文组合，后期解锁自由配置 |
| 打击感需要打磨 | 增加命中帧冻结（2-4 帧）+ 受击晃动 + 爆头/暴击音效 |
| 死亡缺少意义感 | 死亡界面展示"本局最佳表现"数据，给玩家成就感和方向感 |

### 4.2 中优先级（影响口碑二次传播）

| 问题 | 设计建议 |
|---|---|
| Build 多样性是否可感知 | UI 上显示"当前激活符文 Flow"，让玩家清晰知道自己当前是什么 Build |
| 连击缺少演出高潮 | 最后一击如果命中多个敌人考虑短暂慢动作（0.2s）强调爽快感 |
| 传送门选择信息量少 | 在传送门选择 UI 中显示房间类型图标 + 推荐词条预告 |

### 4.3 长期（EA 后迭代方向）

| 优先级 | 内容 |
|---|---|
| 高 | 成就/解锁系统（让失败有外在奖励，提升"再玩一局"驱动力）|
| 高 | 每周挑战 / 日常词条（维持长期活跃玩家回访）|
| 中 | 局外元进度可视化地图（让玩家感知自己的成长曲线）|
| 低 | 排行榜 / 跑单挑战（硬核玩家长尾需求）|

## 五、竞品分析速览

| 游戏 | 优势参考 | 规避点 |
|---|---|---|
| 《黑帝斯》| 世界观融入游玩 / 死亡对话不重复 | 高投入的剧本体量我们短期无法复制 |
| 《死亡细胞》| 打击感顶尖 / 武器多样性强 | 精准输入要求对休闲玩家有门槛 |
| 《Risk of Rain 2》| Build 天花板极高 / 物品联动感 | 早期难度陡峭导致流失 |
| 《Brotato》| 前期引导优秀 / 局内变化丰富 | 美术风格简单，无法靠视觉吸量 |

## 六、目标用户规模估算（Steam）

> 以下为参考估算，非精确数据。

| 指标 | 估算 |
|---|---|
| Steam Roguelite 月活跃买家 | 约 2000-4000 万（含各类 Roguelite 子类）|

## 七、玩家反馈收集计划（开发期）

| 阶段 | 行动 |
|---|---|
| Demo 阶段（P1 完成后）| Steam Next Fest 参展 / 向 Roguelite 社区 KOL 发 Key |
