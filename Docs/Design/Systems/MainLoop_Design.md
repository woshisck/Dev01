# 游戏主循环设计文档

> 适用范围：星骸降临 全项目  
> 适用人群：策划 + 程序  
> 配套文档：[关卡系统配置指南](LevelSystem_ConfigGuide.md)、[关卡系统技术文档](LevelSystem_ProgrammerDoc.md)  
> 最后更新：2026-04-10

---

## 一、系统定位

本作为纯单人动作冒险游戏，不含联网或多人要素。

采用类《黑帝斯》的 Roguelite 推进结构：地图以**节点分支**形式呈现，每个节点为独立房间，房间内容从预设关卡池中随机抽取。游戏分为**局内**与**局外**两个层次，玩家在局内推进并获取临时增益，在局外积累永久成长。

---

## 二、核心循环总览

```
选关（分支路径选择）
    ↓
进入房间（关卡加载）
    ↓
战斗（核心玩法）
    ↓
结算
    ├─ 发放资源（金币 / 局外成长资源）
    ├─ 奖励选择（赐福 / 道具 / 强化，三选一）
    └─ 分支选择（下一房间路径选择）
    ↓
进入下一房间（循环）
    ↓    ↓    ↓
普通战斗  精英战斗  特殊房间（商店/休息/事件）
    ↓
中 Boss 战（章节中段）
    ↓
章节 Boss 战（章节终点）
    ↓
进入下一章节
```

---

## 三、局内流程详解

### 3.1 选关（分支路径选择）

战斗结束后，玩家面对地图上 **2-3 个分叉路口**，每条路径预览下一房间信息。

| 预览信息 | 说明 |
|---|---|
| **房间类型** | 普通战斗 / 精英 / 商店 / 祭坛 / 事件 / Boss |
| **难度标记** | 普通 / 精英（精英额外奖励，但敌人更强） |
| **奖励类型** | 本房间可能产出的奖励种类预告 |

**策略意义：**

| 当前状态 | 推荐路线 |
|---|---|
| 血量偏低 | 优先回血 / 商店 / 弱敌路径 |
| 强化需求强 | 选赐福 / 精英路径（高收益高风险）|
| 接近 Boss | 选商店补给 / 休息室 |

> 类比：分支选择就是"地图上同时有 2-3 条路，每条路门口挂着告示牌，告诉你里面大概有什么"。

---

### 3.2 进入房间（关卡加载）

玩家选定路径后，加载并进入新房间。

| 要素 | 说明 |
|---|---|
| **固定地形** | 每个房间地形预设，给玩家确定的环境信息 |
| **固定敌人类型** | 每个预设房间配置固定种类的敌人，新玩家可预判 |
| **随机关卡词条** | 房间随机附带 1-2 个词条，修改本房间战斗规则（如：敌人加速、场内毒雾）|
| **互动元素** | 部分房间含机关陷阱、可破坏物体 |

> 固定敌人 + 随机词条 = 熟悉的挑战框架 + 不确定的变量，帮助新手决策同时维持重玩性。

---

### 3.3 战斗（核心玩法）

实时俯视角战斗，清除房间内所有敌人后战斗阶段结束。

**战斗机制联动：**

```
攻击命中 → 积累热度
    ↓
热度满 + LastHit → 升阶
    ↓
激活区扩展 → 更多符文生效 → 战斗力提升
    ↓
脱战后热度衰减 → 阶段下降（促使玩家保持进攻节奏）
```

**敌人种类：**

| 敌人类型 | 出现位置 | 特点 |
|---|---|---|
| 普通怪 | 所有战斗房 | 基础行为，数量多 |
| 精英怪 | 精英房（Elite Only） | 强化属性或特殊技能 |
| 中 Boss | 章节中段指定节点 | 多阶段攻击 |
| 章节 Boss | 章节终点 | 综合考验本章积累的赐福与操作 |

---

### 3.4 结算（战斗结果）

所有敌人死亡后，进入结算阶段。

```
所有敌人死亡 → CheckLevelComplete()
    ↓
EnterArrangementPhase()
    ├─ 背包解锁（玩家可重排赐福顺序）
    ├─ 发放金币（按当前难度配置的 GoldMin ~ GoldMax 随机）
    ├─ 发放局外成长资源（待设计）
    └─ 生成赐福/奖励选项（最多 3 个，三选一）
```

**结算阶段玩家操作：**

1. 查看并重排背包中的符文（赐福）顺序
2. 从 2-3 个奖励中选择一个
3. 查看下一房间分支信息，选择路径
4. 确认进入下一关

---

### 3.5 奖励选择

每次结算提供 **2-3 个互斥选项**，玩家只能选其一。

| 奖励类型 | 说明 | 局内/局外 |
|---|---|---|
| **赐福（符文）** | 新增符文，放入背包后根据激活区生效 | 局内 |
| **赐福升级** | 强化已有符文的效果 | 局内 |
| **消耗道具** | 局内可使用的一次性道具 | 局内 |
| **金币** | 直接获得额外金币 | 局内临时 |
| **局外成长资源** | 本局结束后保留，用于局外解锁 | 局外 |
| **武器强化** | 提升武器属性或添加新效果 | 局内 |

**奖励来源：**

| 来源 | 产出类型 |
|---|---|
| 普通战斗房结算 | 金币 + 赐福三选一 |
| 精英房结算 | 金币更多 + 额外稀有奖励 |
| 宝箱 / 可破坏物 | 道具 / 金币 |
| 特定房间固定产出 | 根据房间类型决定 |

---

### 3.6 分支选择（奖励后）

完成奖励选择后，玩家再次面对路径选择。此时可见各分支的房间类型、词条预览和奖励类型预告，玩家根据当前状态决策下一步路线。

---

## 四、房间类型设计

| 房间类型 | 主要内容 | 奖励 | 风险 |
|---|---|---|---|
| **普通战斗** | 2-3 波普通怪 | 金币 + 赐福三选一 | 低 |
| **精英战斗** | 精英专属强化敌人 | 金币更多 + 额外稀有赐福 | 高 |
| **商店** | 用金币购买道具 / 赐福 | 按需选购 | 无战斗 |
| **祭坛** | 消耗资源换取特定强化（可能有代价）| 高倍率收益 | 资源风险 |
| **事件** | 随机特殊选择题事件 | 不确定 | 不确定 |
| **休息** | 恢复 HP（具体量待定）| 无其他奖励 | 无战斗 |
| **中 Boss** | 章节中段强敌，多阶段 | 高额奖励 | 极高 |
| **章节 Boss** | 章节终点，通过即进入下章 | 进入下一章 | 极高 |

## 五、章节结构

### 5.1 章节概览

| 章节 | 场景名 | 主题 | 敌人特点 | 预计时长 |
|---|---|---|---|---|
| 第 1 章 | 维尔斯卡牢狱 | 教学引导 | 低伤害近战 + 简单射击怪 | 40-60 min |
| 第 2 章 | 凯莱斯蒂斯城 | 世界观铺开 | 投射怪 / 急攻怪 / 战术敌人 | 40-60 min |
| 第 3 章 | 白昼教堂 | 接近真相 | 复杂精英 + 特殊关卡规则机制 | 40-60 min |
| 最终章 | 归一之地 | 最终决战 | 最终 Boss，无分支选择 | 20-30 min |

**总流程约 3.5 小时，首通预计约 10 小时。**

---

### 5.2 章节节奏规则

```
章节开始
    ↓
普通战斗房 × 3-4（低压，积累赐福）
    ↓
商店 / 休息 / 事件（资源补给）
    ↓
精英房（节奏提升）
    ↓
中 Boss 战（章节节奏高潮，检验阶段成长）
    ↓
普通战斗房 × 2-3 + 商店（末段补给）
    ↓
章节 Boss 战（综合考验）
    ↓
进入下一章节
```

---

### 5.3 各章重点设计

**第 1 章 — 维尔斯卡牢狱（教学期）**
- 多普通战斗房，少商店，章末弱 Boss
- 介绍核心操作、赐福选择、热度系统

**第 2 章 — 凯莱斯蒂斯城（难度升级）**
- 引入新敌人类型（远程投射、加速急攻）
- 引入强化道具机制、队长敌人（队长死亡使附近小怪溃散）

**第 3 章 — 白昼教堂（规则扭曲）**
- 引入特殊关卡规则词条（玩家和敌人都受规则约束）
- 大型中 Boss（多阶段攻击模式）

**最终章 — 归一之地**
- Boss 前设一个小型补给 / 商店房
- Boss 战综合考验全程获得的赐福与操作
- 通关后解锁**深渊模式**

## 六、局外部分

### 6.1 局外成长

| 内容 | 说明 |
|---|---|
| **局外成长资源** | 局内战斗掉落，本局结束后保留 |
| **解锁内容** | 新角色 / 武器 / 初始加成 / 新符文池 |
| **持久化** | 局外资源独立于局内存档，永久保留 |

### 6.2 深渊模式

通关第 3 章后解锁。提供更高难度的刷取循环，供玩家刷取角色词条。

### 6.3 多结局

玩家通过解锁所有重要支线后迎来最终结局选择。初期关卡含隐藏支线道路，重复游玩可逐步发现并解锁。

## 七、节奏目标

| 指标 | 数值 |
|---|---|
| 单房间游玩时长 | 1-3 分钟 |
| 每章房间数 | 15-20 个 |
| 单章时长目标 | 40-60 分钟 |
| 设计总流程时长 | 约 3.5 小时 |
| 首通预计时长 | 约 10 小时 |
| 重复游玩目标 | 解锁所有支线后的多结局 |
