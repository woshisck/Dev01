# 油瓶 / 雷石 / 烟雾配置说明

## 配置入口

- 玩家组件：`PlayerCharacterBase.CombatItemComponent`
- 道具资产：创建 `CombatItemDataAsset`，把资产填入 `CombatItemComponent.DefaultItemAssets`
- HUD：系统会自动创建 3 格道具栏；图标来自 `FCombatItemConfig.Icon`
- 输入：未配置 Enhanced Input 时，默认 `F` 使用当前道具，`Q` 切换下一个道具
- 手柄默认：肩键/扳机键使用当前道具，十字键右切下一个，十字键左切上一个

Enhanced Input 可选字段：
- `Input_UseCombatItem`：建议绑定肩键或扳机键
- `Input_SwitchCombatItem`：建议绑定十字键右
- `Input_SwitchCombatItemPrevious`：建议绑定十字键左

如果 `DefaultItemAssets` 为空，并且 `bUseBuiltInDefaultItemsWhenEmpty=true`，系统会自动装配油瓶、雷石、烟雾各 2 次。

## 通用字段

- `ItemId`：道具 ID，例如 `OilBottle`、`ThunderStone`、`SmokeBomb`
- `DisplayName`：HUD 显示名
- `Icon`：HUD 图标
- `EffectType`：选择 `OilBottle`、`ThunderStone` 或 `SmokeBomb`
- `MaxCharges`：最大次数，V1 默认 2
- `InitialCharges`：初始次数，V1 默认 2
- `Cooldown`：使用后的冷却时间
- `NiagaraSystem`：范围类道具表现，可留空
- `DecalMaterial`：范围提示贴花，可留空

## 油瓶

推荐默认：
- `Cooldown=12`
- `Oil.OilBladeDuration=8`
- `Oil.StickyDuration=5`
- `Oil.MaxStickyStacks=10`
- `Oil.MoveSpeedSlowPerStack=0.03`
- `Oil.TurnSpeedSlowPerStack=0.04`
- `Oil.AttackSpeedSlowPerStack=0.02`
- `Oil.FireBonusDamagePerStack=3`
- `Oil.FireBonusDamageCap=30`

逻辑：使用后玩家获得油刃，近战命中敌人叠加 `StickyOil`。敌人每层降低移动、转身、攻击速度。玩家造成火焰/燃烧伤害时，对带油敌人追加火焰伤害。

## 雷石

推荐默认：
- `Cooldown=16`
- `Thunder.Duration=5`
- `Thunder.Radius=450`
- `Thunder.TickInterval=1`
- `Thunder.DamagePerTick=15`
- `Thunder.bInterruptMinions=true`

逻辑：使用后生成跟随玩家的雷场。范围内敌人每 Tick 受伤；小怪会收到受击打断，霸体、死亡、较高难度敌人不被打断。

## 烟雾

推荐默认：
- `Cooldown=18`
- `Smoke.Duration=6`
- `Smoke.InitialRadius=400`
- `Smoke.FinalRadius=600`
- `Smoke.TickInterval=0.1`
- `Smoke.DodgeBonus=0.5`

逻辑：使用后在当前位置生成烟雾，半径从 400 扩散到 600。玩家在烟雾中获得 `InSmoke` 和 50% 闪避；敌人攻击选择检测到目标有 `InSmoke` 时不能发动攻击，但仍会移动和追踪。

## 注意

- 策划不需要配置 GAS BlockTag、攻击临时 Tag 或连招 Tag。
- 道具不消耗符文卡，不触发连招倍率，不阻塞攻击和冲刺。
- 油瓶只强化玩家近战命中，不强化远程子弹。
- 雷石和烟雾共用 `CombatItemAreaActor`，表现资源可以后续替换。

## 验收

- 3 格道具栏显示图标、次数、冷却和当前选中槽。
- 油瓶命中叠层最多 10 层，火焰伤害能触发追加伤害。
- 雷石每秒造成 15 点伤害，并只打断小怪。
- 烟雾内敌人不能启动攻击，玩家离开或烟雾结束后效果移除。
