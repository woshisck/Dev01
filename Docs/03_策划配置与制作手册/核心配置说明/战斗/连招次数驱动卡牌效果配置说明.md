# 旧连招次数驱动卡牌效果说明

## 当前状态

连招次数驱动卡牌倍率已经废弃。当前玩家战斗不再依赖连招段数，卡牌效果不再从 `ComboIndex` 获得倍率或额外投射物。

当前构筑强度来源：

- Attack 卡组顺序。
- `RequiredActionSlot` 区分 Attack / Skill / WeaponSkill / Dash。
- `RequiredFlowRole` 区分 Starter / Catalyst / Finisher。
- Link Recipe 读取前后卡、CardIdTag、EffectTags 和 FlowRole。
- Passive Flow 提供装备后持续生效的被动构筑。

## 旧字段兼容

| 旧字段 | 当前行为 |
| --- | --- |
| `bUseComboEffectScaling` | 运行时忽略，效果倍率仍为 1.0 |
| `ComboScalarPerIndex` | 运行时忽略 |
| `MaxComboScalar` | 运行时忽略 |
| `Add Combo Stacks To Projectile Count` | `ComboBonusStacks` 固定为 0，不会增加投射物 |
| `Projectiles Per Combo Stack` | 运行时忽略 |
| `Max Bonus Projectiles` | 运行时忽略 |

旧字段保留是为了让旧资产能加载；新卡牌和新 Flow 不要继续配置这些字段。

## 替代配置

| 目标 | 当前做法 |
| --- | --- |
| 第 2、3 张牌更强 | 用 Link Recipe 或根据前后卡 EffectTags 触发 Matched/Link Flow |
| 某张牌只在引爆时更强 | 配 `RequiredActionSlot = WeaponSkill` 或 `RequiredFlowRole = Finisher` |
| 月光多段变化 | 直接在对应 Link Flow 中配置固定投射物数量、分裂、弹跳或范围 |
| 攻击力逐步累积 | Starter 上 Buff，Catalyst 提升 Buff，Finisher 引爆 |

## 验收

1. 开启旧 Combo Scaling 字段的资产，运行时倍率仍为 1.0。
2. Attack 卡按照卡组顺序结算并触发 Link。
3. WeaponSkill 卡作为 Finisher / 引爆逻辑结算。
4. 月光等投射物变化由具体 Flow 配置决定，不读取连招段数。
