# 冲刺空气墙配置说明

## 用途

用于配置玩家冲刺时哪些空气墙可以被穿越，哪些空气墙必须阻挡。当前规则只认 `AAirWall`，不要用普通 `DashThrough` 物体替代空气墙。

## 配置位置

- 关卡中放置 `AAirWall`。
- 在空气墙 Details 面板配置 `bAllowDashThrough`。
- 冲刺距离仍由玩家 Dash GA 的 `DashMaxDistance` 决定，默认 600。

## 字段含义

| 字段 | 说明 |
| --- | --- |
| `bAllowDashThrough=false` | 实心空气墙，冲刺会停在墙前。 |
| `bAllowDashThrough=true` | 允许进入冲刺穿越判定。不是无条件穿越。 |
| `DashMaxDistance` | 本次冲刺最大距离，也是穿越后落点硬上限。 |
| `DashBarrier` Tag | 旧阻挡标记，带该 Tag 的物体始终阻挡冲刺。 |

## 穿越规则

1. 空气墙距离玩家超过 `DashMaxDistance * 2 / 3`：停在墙前。
2. 空气墙距离玩家小于等于 `DashMaxDistance * 2 / 3`，且出口仍在本次 `DashMaxDistance` 内：允许穿越。
3. 空气墙太厚，出口超过 `DashMaxDistance`：停在墙前。
4. 穿越成功时，玩家最终最多落在 `DashMaxDistance` 的原始冲刺落点，不会额外延伸。

## 不需要配置

- 不需要让策划手动配置玩家 Capsule 对 `DashThrough` 的 Overlap。
- 不需要为了敌人配置空气墙规则；敌人和 Pawn 不参与空气墙距离/厚度判定。
- 不需要在敌人上配置特殊 DashTrace 来防止截断；代码会过滤 Pawn/Enemy。

## 验收建议

- 600 距离冲刺时，空气墙在 450 位置应停在墙前。
- 600 距离冲刺时，空气墙在 300 位置且厚度较薄，应穿过并落在 600 位置。
- 600 距离冲刺时，空气墙在 300 位置但厚度超过出口范围，应停在墙前。
- 冲刺穿过敌人后，结束时不应和敌人碰撞体重叠卡住。
