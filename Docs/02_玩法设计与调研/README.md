## 使用规则

这里回答“为什么这样设计”。已经进入开发执行的方案，应提炼到 `01_策划需求与版本方案/`；已经实现并长期有效的系统规则，应沉淀到 `04_开发实现与系统文档/`；需要策划/TA 在编辑器执行的步骤应进入 `03_策划配置与制作手册/`。
