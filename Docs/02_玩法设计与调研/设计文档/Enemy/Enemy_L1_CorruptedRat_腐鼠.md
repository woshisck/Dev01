## 2. 敌人需求定位

### 2.1 核心定位

腐鼠是由鼠巢和鼠刑相关敌人产生的低血召唤物，主要用于制造短暂干扰、放大范围卡收益，并让玩家意识到召唤源必须优先处理。

### 2.2 给玩家制造的战斗难题

| 维度 | 内容 |
| --- | --- |
| 主要压力 | 数量、快速贴身、打断移动节奏 |
| 玩家应对方式 | 范围攻击、轻击清理、不要被召唤源拖住 |
| 服务的卡组体验 | 溅射、燃烧、毒、击退、清场连锁 |
| 失败惩罚 | 被持续骚扰，闪避和攻击节奏被打乱 |
| 成功反馈 | 一次范围效果清掉多只腐鼠 |
| 新手教学价值 | 教会玩家处理召唤物和召唤源 |
| 熟练玩家价值 | 作为范围构筑的资源，但不应单独构成难题 |

## 3. 剧情描述与形象设定

### 3.1 背景描述

腐鼠原本只是监狱底层随处可见的老鼠。血肉诅咒爆发后，它们啃食了囚犯尸体、炼金废液和地下暗河带来的异质组织，逐渐变成被血肉驱动的小型污染体。

它们没有复杂意识，只会被铃声、血味和鼠巢的脉动吸引。它们的存在不是单独的灾难，而是监狱腐败程度的最小单位。

### 3.2 当前状态

| 精神状态 | 本能、饥饿、群体躁动 |
| 与监狱的关系 | 底层监狱腐败生态的一部分 |
| 叙事关键词 | 腐烂、啃食、鼠群、血味、召唤物 |
| 可投放环境叙事 | 鼠洞、粪污、啃咬痕、破碎骨头 |

## 4. 基础行为设计

### 4.1 行为树概览

```text
被召唤生成
-> 迅速寻找玩家
-> 成群侧绕
-> 近身扑咬
-> 攻击后短暂停顿
-> 死亡
```

行为逻辑说明：

| 行为 | 内容 |
| --- | --- |
| 巡逻 | 无，通常从召唤源生成 |
| 发现玩家 | 生成后直接锁定玩家 |
| 接近方式 | SwarmFlank，轻微侧绕 |
| 攻击节奏 | 快速低伤害 |
| 后摇设计 | 攻击后短暂停顿，方便玩家清理 |
| 群体定位 | 召唤物，不应拖长房间结算 |
