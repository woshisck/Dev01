## 2. 敌人需求定位

### 2.1 核心定位

哨铃狱卒是低直接伤害但高战术威胁的辅助敌人，用警报、增援和短时强化迫使玩家改变击杀顺序，优先处理召唤或警报源。

### 2.2 给玩家制造的战斗难题

| 维度 | 内容 |
| --- | --- |
| 主要压力 | 警报召唤、敌人强化、打乱优先级 |
| 玩家应对方式 | 优先击杀、打断敲铃、控制其走位 |
| 服务的卡组体验 | 突进切后排、打断、爆发、清召唤物 |
| 失败惩罚 | 场上腐鼠或杂兵增多，战斗节奏被拖慢 |
| 成功反馈 | 打断敲铃后获得清晰喘息窗口 |
| 新手教学价值 | 教会玩家识别辅助敌人和优先击杀目标 |
| 熟练玩家价值 | 和前排敌人搭配时制造路线与目标选择压力 |

### 2.3 关卡节奏定位

| 字段 | 内容 |
| --- | --- |
| 普通房作用 | 少量增加变化，避免房间只由近战敌人组成 |
| 精英房作用 | 作为压力放大器，配合精英制造召唤物干扰 |
| 是否适合作为首见敌人 | 是，但首见应单只搭配少量弱敌 |
| 是否适合作为压轴敌人 | 否 |
| 是否适合重复大量出现 | 否 |
| 推荐首见方式 | 玩家看到其举铃并听到警报声，再出现增援 |

## 3. 剧情描述与形象设定

### 3.1 背景描述

哨铃狱卒原本负责夜巡、点名和封锁牢区。他们不是最凶残的看守，却是监狱秩序真正运作的节点。每一次铃响都意味着门闩落下、通道封闭、某个囚犯被拖走。

血肉诅咒爆发后，哨铃与它的喉咙、肋骨和手臂长在一起。它不再需要说话，只要摇响铜铃，藏在墙洞、稻草和尸堆里的腐鼠与失控囚犯就会被声音驱赶出来。

### 3.2 当前状态

| 字段 | 内容 |
| --- | --- |
| 生前身份 | 正式狱卒 / 巡逻执勤者 |
| 监狱身份 | 警报与封锁执行者 |
| 腐化来源 | 血肉诅咒侵蚀铜铃与喉管 |
| 精神状态 | 执法执念、恐慌、机械性重复职责 |
| 与监狱的关系 | 监狱警戒系统的人形残留 |
| 叙事关键词 | 铜铃、点名、封锁、警报、秩序残响 |
| 可投放环境叙事 | 墙上巡逻时刻表、断裂铜铃、被铃声吸引来的鼠群痕迹 |

## 4. 基础行为设计

### 4.1 行为树概览

```text
巡逻
-> 发现玩家
-> 后撤并尝试敲响警铃
-> 若敲铃完成，召唤少量腐鼠或唤醒附近敌人
-> 保持中距离移动
-> 若被近身，使用短促推击或恐慌后撤
-> 冷却结束后再次尝试敲铃
-> 死亡
```

行为逻辑说明：

| 行为 | 内容 |
| --- | --- |
| 巡逻 | 沿牢房门、闸门或警报点移动 |
| 发现玩家 | 优先后撤并举铃 |
| 接近方式 | 不主动贴身，保持中距离 |
| 攻击节奏 | 低伤害，主要威胁来自敲铃 |
| 后摇设计 | 敲铃前摇明显，可被打断 |
| 群体定位 | 辅助敌人，必须依赖其他敌人形成压力 |

### 4.2 攻击招式表

| 招式名 | 类型 | 距离 | 前摇 | 冷却 | 伤害 | 效果 | 玩家应对 | 备注 |

| Skill1_敲响警铃 | 技能 | 中 | 长 | 长 | 无 | 召唤腐鼠或激活附近敌人 | 快速打断 | 核心技能 |

### 4.3 特殊机制

| 机制 | 内容 |
| --- | --- |
| 召唤机制 | 敲铃完成后召唤 2-3 只腐鼠；同屏腐鼠达到上限时不再召唤 |
| 死亡触发 | 死亡后铃声哑掉，停止后续召唤；不产生爆炸 |
| 阶段变化 | 无 |
| 环境交互 | 可与牢门、鼠洞、尸堆等召唤点绑定 |
| 可打断规则 | 敲铃前摇可被普通强攻击、击退、破甲、终结技打断 |
| 霸体规则 | 无霸体 |
| 抗性规则 | 低生命、低抗性，鼓励优先击杀 |
