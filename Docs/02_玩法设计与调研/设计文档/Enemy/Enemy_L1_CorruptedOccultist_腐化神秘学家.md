## 2. 敌人需求定位

### 2.1 核心定位

腐化神秘学家是 L1 的远程精英敌人，用禁行区域、追踪法术和血肉信仰强化制造持续走位压力，考验玩家在弹幕和敌群之间维持攻击节奏的能力。

### 2.2 给玩家制造的战斗难题

| 维度 | 内容 |
| --- | --- |
| 主要压力 | 远程弹幕、区域封锁、强化敌人、持续走位压力 |
| 玩家应对方式 | 观察地面预警、切入打断、利用空窗爆发、优先清理强化目标 |
| 服务的卡组体验 | 位移、爆发、打断、范围清理、终结技 |
| 失败惩罚 | 被区域压制拖慢输出；被追踪弹打断连招；被强化敌人夹击 |
| 成功反馈 | 打断仪式、破除区域、抓住施法后摇爆发 |
| 新手教学价值 | 作为精英，不承担新手教学，主要验证综合走位能力 |
| 熟练玩家价值 | 强迫玩家在卡组爆发窗口前处理场地和目标优先级 |

## 3. 剧情描述与形象设定

### 3.1 背景描述

腐化神秘学家曾经是圣城最受尊敬的学者之一。他们通过星象、梦境和仪式解释世界的“名、相、响”，与炼金术士共同推动技术和神秘学的发展。星空沉默后，他们逐渐被教会和新秩序视为异端。

被关入地下后，最折磨他的不是地位坠落，而是再也听不见星星与夜空的声音。血肉爆发后，他把体内脉动的血肉误认为新的神谕，最终背离原本信仰，成为血肉的忠实信徒。

## 4. 基础行为设计

### 4.1 行为树概览

```text
待机于仪式区域
-> 发现玩家
-> 与玩家保持中远距离
-> 释放追踪星血弹
-> 在玩家路径上生成禁行区域
-> 若被近身，短距离瞬移或释放推拒
-> 低血时进入血肉祷告，强化下一轮技能
-> 死亡
```

行为逻辑说明：

| 行为 | 内容 |
| --- | --- |
| 巡逻 | 通常不巡逻，作为仪式房核心敌人 |
| 发现玩家 | 立即启动地面仪式预警 |
| 接近方式 | 保持中远距离，避免贴身 |
| 攻击节奏 | 技能前摇明显，但覆盖范围大 |
| 后摇设计 | 大型仪式后有明显输出窗口 |
| 群体定位 | 精英核心，少量杂兵负责干扰 |
