
## 4. 基础行为设计

### 4.1 行为树概览

用自然语言描述它的战斗逻辑。

```text
巡逻
-> 发现玩家
-> 进入警戒
-> 根据距离选择移动方式
-> 根据攻击权重选择技能
-> 攻击后进入冷却、后撤或继续压迫
-> 死亡或触发特殊机制
```
