## 2. 敌人需求定位

### 2.1 核心定位

鼠巢是场景召唤源，用持续生成腐鼠制造局部时间压力，迫使玩家在处理主敌和清理源头之间做选择。

### 2.2 给玩家制造的战斗难题

| 维度 | 内容 |
| --- | --- |
| 主要压力 | 周期刷腐鼠、拖慢节奏、限制安全区域 |
| 玩家应对方式 | 优先摧毁源头、利用范围卡清理腐鼠、控制战斗位置 |
| 服务的卡组体验 | 溅射、燃烧、毒、范围清场、目标优先级 |
| 失败惩罚 | 腐鼠不断补充，玩家被持续骚扰 |
| 成功反馈 | 摧毁鼠巢后刷新停止，场面立即变干净 |
| 新手教学价值 | 教会玩家“召唤源必须处理” |
| 熟练玩家价值 | 可作为高收益范围卡组的材料，但不能放任过久 |

## 3. 剧情描述与形象设定

### 3.1 背景描述

鼠巢不是普通动物巢穴，而是监狱刑罚、尸体堆积和血肉诅咒共同形成的污染节点。被鼠刑处置过的囚犯、腐烂稻草、破碎铁笼和暗河湿气在这里纠缠，形成会持续孵化腐鼠的活体洞穴。

它没有意志，却像监狱本身的伤口。只要不被清除，鼠群就会从里面爬出，把战斗拖入肮脏、混乱和无法停歇的状态。

| 精神状态 | 无意识，本能孵化 |
| 与监狱的关系 | 监狱底层刑罚与腐败的环境化表现 |
| 叙事关键词 | 鼠洞、尸堆、稻草、铁笼、活体巢穴 |
| 可投放环境叙事 | 破碎鼠笼、囚犯尸体、墙洞、腐烂稻草、血肉管道 |

## 4. 基础行为设计

### 4.1 行为树概览

```text
房间开始时存在
-> 周期检测同屏腐鼠数量
-> 若低于上限，孵化腐鼠
-> 被攻击时播放受击反馈
-> 生命归零后破裂
-> 停止召唤
```

行为逻辑说明：

| 行为 | 内容 |
| --- | --- |
| 巡逻 | 无 |
| 发现玩家 | 无需发现，进入战斗后自动工作 |
| 接近方式 | 不移动 |
| 攻击节奏 | 不直接攻击 |
| 后摇设计 | 每次孵化有明显前兆 |
| 群体定位 | 召唤源，必须配合腐鼠或其他敌人 |
