## 2. 敌人需求定位

### 2.1 核心定位

鼠刑囚犯是低血量近战骚扰敌人，用快速接近和死亡爆鼠制造战场噪音，迫使玩家注意击杀时机和召唤物清理。

### 2.2 给玩家制造的战斗难题

| 维度 | 内容 |
| --- | --- |
| 主要压力 | 数量、快速贴身、死亡后释放腐鼠 |
| 玩家应对方式 | 群体清理、控制击杀位置、使用溅射或燃烧处理召唤物 |
| 服务的卡组体验 | 溅射、燃烧、毒、群体击退、清场连锁 |
| 失败惩罚 | 击杀后腐鼠过多，玩家被持续干扰 |
| 成功反馈 | 一次范围攻击清理囚犯和腐鼠 |
| 新手教学价值 | 教会玩家死亡触发机制和召唤物上限概念 |
| 熟练玩家价值 | 作为群体材料，放大玩家范围卡组的爽感 |

### 2.3 关卡节奏定位

| 字段 | 内容 |
| --- | --- |
| 普通房作用 | 提供群体目标和死亡触发变化 |

| 是否适合作为首见敌人 | 是，但首见数量不宜过多 |
| 是否适合作为压轴敌人 | 否 |

## 3. 剧情描述与形象设定

### 3.1 背景描述

鼠刑囚犯曾被关在底层潮湿牢房中，接受用饥饿老鼠折磨肉体的刑罚。刑罚结束后他们没有真正死去，伤口里长期残留的鼠群、稻草和腐肉成为血肉诅咒最容易寄生的温床。

诅咒爆发后，这些囚犯的腹腔和肋骨成了小型鼠巢。它们仍以人的形状奔跑和抓挠，但死亡时身体会像破裂的布袋一样打开，释放出藏在体内的腐鼠。

## 4. 基础行为设计

### 4.1 行为树概览

```text
游荡
-> 发现玩家
-> 快速接近
-> 使用抓击或扑咬
-> 攻击后短暂绕侧
-> 低血时更加激进
-> 死亡时爆出腐鼠
```

行为逻辑说明：

| 行为 | 内容 |
| --- | --- |
| 巡逻 | 在牢房、鼠洞、尸堆附近无序游荡 |
| 发现玩家 | 发出尖叫并快速靠近 |
| 接近方式 | SwarmFlank，适合成群侧绕 |
| 攻击节奏 | 轻快但单次伤害低 |
| 后摇设计 | 攻击后短暂停顿，便于玩家反击 |
| 群体定位 | 群体骚扰，不应单只构成高威胁 |
