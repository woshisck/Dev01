## 2. 敌人需求定位

### 2.1 核心定位

```text
看守队长是 L1 监狱看守体系的精英指挥官。它通过远程攻击压制玩家，通过召唤狱卒和强化狱卒扩大场面压力；当玩家贴近时，它会用武器重击推开玩家并后撤，迫使玩家在“追击队长”和“清理被增强狱卒”之间做选择。
```

### 2.2 给玩家制造的战斗难题

| 维度 | 内容 |
| --- | --- |
| 主要压力 | 远程攻击、召唤狱卒、强化狱卒、近身击退后撤、低血火焰阶段 |
| 玩家应对方式 | 抓住远程攻击后摇切入；打断召唤和强化；优先清理被 Buff 的狱卒；低血阶段注意火焰伤害 |
| 服务的卡组体验 | 突进切后排、打断、单体爆发、范围清场、终结技、火焰抗压 |
| 失败惩罚 | 队长不断拉开距离，场上狱卒被强化后持续压迫；玩家贴近后被推开，输出窗口丢失 |
| 成功反馈 | 打断队长号令；快速清理召唤物；在队长后撤前完成爆发或终结技 |
| 新手教学价值 | 不承担基础教学，作为玩家理解“精英指挥官 + 小怪配合”的综合测试 |
| 熟练玩家价值 | 要求玩家判断目标优先级、保留爆发窗口，并利用卡组处理增援 |

## 3. 剧情描述与形象设定

### 3.1 背景描述

```text
看守队长曾是维尔斯卡牢狱中真正掌握基层暴力的人。他不是最高级的典狱官，却负责调度巡逻、处刑、追捕和封锁。那些被收编的街头恶霸、持枪看守和低阶狱卒，都在他的哨令和审讯名单下行动。

血肉诅咒爆发后，它没有像普通看守一样完全沦为进食本能。相反，它保留了“指挥”和“维持秩序”的执念，只是这种秩序已经变成了更极端的暴力体系。它会召来狱卒围攻入侵者，也会用血肉赐福强化其他看守，让他们像被点燃的刑具一样执行命令。

当生命降低到危险状态时，它头部铁笼中的火炬会被点燃。火焰不是普通灯火，而是监狱处刑仪式、煤油、血肉和赐福混合后的刑焰。进入该阶段后，它的远程攻击和近身推击都会附带火焰伤害。
```

| 腐化来源 | 血肉赐福 / 处刑火炬仪式 / 监狱权力执念 |

## 4. 基础行为设计

### 4.1 行为树概览

```text
待机或巡逻
-> 发现玩家
-> 保持中远距离并进行远程攻击
-> 周期性判断场上狱卒数量
-> 若狱卒不足，召唤狱卒
-> 若已有狱卒，释放队长号令，为狱卒提供增强 Buff
-> 玩家靠近时，使用重击推击击退玩家
-> 推击成功或近身压力过高时，向远处后撤
-> 生命较低时，头部铁笼火炬点燃，进入火焰阶段
-> 火焰阶段攻击附带火焰伤害，强化 Buff 变得更危险
-> 死亡
```

行为逻辑说明：

| 行为 | 内容 |
| --- | --- |
| 巡逻 | 在看守驻守区、处刑台或牢房中层平台巡逻 |
| 发现玩家 | 举起远程武器或发出号令，进入指挥战斗 |
| 接近方式 | 主动保持中远距离，不主动贴身 |
| 攻击节奏 | 远程攻击负责压制；召唤和强化负责扩大场面压力 |
| 后摇设计 | 远程攻击、召唤和强化都需要有明显前摇或后摇，允许玩家打断 |
| 群体定位 | 精英核心，依靠狱卒形成压力；单独存在时威胁降低 |
| 近身逻辑 | 玩家靠近后使用重击推击，命中后后撤 |
| 低血逻辑 | 生命低于阈值后进入火焰阶段，提高危险度 |

### 4.3 特殊机制

| 机制 | 内容 |
| --- | --- |
| 召唤机制 | 周期性召唤狱卒类敌人。建议优先召唤腐化看守A或基础狱卒；同屏狱卒达到上限时不再召唤 |
| Buff机制 | 队长号令会增强附近狱卒，可提供攻击力、攻击速度、移速、防御或短霸体。Buff 应有明显视觉表现 |
| 死亡触发 | 队长死亡后，已施加的队长号令 Buff 立即移除或快速衰减；不建议死亡时再次召唤 |
| 阶段变化 | 生命低于阈值后头部火炬点燃，进入火焰阶段；远程攻击和近身推击附带火焰伤害 |
| 环境交互 | 可与看守警钟、火炬架、处刑台、牢门封锁机关产生叙事关联，首版可不做强交互 |
| 可打断规则 | 召集狱卒、队长号令、刑焰点燃前摇可被强击退、破甲、终结技打断；普通远程攻击可被高硬直打断 |
| 霸体规则 | 队长推击和刑焰点燃期间可短霸体；召唤与号令不建议霸体过高，避免玩家无法反制 |
| 抗性规则 | 中等击退抗性，中等火焰抗性；低毒抗性；低血火焰阶段可提高韧性但不应免疫控制 |
| 召唤上限 | 建议同屏由队长召唤的狱卒最多 2 名，避免精英房变成无限拖延 |
| 强化目标 | 优先强化已有狱卒；没有狱卒时优先召唤，不空放 Buff |
