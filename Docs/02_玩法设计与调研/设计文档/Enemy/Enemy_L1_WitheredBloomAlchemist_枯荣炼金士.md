## 2. 敌人需求定位

### 2.1 核心定位

枯荣炼金士是拥有明显二阶段反转的近战精英，前期枯槁虚弱、行动迟缓，血肉爆发后肌肉膨胀并进入高压近战形态，考验玩家对阶段转换和爆发窗口的判断。

### 2.2 给玩家制造的战斗难题

| 维度 | 内容 |
| --- | --- |
| 主要压力 | 二阶段爆发、近战高压、炼金药剂区域、攻击节奏突变 |
| 玩家应对方式 | 前期建立输出，转换时远离，二阶段观察大动作后反击 |
| 服务的卡组体验 | 单体爆发、破甲、终结技、阶段压血 |
| 失败惩罚 | 在转阶段时贪刀被爆发命中；二阶段被连续近战压制 |
| 成功反馈 | 抓住枯槁期输出，打断或惩罚膨胀期大后摇 |
| 新手教学价值 | 不承担基础教学，作为精英验证阶段战斗理解 |
| 熟练玩家价值 | 让玩家规划卡组爆发是否留到二阶段 |

## 3. 剧情描述与形象设定

### 3.1 背景描述

枯荣炼金士曾是圣城中备受尊敬的炼金术士，代表物质、技术和文明进步。星空沉默、教会路线转向后，炼金术逐渐被视为傲慢的异端学问，他也在搜捕中被关入监狱。

地位的急速坠落让他陷入癫狂。他憎恶血肉赐福，认为那是对理性和技术的侮辱，因此长期拒绝进食、拒绝改造，身体变得枯槁到几乎无法行走。可血肉爆发时，退缩的肌肉被强行灌满生命，甚至比过去更强。

## 4. 基础行为设计

### 4.1 行为树概览

```text
一阶段：枯槁游荡
-> 发现玩家
-> 缓慢接近，投掷少量炼金药剂
-> 生命低于阈值或受到足够伤害
-> 血肉爆发转阶段
-> 二阶段：高速近战压迫
-> 使用膨胀手臂重击、冲撞和药剂爆裂
-> 死亡
```

行为逻辑说明：

| 行为 | 内容 |
| --- | --- |
| 巡逻 | 一阶段几乎不巡逻，像病人一样拖行 |
| 发现玩家 | 低声自语，缓慢后退或投掷药剂 |
| 接近方式 | 一阶段保守，二阶段 Direct/BruiserHold |
| 攻击节奏 | 一阶段慢，二阶段突然变快 |
| 后摇设计 | 二阶段重击后摇明显，提供反击窗口 |
| 群体定位 | 单体精英，不依赖大量杂兵 |

| 机制 | 内容 |

| 召唤机制 | 无 |
