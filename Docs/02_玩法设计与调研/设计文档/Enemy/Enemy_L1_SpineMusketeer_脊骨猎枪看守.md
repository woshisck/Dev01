## 2. 敌人需求定位

### 2.1 核心定位

脊骨猎枪看守是监狱中的远程压力敌人，用明显瞄准前摇、骨弹射击和脊骨陷阱迫使玩家处理后排，打破只围绕近战敌人循环的舒适节奏。

### 2.2 给玩家制造的战斗难题

| 维度 | 内容 |
| --- | --- |
| 主要压力 | 远程瞄准、直线弹道、陷阱封路、后排持续消耗 |
| 玩家应对方式 | 观察瞄准红线、横向闪避、优先突进击杀、利用前排敌人遮挡弹道 |
| 服务的卡组体验 | 位移、击退、远程反制、优先击杀、爆发切后排 |
| 失败惩罚 | 被骨弹打断连招；被陷阱限制移动后遭到前排压制 |
| 成功反馈 | 打断装填、绕开弹线、快速切入后排 |
| 新手教学价值 | 教会玩家不要只盯近战目标，需要观察远程前摇和场地危险 |
| 熟练玩家价值 | 在多敌人组合中迫使玩家做目标优先级判断 |

## 3. 剧情描述与形象设定

### 3.1 背景描述

脊骨猎枪看守曾是山地猎人，后来被雇佣为监狱持枪看守。他习惯把逃犯和犯人当成猎物，也喜欢用犯人练习枪法。对他而言，监狱不是秩序机构，而是一座永远不会缺少靶子的猎场。

血肉诅咒爆发后，他收集的脊柱没有腐烂，而是在枪膛、弹药袋和陷阱中继续抽动。那些骨节被削成弹芯，被串成地刺，也被当作战利品挂在枪托旁。

## 4. 基础行为设计

### 4.1 行为树概览

```text
巡逻
-> 发现玩家
-> 与玩家保持中远距离
-> 若距离合适，进入瞄准
-> 发射骨弹
-> 装填后短暂移动换位
-> 若玩家逼近，后撤或放置脊骨陷阱
-> 死亡
```

行为逻辑说明：

| 行为 | 内容 |
| --- | --- |
| 巡逻 | 在牢房上层、走廊尽头或处刑平台附近巡逻 |
| 发现玩家 | 迅速拉开距离，举枪瞄准 |
| 接近方式 | 保持 PreferredRange，不主动贴身 |
| 攻击节奏 | 射击前摇清晰，射击后装填窗口明显 |
| 后摇设计 | 装填期间容易被突进或打断 |
| 群体定位 | 后排压制，需要前排保护 |
