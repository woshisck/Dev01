## 2. 敌人需求定位

### 2.1 核心定位

腐化看守A是监狱区域最基础的近战压力敌人，用无武器肉体攻击、短距离冲撞和高于杂兵的韧性，教会玩家观察前摇、闪避重击、利用击退或破甲处理前排目标。

### 2.2 给玩家制造的战斗难题

| 维度 | 内容 |
| --- | --- |
| 主要压力 | 中等血量、近战压迫、重击伤害、短距离突进 |
| 玩家应对方式 | 观察攻击前摇、侧向闪避、利用击退拉开距离、用破甲或终结技快速处理 |
| 服务的卡组体验 | 击退、伤口、破甲、终结技、单体爆发 |
| 失败惩罚 | 被连续近身压制；被重击打断节奏；被逼入墙角后遭到其他敌人补刀 |
| 成功反馈 | 被击退后露出破绽；重击后摇可反击；破甲后可快速处决 |
| 新手教学价值 | 教会玩家区分轻攻击和重攻击前摇，理解近战敌人攻击后的反击窗口 |
| 熟练玩家价值 | 在敌群中作为前排阻挡玩家路线，迫使玩家决定先处理前排还是先切后排 |

## 3. 剧情描述与形象设定

### 3.1 背景描述

腐化看守A原本是圣城街头的恶霸。他并非正式出身的狱卒，而是在监狱长期扩张和人手不足时被收编，成为最低阶的看守。相比维护秩序，他更享受支配、殴打和恐吓犯人的过程。

血肉赐福让它获得了远超常人的力量。最初，监狱长只把它当作处理暴动和处决畸变囚犯的工具；但血肉诅咒爆发后，它的凶残被完全激活。它不再需要刑具，膨胀的手臂、撕裂的指骨和异常强壮的躯体本身就变成了刑具。

### 3.2 当前状态

| 字段 | 内容 |
| --- | --- |
| 生前身份 | 街头恶霸 / 监狱临时收编暴徒 |
| 监狱身份 | 低阶看守 |
| 腐化来源 | 血肉赐福 |
| 精神状态 | 嗜血、饥饿、施暴成瘾、保留部分“看守职责”的扭曲执念 |
| 与监狱的关系 | 表面维护秩序，实质是监狱暴力的执行者 |
| 叙事关键词 | 街头暴力、低阶权力、肉体刑具、失控看守、饥饿 |
| 可投放环境叙事 | 被砸弯的铁栏、墙面抓痕、被徒手撕开的囚衣、没有武器却残留大量钝击痕迹的尸体 |

| 变体 | 说明 |
| --- | --- |
| 刑棒看守 | 持棍近战，攻击范围更长，重击带击退 |

## 4. 基础行为设计

### 4.1 行为树概览

```text
巡逻
-> 发现玩家
-> 发出低吼并进入警戒
-> 直线接近玩家
-> 距离较近时使用徒手轻攻击
-> 距离稍远时使用短距离冲撞
-> 玩家持续贴身时尝试重击或抓取
-> 攻击后进入短暂后摇
-> 被连续攻击时触发短霸体，强行完成一次重击
-> 死亡
```

行为逻辑说明：

| 行为 | 内容 |
| --- | --- |
| 巡逻 | 在牢房、走廊或处刑区附近缓慢游荡 |
| 发现玩家 | 转向玩家，短暂停顿，发出低吼 |
| 接近方式 | Direct，优先直线压迫 |
| 攻击节奏 | 轻攻击频率中等，重击慢但伤害高 |
| 后摇设计 | 重击后摇明显，提供玩家反击窗口 |
| 群体定位 | 不负责包围，主要作为前排阻挡 |

### 4.3 特殊机制

| 机制 | 内容 |
| --- | --- |
| 召唤机制 | 无 |
| 死亡触发 | 默认无死亡触发；不爆鼠、不尸爆，避免和囚犯类敌人混淆 |
| 阶段变化 | 默认无二阶段；可在高难房间通过敌人 Buff 追加激怒 |
| 环境交互 | 可被击退撞墙；重击可打碎少量脆弱场景物件 |
| 可打断规则 | 轻攻击可被普通攻击打断；重击前摇可被强击退、破甲或终结技打断 |
| 霸体规则 | 连续受击达到阈值后短暂霸体，允许完成一次重击；用于防止被无限轻攻击压制 |
| 抗性规则 | 中等击退抗性；低毒抗性；低燃烧抗性；无远程抗性 |
| 变体规则 | 默认不持武器；武器版本作为独立看守变体，不通过本敌人随机换武器实现 |
