## 2. 敌人需求定位

### 2.1 核心定位

霸道囚犯是囚犯系的高压前排，用更高血量、抓摔和死亡爆鼠迫使玩家谨慎处理击杀时机，是鼠刑囚犯机制的强化版本。

### 2.2 给玩家制造的战斗难题

| 维度 | 内容 |
| --- | --- |
| 主要压力 | 高血量、近身抓摔、死亡爆鼠、小范围爆裂 |
| 玩家应对方式 | 控制血量、拉开击杀位置、保留范围清理手段 |
| 服务的卡组体验 | 单体爆发、终结技、击退、范围清场 |
| 失败惩罚 | 近身被抓摔；死亡后腐鼠包围；爆裂打乱站位 |
| 成功反馈 | 击杀后立即清掉腐鼠，形成漂亮的范围连锁 |
| 新手教学价值 | 鼠刑囚犯死亡机制的升级教学 |
| 熟练玩家价值 | 让玩家规划爆发时机和站位，而不是无脑击杀 |

## 3. 剧情描述与形象设定

### 3.1 背景描述

霸道囚犯曾是牢房里的囚犯头目。他没有任何正式权力，却靠拳头、恐吓和抢夺食物统治同牢犯人。看守默许他的存在，因为一个会压迫其他囚犯的囚犯，往往比看守更能维持牢房秩序。

他也曾受过鼠刑，但没有像其他囚犯一样迅速崩溃。血肉诅咒爆发后，那些伤口被重新打开，鼠群在他的腹部、背部和腿侧筑巢。他仍然保持着支配欲，甚至会把体内的腐鼠当作最后的威胁。

## 4. 基础行为设计

### 4.1 行为树概览

```text
待机或缓慢游荡
-> 发现玩家
-> 直线接近并试图占据近身位置
-> 使用拳击、抓摔或肩撞
-> 低血时攻击欲望提升
-> 死亡时身体爆裂，释放腐鼠
```

行为逻辑说明：

| 行为 | 内容 |
| --- | --- |
| 巡逻 | 在囚犯聚集区或大牢房中徘徊 |
| 发现玩家 | 怒吼并冲向玩家 |
| 接近方式 | BruiserHold，保持近身压迫 |
| 攻击节奏 | 慢于鼠刑囚犯，但伤害和血量更高 |
| 后摇设计 | 抓摔和重击后有明显反击窗口 |
| 群体定位 | 可作为囚犯群中的核心目标 |
