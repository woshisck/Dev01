# 体验定位：玩家一眼选择理由

## 一句话定位

**你不是在探索地牢，而是在一个把你当成异物排斥的神明污染世界里，用越来越失控的卡牌连锁，把每一次攻击推向自动爆发。**

更短的对外表达：

```text
在会排斥你的末日世界里，边战斗边出牌，越久越强，也越危险。
```

## 玩家第一眼应该看到什么

玩家在商店页、短视频、截图或试玩前 10 秒，应立刻看到三件事：

| 第一眼信号 | 玩家理解 | 画面/系统表达 |
| --- | --- | --- |
| 世界在排斥你 | 这里不是普通地牢，而是一个危险降临、正在污染/同化的世界 | 地图边界污染、空间压迫、神明投影、环境像免疫系统一样追杀玩家 |
| 攻击会顺序出牌 | 战斗易上手，不需要先学复杂卡牌规则 | 玩家连续攻击，卡槽按顺序亮起、消耗、连锁 |
| 满足条件自动爆发 | 玩家不用管理额外收尾按钮，系统在合适时刻把爽点打出来 | 卡牌反应达成后自动释放大反馈：范围、击飞、连锁、屏幕震动、音效提升 |

## 核心体验承诺

### 1. 易上手：玩家只管打

玩家的第一层心智应该是：

```text
攻击 -> 当前卡触发 -> 下一张卡准备 -> 连锁达成 -> 自动爆发
```

玩家不需要先理解复杂构筑，也不需要主动管理额外收尾操作、额外确认窗口或轻重攻击匹配。  
战斗输入保持直接，卡牌系统在后台把攻击结果变得更丰富。

### 2. 有构筑：玩家知道自己为什么强

卡牌构筑不是“随机给一堆效果”，而是三段式：

```text
启动卡：给世界或敌人上状态
催化卡：放大、转译、扩散状态
爆发卡：条件满足后自动引爆
```

玩家应该能在一次战斗后复盘：

- 我先给敌人挂了某种状态。
- 后面的卡把它变成了更危险的连锁。
- 系统自动把这个连锁兑现成爆发。

### 3. 有压迫：世界不是关卡容器，而是敌人

世界设定的核心体验不是“我在刷房间”，而是：

```text
我停留得越久，世界越发现我、排斥我、污染我。
```

这让肉鸽循环有更强叙事钩子：

- 重开不是失败读档，而是高维存在重新投射时间线。
- 房间推进不是普通探索，而是和世界排斥反应赛跑。
- 玩家变强不是单纯升级，而是越深入越能撬动这个世界的规则。
- 敌人也变强，因为污染和同化正在同步加剧。

### 4. 有探索：地图不是线性房间，而是可理解的世界切片

地图应像半随机银河城，而不是一次性地牢：

- 玩家能返回上一关或上一片区，形成“我理解了这个世界结构”的感觉。
- 部分路径需要能力、时间线知识或世界污染状态才能打开。
- 每次重开都不是只换房间顺序，而是让玩家更理解世界为什么会这样。

## 3 分钟体验目标

玩家试玩前三分钟应该形成这个判断：

```text
这游戏打起来很直接，但卡牌会让每一次攻击结果不一样。
这个世界很危险，而且不是背景板。
我想再打一局，看看下一组卡会不会炸出更离谱的反应。
```

前三分钟应该避免：

- 长篇世界观解释。
- 复杂卡组编辑。
- 旧背包整理。
- 需要先理解大量关键词才能爽。

前三分钟应该给：

- 一套固定初始卡组。
- 明确可见的顺序出牌 HUD。
- 一次启动 -> 催化 -> 自动爆发。
- 一次世界排斥的压力事件。

## 30 分钟体验目标

玩家完成第一次较完整游玩后，应该形成这个判断：

```text
我不是只在变强，我是在和这个世界互相污染。
我每次重开都像换一条时间线去试探世界。
我的卡组不是数值堆叠，而是一套能被我读懂的连锁机器。
```

30 分钟内应让玩家看到：

- 至少 2 种启动卡。
- 至少 2 种催化方式。
- 至少 1 次高反馈自动爆发。
- 至少 1 次世界排斥升级。
- 至少 1 个“之前打不开，现在理解或能打开”的地图/路径钩子。

## 体验关键词

| 关键词 | 要表达的体验 | 不要滑向 |
| --- | --- | --- |
| 易上手 | 玩家攻击即可触发系统 | 按键复杂、卡牌抢注意力 |
| 顺序出牌 | 卡组像节奏条，玩家能预判下一次攻击 | 随机乱触发、玩家看不懂 |
| 自动爆发 | 条件满足后系统兑现爽点 | 主动收尾按钮、额外确认窗口 |
| 世界排斥 | 地图和敌人都像在驱逐玩家 | 普通倒计时、普通诅咒条 |
| 时间线投射 | 肉鸽重开有世界观解释 | 纯粹死亡重开 |
| 半随机银河城 | 探索让玩家理解世界 | 只有房间随机，没有世界结构 |

## 对外宣传角度

### 主宣传句

```text
攻击出牌，连锁爆发；在排斥你的神明污染世界里，重投时间线。
```

### 商店页短卖点

- **顺手的动作战斗**：不用先学复杂卡牌规则，攻击就能出牌。
- **可读的卡牌连锁**：启动、催化、爆发，构筑结果能在战斗中立刻验证。
- **自动爆发爽点**：条件达成后系统自动引爆，不用额外收尾操作。
- **会排斥你的世界**：停留越久，力量越强，世界和敌人也越危险。
- **时间线肉鸽叙事**：每次重开都是高维存在的一次新投射。
- **半随机银河城探索**：不是刷地牢，而是逐步理解一个正在污染的世界。

## 体验红线

1. 不让卡组抢走动作战斗注意力。
2. 不回到旧 2D 背包整理心智。
3. 不恢复主动收尾操作、额外确认窗口或独立爆发卡体系。
4. 不把世界排斥做成普通倒计时，应体现为地图、敌人、事件和污染规则的变化。
5. 不把肉鸽重开写成普通死亡惩罚，应服务“时间线投射”的设定。
6. 不让自动爆发变成黑箱，HUD 必须让玩家知道是哪组卡牌反应被兑现。

## 当前设计判断

这个方向的“惊艳点”不在于单独某个系统，而在于三者叠加：

```text
低门槛动作输入
+ 可预判的顺序卡牌连锁
+ 世界排斥/时间线投射的强设定解释
= 一眼能懂、越打越想研究的动作肉鸽体验
```

玩家被吸引的理由应该是：

> “我看得懂怎么打，也看得出每次构筑会让战斗爆出不同结果；而且这个世界不是背景，它正在把我排出去。”
