# 终结技卡牌：怒锋入铠

## 设计定位

`怒锋入铠` 是一张德式双手剑主题终结技卡。

它提供一个额外连招动作。玩家按指定轻重输入完成连招后，主动触发一段终结技蒙太奇；如果没有按出完整输入，则在卡牌触发时释放默认技能。

核心体验：

```text
大剑先破阵，再半剑入铠，最后倒持剑柄凿碎护甲。
```

## 德式双手剑特性提炼

| 特性 | 转化为卡牌表现 |
| --- | --- |
| 长柄大刃，适合宽大横扫 | 默认技能是大范围扇形破阵斩 |
| 双手发力，动作承诺强 | 绑定 Heavy 起手，不适合轻攻击随便触发 |
| 半剑术对甲突刺 | 完整输入中段加入短距离突刺 |
| Mordhau 倒持剑柄/护手凿击 | 终结蒙太奇最后一下造成高护甲伤害和短眩晕 |
| 面对多人时负责开路 | 默认效果带扇形击退 |

## 单张卡填写

| 字段 | 内容 |
| --- | --- |
| 卡牌 ID | `Card.Finisher.Zweihander.RageIntoArmor` |
| 卡牌名称 | 怒锋入铠 |
| 稀有度 | 稀有 |
| 卡牌定位 | 终结 / 破甲 / 击退 |
| 一句话描述 | 当玩家用 Heavy 触发卡牌时，释放破阵斩；若继续输入 Light -> Heavy，则追加德式双手剑终结蒙太奇。 |

## 攻击触发

| 字段 | 内容 |
| --- | --- |
| `RequiredAction` | Heavy |
| `ConsumeTiming` | 命中消耗 |
| 是否要求命中 | 是 |
| 是否要求终结击 | 否，卡牌自己提供额外终结动作 |
| 是否允许 DashSave 保留 | 是，仅保留输入窗口，不重置卡组 Index |
| 不匹配是否消耗 | 是 |
| 不匹配是否输出特征 | 否 |

## 默认效果

默认效果在卡牌被 Heavy 命中触发时立即释放。

| 字段 | 内容 |
| --- | --- |
| `EffectForm` | `ConeSlash` |
| 默认伤害 | 160% 武器伤害 |
| 默认状态 | 无 |
| 默认击退 | 扇形内敌人受到一次中等击退 |
| 默认护甲交互 | 读取护甲全局表；额外造成 25% 护甲伤害 |
| 默认 VFX | 宽大金属剑风，地面短弧形裂痕 |
| 默认 HUD | “怒锋：破阵斩” |

默认技能名称：

```text
破阵斩
```

## 额外连招动作

卡牌触发后，开启短输入窗口。

```text
起手：Heavy 命中，卡牌触发
追加输入：Light -> Heavy
成功结果：播放终结技蒙太奇
失败结果：只保留默认破阵斩
```

建议窗口：

| 阶段 | 输入 | 时间 |
| --- | --- | --- |
| 触发 | Heavy 命中 | 消耗卡牌并释放破阵斩 |
| 衔接 1 | Light | 0.55 秒内输入 |
| 衔接 2 | Heavy | Light 命中后 0.65 秒内输入 |
| 成功 | 自动播放蒙太奇 | `Montage_Finisher_Zweihander_Mordhau` |

输入失败、超时、打空、被打断时，不播放终结蒙太奇。

## 终结技

终结技名称：

```text
半剑凿铠
```

动作结构：

```text
1. 半剑压步突刺：短距离前刺，命中主目标。
2. 横向错身推开：用剑身/护手拨开周围敌人。
3. Mordhau 凿击：倒持剑柄或护手向下砸击护甲。
```

| 字段 | 内容 |
| --- | --- |
| 是否需要 `bIsComboFinisher` | 否，完成卡牌指定输入即可 |
| 终结技伤害 | 主目标 320% 武器伤害 |
| 终结技范围 | 主目标 240 半径小范围震荡 |
| 终结技控制 | 主目标短眩晕 0.45 秒；霸体目标免疫眩晕但吃护甲伤害 |
| 终结技护甲伤害 | 对护甲额外 75% 伤害 |
| 终结技击退 | 周围敌人轻击退，主目标不被远推，避免打空后续凿击 |
| 终结技 VFX | 金属白闪、护甲碎片、下砸冲击环 |
| 终结技 HUD | “半剑凿铠：输入完成” |
| 失败降级 | 不播放终结蒙太奇，只保留破阵斩 |

## 特征与连携

这张卡的普通触发不输出强特征。终结技成功时才输出受限特征。

| 字段 | 内容 |
| --- | --- |
| `SignatureTrait` | `ArmorPressure` |
| 是否受限特征 | 是 |
| 是否可传递给下一张 | 否，终结技特征不默认传递 |
| 可读取上一张特征 | `ForceImpulse` / `ArmorPressure` / `AreaScale` |
| 读取效果 | 只影响终结技数值或范围，不改变动作结构 |

可解释规则：

| 上一张特征 | 终结技变化 |
| --- | --- |
| `ForceImpulse` | 破阵斩击退增强，终结技周围震荡半径 +15% |
| `ArmorPressure` | Mordhau 凿击额外护甲伤害 +25% |
| `AreaScale` | 破阵斩扇形角度略增，不扩大眩晕范围 |

不可解释时：

```text
只保留默认破阵斩和终结技基础数值。
```

## 失败与降级

| 场景 | 结果 |
| --- | --- |
| 当前卡用 Light 触发 | 卡牌消耗，只执行一次弱化破阵斩：100% 伤害，无终结输入窗口 |
| Heavy 命中但后续输入失败 | 保留默认破阵斩，不播放蒙太奇 |
| 后续 Light 打空 | 输入链断开，不播放蒙太奇 |
| 后续 Heavy 超时 | 输入链断开，不播放蒙太奇 |
| 目标有护甲 | 默认与终结都获得护甲伤害收益 |
| 目标霸体 | 眩晕失效，护甲伤害和震荡仍生效 |
| 洗牌期间 | 不触发卡牌 |

## 表现反馈

| 项目 | 内容 |
| --- | --- |
| 命中停顿 | 默认破阵斩 0.06 秒；Mordhau 凿击 0.10 秒 |
| 相机反馈 | 终结凿击时短促下压震动 |
| 敌人反馈 | 护甲目标出现银白碎裂；无甲目标出现重击弯腰硬直 |
| 武器特效 | 大剑刃部白色拖尾，终结时护手位置亮起 |
| 地面表现 | 破阵斩扇形划痕；凿击产生圆形裂纹 |
| 音效关键词 | 厚重挥砍、金属碰撞、护甲碎裂、低频冲击 |

## 制作备注

- 这张卡不建议做成基础卡，适合作为稀有终结技卡。
- 它不依赖完整历史，只读取上一张 `TraitContext`。
- 额外连招动作属于卡牌提供的短输入窗口，不改变所有武器的基础连招树。
- 终结蒙太奇需要战斗侧支持“卡牌指定输入完成后触发 Montage”。
- 默认技能必须独立成立，避免玩家没按出连招时感觉卡牌完全浪费。
