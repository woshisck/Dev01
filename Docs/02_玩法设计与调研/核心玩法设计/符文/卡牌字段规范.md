# 卡牌字段规范

## 单张卡字段

| 字段 | 用途 | 示例 |
| --- | --- | --- |
| `RequiredAction` | 攻击要求 | Light / Heavy / Any |
| `ConsumeTiming` | 消耗时机 | 命中消耗 / 出招确认消耗 |
| `EffectForm` | 效果形态 | 近战打击 / 剑气 / 扇形斩 / 圆形爆发 / 地面场 |
| `DefaultEffect` | 默认效果 | 固定伤害 / 默认燃烧 / 默认中毒 / 默认击退 |
| `SignatureTrait` | 匹配后激活的特征 | 传染 / 周围灼烧 / 位移惩罚 / 受击追加 |
| `MatchedReward` | 匹配奖励 | 激活特征；无法激活时改为数值增强 |
| `LinkAffix` | 连携词条 | 读取上一张特征 / 输出当前特征 / 特征转译 |
| `FinisherRule` | 终结技 | 单独设计，可读取当前特征 |
| `Rarity` | 稀有度 | 基础 / 普通 / 稀有 / 特殊 |
| `Feedback` | 表现反馈 | 命中停顿 / 闪光 / 音效 / HUD 文案 |

## EffectForm

| EffectForm | 表现方向 | 可承载内容 | 准绳 |
| --- | --- | --- | --- |
| `MeleeHit` | 武器命中 | 伤害、默认状态、一次击退 | 基础卡优先使用 |
| `MoonlightWave` | 剑气/月光 | 伤害、默认状态、一次击退、范围增幅 | 不默认承载强控 |
| `ConeSlash` | 面前扇形 | 伤害、护甲伤害、击退、燃烧 | 适合重击和破甲 |
| `RadialBurst` | 圆形爆发 | 伤害、状态扩散、震荡、爆炸 | 适合终结技 |
| `Projectile` | 投射物 | 伤害、状态、诅咒、穿透 | 适合精确投放 |
| `GroundField` | 地面场 | 持续伤害、范围控制、空间压制 | 限制数量和持续时间 |
| `Trail` | 轨迹 | 播种、切割、弱持续伤害 | 每目标触发次数封顶 |
| `Mark` | 标记 | 伤口、诅咒、引信、延迟结算 | 必须有明显提示 |
| `Explosion` | 爆炸 | 伤害、击退、状态消耗 | 必须有燃料或触发来源 |

## 匹配奖励规则

```text
动作匹配 + 有合适特征 = 激活特征
动作匹配 + 无合适特征 = 数值增强
动作不匹配 = 默认效果
```

默认效果不能传递特征。
