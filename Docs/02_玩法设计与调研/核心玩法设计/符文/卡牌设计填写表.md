失败和降级都保持原始默认效果，不携带特征。

| 场景 | 降级结果 |
| --- | --- |

| 状态不存在 | 反应不触发 |
| 洗牌期间 | 不触发卡牌 |

## 示例：毒先火后

```text
1 卡：毒刃
- RequiredAction: Light
- EffectForm: MeleeHit
- DefaultEffect: 施加中毒
- 匹配后激活 Trait: Spread
- 输出给下一张：Spread

2 卡：燃烧斩
- RequiredAction: Heavy
- EffectForm: ConeSlash
- DefaultEffect: 施加燃烧
- 匹配后激活 Trait: AuraDamage
- 读取上一张 Spread
- 转译结果：燃烧的 AuraDamage 范围扩大，影响目标数增加
- 输出给下一张：AuraDamage + AreaScale
```

## 示例：火先毒后

```text
1 卡：燃烧斩
- RequiredAction: Heavy
- EffectForm: ConeSlash
- DefaultEffect: 施加燃烧
- 匹配后激活 Trait: AuraDamage
- 输出给下一张：AuraDamage

2 卡：毒刃
- RequiredAction: Light
- EffectForm: MeleeHit
- DefaultEffect: 施加中毒
- 匹配后激活 Trait: Spread
- 读取上一张 AuraDamage
- 转译结果：中毒传播时，在传播落点周围附加小范围持续伤害
- 输出给下一张：Spread
```
