# 待定问题清单

## 优先级说明

当前不需要一次回答所有待定问题。世界观成熟化阶段只优先处理会影响剧情因果、能力系统和关卡推进的承重问题。

优先级定义：

- **P0**：必须先定。影响主线动机、前三关、能力树、死亡循环和核心美术方向。
- **P1**：可以边写边定。影响角色细化、支线、后续关卡和术语统一。
- **P2**：暂缓。主要影响远景宇宙观、完整神谱和最终结局。
