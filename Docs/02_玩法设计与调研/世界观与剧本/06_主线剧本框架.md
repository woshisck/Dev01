# 主线剧本框架

## 剧情启动状态

血肉诅咒爆发后，圣城凯莱斯蒂斯沦为怪物乐园。

白昼教堂被封锁，维尔斯卡牢狱中堕落者失控。外部教派和世俗势力试图处理诅咒，但没有人真正理解教宗的复活伏笔与艾利安的存在。

艾利安在地下复活节点苏醒。《遗圣目录》开始书写任务，黑夜少女会在主城中逐渐成为常驻陪伴者，提供神秘学知识、早期指引，并与玩家一起发现世界。

## 主线体验

主线体验不是“寻找失忆真相”的普通结构，而是：

1. 玩家被《遗圣目录》推动，执行看似合理的任务。
2. 玩家通过战斗获取Boss资源，解锁真言、灵视、灵听等权限。
3. 玩家逐渐发现《遗圣目录》省略了关键事实。
4. 玩家意识到自己不是被选中者，而是候圣体，体内还残留教宗的圣座魂屑。
5. 玩家必须决定自己是服从复活剧本、成为力量容器，还是自证其名。

## 章节结构草案

### 第一章：维尔斯卡牢狱

核心问题：

- 我在哪里？
- 谁在命令我？
- 为什么我会复活？

主要内容：

- 艾利安在地下复活节点苏醒。
- 《遗圣目录》给出第一批任务。
- 玩家穿越底层与中层监牢。
- 暗河、血肉管道、旧教会符号初次出现。
- 第一关Boss审判官达尔索·赫尔曼守卫监牢出口。

Boss功能：

- 达尔索提供“原始之血”。
- 解锁前中期成长。
- 让玩家看到骑士团不是普通敌人，而是曾经阻止教宗复活的封锁者。

### 第二章：圣城外城与教堂大门

核心问题：

- 圣城为什么沦陷？
- 教会是否仍代表神意？
- 真言是否能改变现实？

主要内容：

- 玩家进入污染外城。
- 蒸汽革命、阶级压迫、血肉诅咒在环境中叠加。
- 《遗圣目录》指向教堂大门。
- 守秘人埃弗拉伊姆作为封锁者出现。

Boss功能：

- 埃弗拉伊姆提供“真言残响”。
- 解锁真言类主动技能。
- 展示神秘学者为了封锁诅咒而牺牲自我的悲剧。

### 第三章：白昼教堂与星宫

核心问题：

- 谁杀死了教宗？
- 教宗是否真的死了？
- 艾利安到底为何被制造？

主要内容：

- 玩家进入白昼教堂。
- 圣洁大理石与猩红污染形成强对比。
- 奥蕾莉亚·圣雷留下的布置逐渐显现。
- 玩家接触勘破之眼与启迪之灵光。

Boss功能：

- 奥蕾莉亚提供后期循环关键资源。
- 开启真结局路线的主要道具与任务来源。
- 使玩家第一次系统性理解教宗复活计划。

### 第四章：归一之地

核心问题：

- 世界线为何坍塌？
- 绯红之星究竟连接了什么？
- 自我是否能在原初注视下存在？

主要内容：

- 教堂顶部的神秘空间。
- 连接星空、灵界、物质与原初投影。
- 三条能力树的高阶功能开始影响主线分歧。
- 《遗圣目录》与黑夜少女的立场冲突达到顶点。

结局方向待定：

- 服从律令：完成《遗圣目录》的复活剧本。
- 接受渴求：拥抱血肉与吞噬，成为新的灾厄核心。
- 完成自证：以自我锚定稳定世界线，拒绝被命名为容器。

## 任务系统：遗圣目录

《遗圣目录》用于发布任务，但它不是单纯UI。

设计要求：

- 任务文字应带有冷静、古老、仪式化口吻。
- 任务目标经常是片面真实。
- 玩家解锁灵视、灵听后，可发现任务文本的遗漏、改写和回响。
- 高自我锚定路线中，玩家应能拒绝、改写或重新命名部分任务。

## 陪伴系统：黑夜少女

黑夜少女的作用不是提供标准答案，而是陪伴玩家共同探索。

她可以：

- 在主城长期出现，并在关键节点离开主城进入特殊场景。
- 提供神秘学知识、早期指引和危险预警。
- 对《遗圣目录》的任务表达不安。
- 说出不完整的旧纪元记忆。
- 偶尔误导玩家，但不是恶意。
- 被玩家发现的新事实修正认知。

她不应该：

- 取代任务系统。
- 成为万能解释者。
- 频繁打断玩家战斗节奏。

## 三条叙事驱动力

### 渴求

身体想要力量、生存、吞噬与即时奖励。

对应：

- 血肉资源。
- 高风险强化。
- 牺牲、吞噬、额外收获。

### 律令

世界要求玩家成为某种东西。

对应：

- 《遗圣目录》。
- 教会任务。
- 圣徒遗命。
- Boss封锁使命。

### 自证

玩家通过选择证明“我是谁”。

对应：

- 记忆命名。
- 拒绝任务。
- 改写任务。
- 保持选择一致性。
- 稳定自我锚定。
