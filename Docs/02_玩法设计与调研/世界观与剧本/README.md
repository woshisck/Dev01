# 世界观与剧本设定库

> 版本：v0.1  
> 状态：框架整理稿  
> 用途：为剧情、美术、关卡、敌人、UI叙事和玩法机制提供统一世界观依据。

## 设计目标

本设定库不是百科式世界资料堆积，而是服务游戏开发的“世界观骨架”。所有设定需要回答四个问题：

1. 玩家为什么战斗？
2. 玩家为什么反复死亡、重来、收获？
3. 玩家为什么要寻找自我？
4. 世界观如何转化为关卡、美术、敌人、能力树和任务系统？

当前核心体验定义：

> 玩家扮演艾利安，一个被制造出来的候圣体，在血肉诅咒爆发后的圣城与监牢中反复进入濒临崩塌的世界线，杀死怪物、收获力量、承受注视，并逐渐从“被书写命运的容器”变成“能够自证其名的人”。

## 参考作品使用原则

参考比重暂定：

- 《诡秘之主》：60%，主要参考世界观扎实度、途径/权柄式能力组织、神秘学社会结构。
- 《克苏鲁神话》：20%，主要参考不可知、不可描述、宇宙尺度恐怖。
- 《吉列尔莫·德尔托罗之珍奇柜》：20%，主要参考人在绝望与异质恐怖中仍试图保持人的尊严。

使用边界：

- 不复刻任何参考作品的专有设定、组织、名词、剧情结构。
- 参考重点是“设定工程方法”和“恐怖体验气质”，不是直接借用内容。
- 所有核心名词都应服务本项目已有玩法：动作战斗、符文/卡牌构筑、关卡循环、死亡重开、隐藏探索。
