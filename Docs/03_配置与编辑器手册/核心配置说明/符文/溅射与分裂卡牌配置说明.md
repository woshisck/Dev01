# 溅射与分裂卡牌配置说明

## 作用

`溅射/分裂` 是同一张武器适配卡，不是两张独立设计卡。

| 武器类型 | 游戏内显示 | 效果 |
| --- | --- | --- |
| 近战武器 | `溅射` | 攻击命中后，对主目标周围敌人造成本次伤害的 20% |
| 远程武器 | `分裂` | 开火时额外发射 2 个远程弹道 |

当前项目里保留两个生成 DA，只是为了让近战和远程拥有不同图标、名称、触发时机和 BaseFlow。策划理解上仍按同一张卡处理。

| 对象 | 路径 |

## FA 参数

### `FA_Rune512_Splash_Base`

```text
Start
  -> On Damage Dealt
  -> Math Float (* 0.2)
  -> Apply GE In Radius
```

| 节点 | 字段 | 当前值 |
| --- | --- | --- |
| `Math Float` | `Operation` | `Multiply` |
| `Math Float` | `B` | `0.2` |
| `Apply GE In Radius` | `Radius` | `300` |
| `Apply GE In Radius` | `Location Source` | `LastDamageTarget` |
| `Apply GE In Radius` | `Exclude Location Source Actor` | 勾选 |
| `Apply GE In Radius` | `Enemy Only` | 勾选 |
| `Apply GE In Radius` | `Exclude Self` | 勾选 |
| `Apply GE In Radius` | `Application Count` | `1` |
| `Apply GE In Radius` | `SetByCallerTag1` | `Attribute.ActDamage` |

调整伤害比例时只改 `Math Float -> B`，例如 30% 填 `0.3`。

### `FA_Rune512_Split_Base`

```text
Start
  -> Spawn Ranged Projectiles
```

| 节点 | 字段 | 当前值 |
| --- | --- | --- |
| `Spawn Ranged Projectiles` | `YawOffsets` | `-8, 8` |
| `Spawn Ranged Projectiles` | `Use Combat Card Attack Damage` | 勾选 |
| `Spawn Ranged Projectiles` | `Share Attack Instance Guid` | 勾选 |
| `Spawn Ranged Projectiles` | `BulletClass` | `BP_MusketBullet` |
| `Spawn Ranged Projectiles` | `DamageEffectClass` | `UGE_MusketBullet_Damage` |

分裂弹道伤害默认继承本次远程攻击伤害。要改角度，只改 `YawOffsets`，例如 `-12, 12`。

### `FA_Rune512_Moonlight_Reversed_Split`

这是 `月光反向连携 + 溅射/分裂卡` 的专用效果，不是普通溅射/分裂卡的 BaseFlow。

月光配方只需要检查共享 Tag：

| 字段 | 值 |
| --- | --- |
| `RequiredNeighborEffectTags` | `Card.Effect.SplashSplit` |
| `LinkFlow` | `FA_Rune512_Moonlight_Reversed_Split` |

| 字段 | 当前值 |
| --- | --- |
| `SplitProjectileCount` | `4` |
| `MaxSplitGenerations` | `1` |
| `SplitConeAngle` | `100` |
| `RandomizeSplitDirections` | 勾选 |
| `SplitRandomYawJitter` | `22` |
| `SplitRandomPitch` | `0` |
| `SplitDamageMultiplier` | `0.5` |
| `SplitSpeedMultiplier` | `1.6` |
| `SplitMaxDistanceMultiplier` | `1.25` |
| `SplitCollisionExtentMultiplier` | `0.5, 0.5, 0.5` |
| `BounceSplitChildrenOnEnemyHit` | 勾选 |
| `SplitChildMaxEnemyBounces` | `1` |
| `Destroy On World Static Hit` | 勾选 |

效果：月刃第一次碰到敌人或场景碰撞后，随机扇形分裂成 4 个小月刃。小月刃命中敌人后会按入射角反射弹射 1 次，并在弹射时刷新自身剩余生存时间。

## 武器池配置

当前批量生成器会自动处理默认武器：

| 武器 DA | 配置 |
| --- | --- |
| `/Game/Code/Weapon/TwoHandedSword/DA_WPN_THSword` | 使用 `DA_Rune512_Splash` |
| `/Game/Code/Weapon/GreatSword/DA_WPN_RedSword` | 使用 `DA_Rune512_Splash` |
| `/Game/Code/Weapon/Harquebus/DA_WPN_Harquebus` | 使用 `DA_Rune512_Split` |

后续新增武器时按规则配置：近战奖励池放 `DA_Rune512_Splash`，远程奖励池放 `DA_Rune512_Split`。

## 验收方式

1. 近战武器卡组中看到 `溅射`，远程武器卡组中看到 `分裂`。
2. 两个变体的 `CardIdTag` 都是 `Card.ID.SplashSplit`。
3. 两个变体都带 `Card.Effect.SplashSplit`。
4. 近战命中主目标时，周围敌人受到 20% 溅射伤害，主目标不吃二次溅射。
5. 远程开火时，除原始弹道外，额外出现左右两个弹道。
6. 普通 `溅射/分裂` 不出现月刃。
7. `月光(Reversed) -> Card.Effect.SplashSplit` 连携成功后，月刃碰撞会分裂成 4 个小月刃。
8. `月光(Forward) + Card.Effect.SplashSplit` 不触发月刃分裂。

## 注意事项

- 不要把月光分裂参数填到普通 `DA_Rune512_Splash` 或 `DA_Rune512_Split` 的 `BaseFlow` 中。
- 不要把近战和远程理解为两张设计卡。两个 DA 只是武器适配后的实现变体。
- 如果 `FA_Rune512_Split_Base` 里还能看到 `Spawn Slash Wave Projectile`，说明资产没有重新生成，需要运行 `RuneCardBatchGenerator -Apply`。
