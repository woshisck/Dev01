## 5. 输入绑定

所有 GA 通过 `AbilityInputID` / `InputAction` 触发，具体映射取决于项目输入系统。以下为建议绑定：

| 操作 | 建议输入 | GA | 激活条件 |
|---|---|---|---|
| 轻攻击 | 鼠标左键 / 手柄 R2 | `GA_Musket_LightAttack` | 有弹药，非瞄准/换弹中 |
| 重攻击（蓄力） | 鼠标右键 / 手柄 L2 长按 | `GA_Musket_HeavyAttack` | 有弹药，非换弹中；**松手触发发射** |
| 单发换弹 | R / 手柄 X | `GA_Musket_Reload_Single` | 弹药未满，非瞄准中 |
| 全换弹 | Shift+R / 手柄 Square | `GA_Musket_Reload_All` | 弹药未满，非瞄准中 |
| 冲刺攻击 | 冲刺中按轻攻击键 | `GA_Musket_SprintAttack` | **必须有 `Buff.Status.DashInvincible` Tag** |
| 冲刺换弹 | 冲刺中按换弹键 | `GA_Musket_SprintReload` | **必须有 `Buff.Status.DashInvincible` Tag** |

> **冲刺前置条件**：`GA_Musket_SprintAttack` 和 `GA_Musket_SprintReload` 的 `ActivationRequiredTags` 均包含 `Buff.Status.DashInvincible`。该 Tag 必须由冲刺 GA 的 `ActivationOwnedTags` 添加，冲刺结束时自动移除。若当前冲刺 GA 未添加此 Tag，两个冲刺相关 GA 将无法激活。

## 6. 瞄准弧材质 `M_AimArc`

**用途**：重攻击蓄力期间在地面显示扇形指示弧。

### 6.1 创建材质

1. Content Browser → 右键 → **Material** → 命名 `M_AimArc`
2. 双击打开材质编辑器

### 6.2 材质基础属性

| 属性 | 值 |
|---|---|
| Material Domain | **Deferred Decal** |
| Blend Mode | **Translucent** |
| Decal Blend Mode | **Translucent** |
| Two Sided | ✓（勾选，避免地形法线方向导致消失） |

### 6.3 材质参数（必须与 C++ 常量完全一致，区分大小写）

在材质编辑器中创建以下参数节点（右键 → Parameter）：

| 参数名 | 类型 | 默认值 | 用途 |
|---|---|---|---|
| `HalfAngle` | Scalar Parameter | `45.0` | 扇形半角（度），由 C++ 每帧更新 |
| `ArcRadius` | Scalar Parameter | `300.0` | 扇形半径（cm），可选，用于径向渐变 |
| `Softness` | Scalar Parameter | `0.05` | 边缘柔化宽度（弧度） |
| `Color` | Vector Parameter | `(1, 0.55, 0, 1)` | 弧颜色，橙色=普通蓄力，亮黄=满蓄 |

### 6.4 Custom 节点设置（核心遮罩逻辑）

1. 在材质编辑器中右键 → **Custom**，创建一个 Custom 节点
2. 按以下填写 Custom 节点属性：

| 属性 | 值 |
|---|---|
| Description | `AimArcMask` |
| Output Type | `CMOT Float 1` |
| Code | 见下方 |

**Code 字段内容：**
```hlsl
#include "/Project/MusketAimArc.ush"
return MusketAimArc_GetMask(UV, HalfAngle, Softness);
```

3. Custom 节点的 **Inputs** 添加以下引脚：

| 引脚名 | 连接来源 |
|---|---|
| `UV` | `TexCoord[0]` 节点 |
| `HalfAngle` | `HalfAngle` Scalar Parameter |
| `Softness` | `Softness` Scalar Parameter |

### 6.5 节点连接

```
TexCoord[0] ──────────────────── UV ─┐
HalfAngle Parameter ──────── HalfAngle ─┤ Custom(AimArcMask) → Float1 Mask
Softness Parameter ──────── Softness ─┘

Mask ─────────────────────────────────────────────────────── Opacity
Color Parameter × Mask ─────────────────────────────────── Emissive Color
```

> `.ush` 文件已存在于 `Shaders/MusketAimArc.ush`，`/Project` 虚拟路径已注册（commit `967eb8d9`），直接 include 即可。

## 8. BP_MusketBullet

**用途**：实现命中/消散视觉事件（特效、音效）。无此 Blueprint 时，C++ `AMusketBullet` 照常飞行并造成伤害。

### 创建步骤

1. Content Browser → 右键 → **Blueprint Class**
2. 父类搜索 `MusketBullet` → 选中 → 命名 `BP_MusketBullet`

### Class Defaults 填写

| 字段（分类 Projectile） | 值 |
|---|---|
| `Speed` | `2800`（cm/s） |
| `Lifetime` | `0.8`（秒） |

### 实现 Blueprint 事件（唯一需要写蓝图节点的地方）

在事件图谱（EventGraph）中实现：

**`BP_OnHitEnemy`**（由 C++ 在命中有效目标后调用）
- 参数：`HitActor`（AActor）、`HitLocation`（Vector）
- 建议节点：`Spawn System At Location`（Niagara 命中特效）+ `Play Sound at Location`

**`BP_OnMiss`**（子弹超时销毁时由 C++ 调用）
- 建议节点：`Spawn System At Location`（消散特效）

### 在 GA 中生效

创建 `BP_MusketBullet` 后，在角色 BP 或 GA CDO 中将 `BulletClass` 从 `AMusketBullet`（C++ 默认）改为 `BP_MusketBullet`。  
操作位置：各 GA 的 Class Defaults → `Musket|Config` → `BulletClass`。

## 9. WBP_AmmoCounter

**说明**：C++ 类 `UWBP_AmmoCounter` 已完成所有逻辑（属性绑定、图标创建、颜色刷新）。Blueprint 只需提供一个命名正确的 HorizontalBox 容器。

### 创建步骤

1. Content Browser → 右键 → **User Interface → Widget Blueprint**
2. 父类搜索 `WBP_AmmoCounter`（即 C++ 类 `UWBP_AmmoCounter`）→ 选中 → 命名 `WBP_AmmoCounter`

### Designer 布局

在 Designer 面板中，**只需放置一个控件**：

| 控件类型 | 命名（必须精确一致） | 说明 |
|---|---|---|
| `Horizontal Box` | `BulletIconBox` | C++ 会动态在此容器内创建弹药图标 |

> **命名必须精确为 `BulletIconBox`**（区分大小写），这是 C++ `BindWidget` 的绑定名。

### Class Defaults 可调参数（Designer 或蓝图 CDO 中调整）

| 参数 | 默认值 | 说明 |
|---|---|---|
| `IconSize` | `(22, 22)` | 单个图标尺寸（px） |
| `IconPadding` | `4.0` | 图标之间的间距 |
| `FilledColor` | `(1.0, 0.8, 0.1, 1.0)` 金色 | 有弹时图标颜色 |
| `EmptyColor` | `(0.2, 0.2, 0.2, 0.6)` 灰色 | 空仓时图标颜色 |

### 在 HUD 中使用

在 `WBP_HUD`（或 `YogHUD`）中将 `WBP_AmmoCounter` 添加为子控件并放置在屏幕角落。  
C++ 会在 `NativeConstruct` 时自动查找玩家 ASC，绑定 `CurrentAmmo` / `MaxAmmo` 属性变化并刷新图标。

> **无纹理图标**：目前图标使用 `RoundedBox` 实线色块，有弹=金色，空仓=灰色。有图标素材后，可在 C++ `RebuildIcons()` 中修改 `FSlateBrush` 的纹理引用。

- [ ] 开枪期间角色无法移动（LockMovement 生效）
- [ ] 开枪结束后恢复移动

### 重攻击蓄力（GA_Musket_HeavyAttack）
- [ ] 长按激活，子弹不立即发射
- [ ] M_AimArc 创建后：地面出现扇形弧，弧随时间收窄变长
- [ ] 蓄力满（1.8s）：弧变黄色，触发提示 Cue
- [ ] 松手：发射子弹，弧消失，角色恢复移动
- [ ] 满蓄时松手伤害 > 未满蓄时松手伤害

### 单发上弹（GA_Musket_Reload_Single）
- [ ] 弹药不满时可激活
- [ ] 弹药满时无法激活（GA 内部 return）
- [ ] 有蒙太奇后：每次蒙太奇结束加 1 发，循环到满
- [ ] 无蒙太奇时：瞬间补满（降级行为正常）
- [ ] 按轻攻击键时换弹中断，弹药保留已上数量

### 全换弹（GA_Musket_Reload_All）
- [ ] 弹药不满时可激活
- [ ] 播放蒙太奇（或瞬间），弹药一次性补满

### 冲刺扇射（GA_Musket_SprintAttack）
- [ ] 非冲刺状态无法激活
- [ ] 冲刺中激活：所有子弹扇形飞出（弹药数=6 时飞出 6 颗）
- [ ] 发射后弹药清零
- [ ] 冲刺 GA 被取消

### 冲刺换弹（GA_Musket_SprintReload）
- [ ] 非冲刺状态无法激活
- [ ] 冲刺中激活：弹药先清零，0.3s 后补满
- [ ] 冲刺 GA 继续运行（不被取消）
