# 月光连携符文配置说明

## 作用

配置“月光”作为战斗卡组中的 Link 卡。月光支持三种效果：

| 效果 | 触发条件 | 结果 |
| --- | --- | --- |
| 默认效果 | 没有触发连携 | 发射普通月光刃 |
| 连携 A | 前一张是 Attack 卡 | 发射强化月光刃，范围扩大、伤害提升 |
| 连携 B | 月光后一张是 Attack 卡 | 下一张攻击牌触发慢速月光刃 |

## 和旧版月光刃的关系

旧版 `1008 月光刃` 已经有一套可用资源：

- `GA_SlashWaveCounter`
- `BP_SlashWaveProjectile`
- `GE_SlashWaveDamage`
- `GameplayCue.Rune.SlashWaveFire`
- `GameplayCue.Rune.SlashWaveHit`

本次卡牌连携优先复用 `BP_SlashWaveProjectile`、`GE_SlashWaveDamage` 和表现 Cue。

`GA_SlashWaveCounter` 不建议直接接到月光卡牌连携上。它的逻辑是“监听事件并计数，默认每 3 次触发一次月光刃”，适合旧版被动符文；本次月光卡牌需要“出牌时立即按不同连携形态发射”，所以在 FA 里使用 `Spawn Slash Wave Projectile` 节点直接生成同一个月光刃投射物。

## RuneDataAsset 配置位置

打开月光的 `RuneDataAsset`：

`RuneInfo -> CombatCard`

## 基础字段

| 字段                        | 配置                         |
| ------------------------- | -------------------------- |
| `Is Combat Card`          | 勾选                         |
| `Card Type`               | `Link`                     |
| `Required Action`         | `Any`                      |
| `Trigger Timing`          | `On Commit`                |
| `Card Tags`               | 建议添加 `Card.Link.Moonlight` |
| `Base Flow`               | `FA_Moonlight_Base`        |
| `Matched Flow`            | 留空                         |
| `Link Mode`               | `None`                     |
| `Requires Combo Finisher` | 不勾选                        |
| `Display Name`            | 月光                         |
| `HUDReason Text`          | 月光连携                       |

## LinkConfig 配置

`LinkConfig` 中配置：

| 字段                                               | 配置                      |
| ------------------------------------------------ | ----------------------- |
| `Direction`                                      | `Both`                  |
| `ForwardEffect.Condition.RequiredNeighborTypes`  | 添加 `Attack`             |
| `ForwardEffect.Flow`                             | `FA_Moonlight_Forward`  |
| `ForwardEffect.Multiplier`                       | 1.0                     |
| `ForwardEffect.ReasonText`                       | 前一张攻击牌强化月光              |
| `BackwardEffect.Condition.RequiredNeighborTypes` | 添加 `Attack`             |
| `BackwardEffect.Flow`                            | `FA_Moonlight_Backward` |
| `BackwardEffect.Multiplier`                      | 1.0                     |
| `BackwardEffect.ReasonText`                      | 月光赋能下一张攻击牌              |
| `BreakFlow`                                      | 暂时留空                    |
| `BreakPolicy`                                    | 暂时保持默认，不作为本次验收重点        |

`Direction = Both` 时，运行规则是前向优先：

- 如果前一张是 Attack，月光直接触发连携 A，不再等待后一张触发连携 B。
- 如果前一张不是 Attack，月光先打出默认效果，并等待后一张 Attack 触发连携 B。

## FA 配置

需要做三个 FA：

```text
FA_Moonlight_Base
FA_Moonlight_Forward
FA_Moonlight_Backward
```

三个 FA 都使用节点：

```text
Start -> Spawn Slash Wave Projectile
```

## FA_Moonlight_Base

普通月光刃：

| 字段                     | 推荐值                              |
| ---------------------- | -------------------------------- |
| `Projectile Class`     | 月光刃 BP，父类为 `SlashWaveProjectile` |
| `Damage Effect`        | `GE_SlashWaveDamage`             |
| `Damage`               | 按当前测试需要填写                        |
| `Speed`                | 1400                             |
| `Max Distance`         | 800                              |
| `Max Hit Count`        | 2                                |
| `Collision Box Extent` | 例如 `(30, 60, 35)`                |
| `Spawn Offset`         | 例如 `(80, 0, 45)`                 |

说明：

- `Max Hit Count = 2` 表示碰到两个敌人后消失。
- 如果玩家贴着敌人攻击，`Spawn Offset` 会让月光刃从玩家前方生成，仍能看到飞出去的弹道。

## FA_Moonlight_Forward

连携 A，强化月光刃：

| 字段 | 推荐值 |
| --- | --- |
| `Projectile Class` | 月光刃 BP |
| `Damage Effect` | `GE_SlashWaveDamage` |
| `Damage` | 高于默认值 |
| `Speed` | 1400 |
| `Max Distance` | 800 |
| `Max Hit Count` | 2 |
| `Collision Box Extent` | 比默认更大，例如 `(45, 90, 45)` |
| `Spawn Offset` | 例如 `(80, 0, 45)` |

## FA_Moonlight_Backward

连携 B，慢速短距离月光刃：

| 字段 | 推荐值 |
| --- | --- |
| `Projectile Class` | 月光刃 BP |
| `Damage Effect` | `GE_SlashWaveDamage` |
| `Damage` | 按连携 B 需要填写 |
| `Speed` | 200 |
| `Max Distance` | 200 |
| `Max Hit Count` | 0 |
| `Collision Box Extent` | 例如 `(30, 60, 35)` |
| `Spawn Offset` | 例如 `(80, 0, 45)` |

说明：

- `Max Hit Count = 0` 表示没有碰撞血量限制。
- 同一个敌人仍只会被同一道月光刃命中一次。
- `Max Distance = 200` 且 `Speed = 200` 时，月光刃约 1 秒后消失。

## WBP 说明文本

如果需要在卡牌浮窗解释月光效果，建议使用 `Yog Common Rich Text Block`：

```text
<key>正向连携</key>：前一张为攻击牌时，月光刃范围扩大、伤害提升。
<key>反向连携</key>：后一张为攻击牌时，触发慢速短距离月光刃。
```

## 测试方式

1. 配置卡组顺序：`Attack -> Moonlight`，攻击两次，确认第二次触发连携 A。
2. 配置卡组顺序：`Moonlight -> Attack`，攻击两次，确认第二次触发连携 B。
3. 单独打出 `Moonlight`，确认触发默认月光刃。
4. 默认和连携 A 的月光刃飞行约 800 距离。
5. 默认和连携 A 命中两个敌人后消失。
6. 连携 B 月光刃速度更慢、飞行约 200 距离，约 1 秒后消失。
