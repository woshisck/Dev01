# 月光连携符文配置说明

## 1. 作用

月光是 `Link` 卡。它通过 `LinkRecipes` 根据邻近卡牌的 `CardIdTag` 或 `CardEffectTags` 选择不同 LinkFlow。

当前重点规则：

1. 月光本体效果放在 `BaseFlow`。
2. 月光连携效果放在各自 `LinkFlow`。
3. 连携特效放在 FA 的独立 `Play Niagara` 节点，不放在 CombatCard 字段里。
4. 月刃投射物节点只负责发射、伤害和命中事件。
5. 基础月光和正向连携月光会读取连招次数增加月光数量；额外月光沿同一路径连续发射，不做扇形散射。

## 2. DA 配置

位置：

```text
RuneDataAsset -> RuneInfo -> CombatCard
```

| 字段 | 值 |
| --- | --- |
| `bIsCombatCard` | 勾选 |
| `CardType` | `Link` |
| `CardIdTag` | `Card.ID.Moonlight` |
| `CardEffectTags` | `Card.Effect.Moonlight` |
| `RequiredAction` | `Any` |
| `TriggerTiming` | `On Hit` |
| `BaseFlow` | `FA_Rune512_Moonlight_Base` |
| `DefaultLinkOrientation` | 默认 `Forward`；反向测试时设为 `Reversed` |
| `DisplayName` | `月光` |

卡牌应在攻击 AN 命中阶段打出，不在蒙太奇刚开始时打出。

## 3. LinkRecipes

示例：月光 + 攻击效果卡。

| 字段 | 正向 | 反向 |
| --- | --- | --- |
| `Direction` | `Forward` | `Reversed` |
| `RequiredNeighborEffectTags` | `Card.Effect.Attack` | `Card.Effect.Attack` |
| `RequiredNeighborIdTags` | 可留空 | 可留空 |
| `LinkFlow` | `FA_Rune512_Moonlight_Forward_Attack` | `FA_Rune512_Moonlight_Reversed_Attack` |
| `Multiplier` | 按需要填写 | 按需要填写 |
| `HUDText` | 月光强化 | 反向月光 |

如果要求“必须是某张指定卡”，填 `RequiredNeighborIdTags`。如果要求“任意攻击效果卡”，填 `RequiredNeighborEffectTags=Card.Effect.Attack`。

## 4. 月光 + 燃烧

Flow：

```text
FA_Rune512_Moonlight_Forward_Burn
FA_Rune512_Moonlight_Reversed_Burn
```

节点链：

```text
Spawn Slash Wave Projectile
  -> Wait Gameplay Event(Action.Rune.MoonlightBurnHit)
  -> Play Niagara(Rune.Moonlight.BurnHitNiagara)
  -> Apply Gameplay Effect Class(UGE_RuneBurn)
```

关键字段：

| 节点 | 字段 | 值 |
| --- | --- | --- |
| `Spawn Slash Wave Projectile` | `Hit Gameplay Event Tag` | `Action.Rune.MoonlightBurnHit` |
| `Spawn Slash Wave Projectile` | `Launch/Hit/Expire Niagara` | `None` |
| `Play Niagara` | `Niagara System` | `NS_Fire_Floor` |
| `Play Niagara` | `Attach Target` | `LastDamageTarget` |
| `Play Niagara` | `Attach Socket Name` | `spine_03` |
| `Play Niagara` | `Scale` | `(0.28,0.28,0.28)` |
| `Play Niagara` | `Lifetime` | `3.2` |
| `Apply Gameplay Effect Class` | `Effect` | `UGE_RuneBurn` |
| `Apply Gameplay Effect Class` | `Target` | `LastDamageTarget` |
| `Apply Gameplay Effect Class` | `bRemoveEffectOnCleanup` | 不勾选 |
| `Apply Gameplay Effect Class` | `SetByCallerTag1 / Value1` | `Data.Damage.Burn / 6` |

## 5. 月光 + 中毒

Flow：

```text
FA_Rune512_Moonlight_Forward_Poison
FA_Rune512_Moonlight_Reversed_Poison
```

节点链：

```text
Spawn Slash Wave Projectile
  -> Wait Gameplay Event(Action.Rune.MoonlightPoisonHit)
  -> Play Niagara(Rune.Moonlight.PoisonHitNiagara)
  -> Apply Gameplay Effect Class(GE_Poison)
  -> Play Niagara(Rune.Moonlight.PoisonSpreadNiagara)
  -> ApplyGEInRadius(GE_PoisonSplash/GE_Poison)
```

详细字段见 `月光毒素连携FA配置说明.md`。

## 6. 测试

1. `Moonlight -> Moonlight`：不触发连携，月光走 `BaseFlow`。
2. `Attack(Card.Effect.Attack) -> Moonlight(Forward)`：触发 `FA_Rune512_Moonlight_Forward_Attack`。
3. `Moonlight(Reversed) -> Attack(Card.Effect.Attack)`：连招有效时触发 `FA_Rune512_Moonlight_Reversed_Attack`。
4. `Moonlight(Reversed)` 后如果玩家停顿、重新起手，或战斗系统判定连招无效，下一张攻击不触发反向连携。
5. 燃烧/中毒连携成功时，日志应出现 `[PlayNiagara] Spawned`，并且 `Spawn Slash Wave Projectile` 内置 Niagara 字段保持 `None`。
6. 正向连携在 Combo2/Combo3 时，月光应表现为同一路径连续发射的 2/3 道月光；每一道命中都会独立触发对应的命中事件和后续连携效果。
7. 分裂月光仍表现为多方向弹道，不使用连续同路径月光数量加成。

## 7. 当前补充规则：反向毒/火走地面路径

当前版本中，`FA_Rune512_Moonlight_Reversed_Poison` 和 `FA_Rune512_Moonlight_Reversed_Burn` 不再表现为普通月光刃。

| Flow | 当前表现 | 关键节点 |
| --- | --- | --- |
| `FA_Rune512_Moonlight_Reversed_Poison` | 玩家前方留下毒路径 | `Spawn Rune Ground Path Effect` |
| `FA_Rune512_Moonlight_Reversed_Burn` | 玩家前方留下火路径 | `Spawn Rune Ground Path Effect` |

反向月光被打出后，只有在连招仍有效且下一张卡满足配方时才触发反向 `LinkFlow`。如果连招断开或下一张卡不匹配，按 `BreakPolicy=ReleaseBaseFlow` 释放普通月光。

## 8. 月光 + 分裂当前规则

`分裂` 已拆为远程普通卡，普通分裂不再发射月刃。

| 组合 | 当前结果 |
| --- | --- |
| `Moonlight(Forward) + Card.Effect.Split` | 不触发特殊分裂连携 |
| `Moonlight(Reversed) + Card.Effect.Split` | 触发 `FA_Rune512_Moonlight_Reversed_Split` |

`FA_Rune512_Moonlight_Reversed_Split` 的行为是：先发射 1 个普通月刃，月刃第一次碰到敌人或场景后分裂成 4 个小月刃。

关键参数：

| 字段 | 当前值 |
| --- | --- |
| `SplitProjectileCount` | `4` |
| `MaxSplitGenerations` | `1` |
| `SplitConeAngle` | `70` |
| `SplitDamageMultiplier` | `0.5` |
| `SplitSpeedMultiplier` | `1.6` |
| `SplitMaxDistanceMultiplier` | `0.55` |
| `SplitCollisionExtentMultiplier` | `0.5, 0.5, 0.5` |
| `Destroy On World Static Hit` | 勾选 |
