# 月光连携符文配置说明

## 作用

月光是 `Link` 卡。它通过 `LinkRecipes` 根据前后卡牌的效果 Tag 切换效果。

## DA 配置

位置：`RuneDataAsset -> RuneInfo -> CombatCard`

| 字段 | 值 |
| --- | --- |
| `bIsCombatCard` | 勾选 |
| `CardType` | `Link` |
| `CardIdTag` | `Card.ID.Moonlight` |
| `CardEffectTags` | `Card.Effect.Moonlight` |
| `RequiredAction` | `Any` |
| `TriggerTiming` | `OnCommit` |
| `BaseFlow` | `FA_Moonlight_Base` |
| `DefaultLinkOrientation` | 默认 `Forward`；反向测试可改 `Reversed` |
| `DisplayName` | 月光 |

## LinkRecipes

正向配方：

| 字段                                     | 值                      |
| -------------------------------------- | ---------------------- |
| `Direction`                            | `Forward`              |
| `Condition.RequiredNeighborEffectTags` | `Card.Effect.Attack`   |
| `LinkFlow`                             | `FA_Moonlight_Forward` |
| `ReasonText`                           | 攻击牌强化月光                |

反向配方：

| 字段                                     | 值                       |
| -------------------------------------- | ----------------------- |
| `Direction`                            | `Reversed`              |
| `Condition.RequiredNeighborEffectTags` | `Card.Effect.Attack`    |
| `LinkFlow`                             | `FA_Moonlight_Backward` |
| `ReasonText`                           | 月光赋能下一张攻击牌              |

## 月光 FA 参数

| FA | 距离 | 速度 | 命中数 | 单目标伤害次数 | 碰撞范围 | 用途 |
| --- | --- | --- | --- | --- | --- | --- |
| `FA_Moonlight_Base` | 800 | 1400 | 2 | 1 | `(30,60,35)` | 默认月光刃 |
| `FA_Moonlight_Forward` | 800 | 1400 | 2 | 1 | `(45,90,45)` 或更大 | 强化月光刃 |
| `FA_Moonlight_Backward` | 120 | 120 | 0 | 3 | `(75,150,65)` 或更大 | 巨大慢速月光，只在玩家面前出现 |

`Max Hit Count = 0` 表示没有命中目标数量上限。反向连携用 `Damage Applications Per Target = 3` 表示同一道巨大月光对同一目标造成 3 次伤害。

`FA_Moonlight_Forward` 的 `Spawn Slash Wave Projectile` 节点需要勾选：

```text
Scale Visual With Collision Extent = true
Visual Scale Multiplier = (1, 1, 1)
```

这样当 `Collision Box Extent` 从 `(30,60,35)` 调整到 `(45,90,45)` 时，碰撞范围和月光刃视觉会一起变大。`Collision Box Extent` 仍然代表最终碰撞范围，不会因为视觉缩放被二次放大。

## 反向连携巨大月光

`FA_Moonlight_Backward` 的 `Spawn Slash Wave Projectile` 推荐配置：

```text
Damage = 按单跳伤害填写
Speed = 120
Max Distance = 120
Max Hit Count = 0
Damage Applications Per Target = 3
Damage Application Interval = 0.25
Collision Box Extent = (75, 150, 65)
Scale Visual With Collision Extent = true
Visual Scale Multiplier = (1, 1, 1)
Spawn Offset = (100, 0, 45)
DamageLogType = Rune_Moonlight_Reversed
```

解释：

- `Speed` 和 `Max Distance` 都较小，所以月光只会在玩家面前短距离缓慢出现。
- `Collision Box Extent` 放大后，月光视觉也会同步变大。
- `Damage Applications Per Target = 3` 会让同一目标被这道月光伤害 3 次，每次间隔 `0.25` 秒。

## 月光 DamageEffect GE

`Spawn Slash Wave Projectile` 节点里的 `DamageEffect` 应填一个 Instant 伤害 GE，例如 `GE_SlashWaveDamage`。

如果项目里没有这个 GE，创建方式：

| 配置项 | 值 |
| --- | --- |
| 父类 | `GameplayEffect` |
| `Duration Policy` | `Instant` |
| `Modifiers[0].Attribute` | `DamageAttributeSet.DamagePure` |
| `Modifiers[0].Modifier Op` | `Add` |
| `Modifiers[0].Magnitude Calculation Type` | `Set By Caller` |
| `Set By Caller Data Tag` | `Attribute.ActDamage` |

说明：

- 月光投射物会把 FA 节点里的 `Damage` 写入 `Attribute.ActDamage`。
- GE 只负责把这个值写到 `DamagePure`。
- 如果 `DamageEffect` 留空，代码会临时创建一个瞬发伤害 GE，能造成伤害，但不利于后续统一配置伤害类型、命中特效和数值调试。

## 测试

1. `Moonlight -> Moonlight`：不触发连携，月光走默认 `BaseFlow`。
2. `Attack(Card.Effect.Attack) -> Moonlight(Forward)`：月光触发 `FA_Moonlight_Forward`。
3. `Moonlight(Reversed) -> Attack(Card.Effect.Attack)`：下一张攻击触发 `FA_Moonlight_Backward`。
