# 月光连携符文配置说明

## 作用

月光是 `Link` 卡。它通过 `LinkRecipes` 根据前后卡牌的效果 Tag 切换不同 FA。

## DA 配置

位置：`RuneDataAsset -> RuneInfo -> CombatCard`

| 字段 | 值 |
| --- | --- |
| `bIsCombatCard` | 勾选 |
| `CardType` | `Link` |
| `CardIdTag` | `Card.ID.Moonlight` |
| `CardEffectTags` | `Card.Effect.Moonlight` |
| `RequiredAction` | `Any` |
| `TriggerTiming` | `On Hit` |
| `BaseFlow` | `FA_Rune512_Moonlight_Base` |
| `DefaultLinkOrientation` | 默认 `Forward`；反向测试设为 `Reversed` |
| `DisplayName` | 月光 |

卡牌在 `AN_MeleeDamage` 生效后才会打出，不在蒙太奇开始时打出。

## LinkRecipes

正向：攻击牌 -> 月光

| 字段 | 值 |
| --- | --- |
| `Direction` | `Forward` |
| `Condition.RequiredNeighborEffectTags` | `Card.Effect.Attack` |
| `LinkFlow` | `FA_Rune512_Moonlight_Forward_Attack` |
| `ReasonText` | 攻击牌强化月光 |

反向：月光 -> 攻击牌

| 字段 | 值 |
| --- | --- |
| `Direction` | `Reversed` |
| `Condition.RequiredNeighborEffectTags` | `Card.Effect.Attack` |
| `LinkFlow` | `FA_Rune512_Moonlight_Reversed_Attack` |
| `ReasonText` | 月光赋能下一张攻击牌 |

## 月光 FA 参数

配置位置：打开对应 FA，选中 `Spawn Slash Wave Projectile` 节点。

| FA | 距离 | 速度 | 命中数 | 单目标伤害次数 | 碰撞范围 | 视觉倍率 | Projectile Visual Niagara |
| --- | --- | --- | --- | --- | --- | --- | --- |
| `FA_Rune512_Moonlight_Base` | 800 | 1400 | 2 | 1 | `(30,60,35)` | `(1,1,1)` | `NS_Free_Magic_Slash` |
| `FA_Rune512_Moonlight_Forward_Attack` | 800 | 1400 | 2 | 1 | `(60,120,55)` | `(1.35,1.35,1.2)` | `NS_Free_Magic_Slash2` |
| `FA_Rune512_Moonlight_Reversed_Attack` | 200 | 180 | 0 | 3 | `(140,260,90)` | `(2.1,2.1,1.5)` | `NS_Free_Magic_Area2` |

`Max Hit Count = 0` 表示没有命中目标数量上限。反向连携用 `Damage Applications Per Target = 3`，表示同一道巨大月光对同一目标造成 3 次伤害。

## 特效

月光弹道、碰撞、伤害和刀光主体都配置在 `Spawn Slash Wave Projectile` 节点里。

| 字段 | 用途 |
| --- | --- |
| `Projectile Visual Niagara System` | 月光飞行刀光主体 Niagara |
| `Projectile Visual Niagara Scale` | 刀光主体缩放 |
| `bHideDefaultProjectileVisuals` | 勾选后隐藏 `BP_SlashWaveProjectile` 旧视觉组件，避免旧刀光和 Niagara 同时出现 |
| `Launch Niagara System` | 只用于额外的一次性发射闪光；月光主体已用 `Projectile Visual Niagara System` 时建议留空 |

如果只是额外做一个独立闪光、命中特效或持续环绕效果，再单独在 FA 里添加 `Play Niagara` 节点。

## DamageEffect GE

`Spawn Slash Wave Projectile` 节点里的 `DamageEffect` 建议填写项目已有的月光伤害 GE，例如 `GE_MoonWave`。伤害数值优先看节点里的 `Damage` 字段。

如果项目里没有可用 GE，创建一个 `GameplayEffect`：

| 配置项 | 值 |
| --- | --- |
| `Duration Policy` | `Instant` |
| `Modifiers[0].Attribute` | `DamageAttributeSet.DamagePure` |
| `Modifiers[0].Modifier Op` | `Add` |
| `Modifiers[0].Magnitude Calculation Type` | `Set By Caller` |
| `Set By Caller Data Tag` | `Attribute.ActDamage` |

## 测试

1. `Moonlight -> Moonlight`：不触发连携，月光走默认 `BaseFlow`。
2. `Attack(Card.Effect.Attack) -> Moonlight(Forward)`：触发 `FA_Rune512_Moonlight_Forward_Attack`，范围和视觉都变大。
3. `Moonlight(Reversed) -> Attack(Card.Effect.Attack)`：连招有效时触发 `FA_Rune512_Moonlight_Reversed_Attack`，巨大慢速月光造成 3 次伤害。
4. `Moonlight(Reversed)` 后如果玩家停顿、重新起手，或战斗系统判定连招无效，下一张攻击不触发反向连携，只执行攻击牌自己的 `BaseFlow`。
