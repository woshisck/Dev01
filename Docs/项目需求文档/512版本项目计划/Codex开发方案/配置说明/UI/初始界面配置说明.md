# 初始界面配置说明

## 资产入口

- 源图文件：`SourceArt/UI/Frontend/T_MainMenu_Dungeon.png`
- 导入纹理：`/Game/UI/Playtest_UI/UI_Tex/Frontend/T_MainMenu_Dungeon`
- 运行入口：`UYogGameInstanceBase::ShowMainMenu()`
- 导入命令let：`MainUISetupCommandlet`

## 界面结构

- 背景使用全屏 `SImage`，并通过 `ScaleToFill` 适配 16:9 屏幕。
- 菜单文字由 Slate 按钮渲染，不依赖图片文字，便于后续本地化和手柄焦点管理。
- 当前菜单项为 `Start / Continue / Options / Quit`。
- 左下角固定显示按键提示：`A Select / D-Pad / Left Stick Navigate`。

## 手柄适配

- 进入初始界面后默认焦点设置到 `Start`。
- 手柄 `A` 或键盘 `Enter` 激活当前按钮。
- 十字键或左摇杆上下移动按钮焦点。
- 鼠标仍可点击按钮；初始界面使用 `UIOnly` 输入模式。

## 生成流程

1. 编译 `DevKitEditor`。
2. 运行 `UnrealEditor-Cmd.exe DevKit.uproject -run=MainUISetup -unattended -nop4 -nosplash` 检查报告。
3. 运行 `-Apply` 导入或重导入 `T_MainMenu_Dungeon.png`。
4. 报告输出到 `Saved/MainUISetupReport.md`。
