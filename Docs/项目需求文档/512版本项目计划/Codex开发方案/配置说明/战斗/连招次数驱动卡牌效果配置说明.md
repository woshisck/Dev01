# 连招次数驱动卡牌效果配置说明

## 用途

让卡牌效果可以读取当前攻击的连招次数。默认旧卡不受影响，只有显式开启配置的卡牌或 BuffFlow 节点才会吃连招加成。

## 战斗侧配置

位置：`RuneDataAsset -> CombatCard -> Combat Card|Combo Scaling`

- `bUseComboEffectScaling`：是否让该卡牌按连招次数计算效果倍率。
- `ComboScalarPerIndex`：每层连招加成。连招层数为 `max(0, ComboIndex - 1)`。
- `MaxComboScalar`：最大加成上限。

推荐攻击卡 V1：

- `bUseComboEffectScaling = true`
- `ComboScalarPerIndex = 0.25`
- `MaxComboScalar = 0.5`

结果：

- Combo1：1.0 倍
- Combo2：1.25 倍
- Combo3 及以上：1.5 倍

如果卡牌同时触发 Link 倍率，最终倍率为：

`最终倍率 = Link倍率 * 连招倍率`

### 已批量配置的 512 资产

- `/Game/Docs/BuffDocs/V2-RuneCard/512Generated/DA_Rune512_Attack`
  - `bUseComboEffectScaling = true`
  - `ComboScalarPerIndex = 0.25`
  - `MaxComboScalar = 0.5`
- `/Game/Docs/BuffDocs/V2-RuneCard/512Generated/Flow/FA_Rune512_Attack_Base`
  - `Apply Attribute Modifier -> Use Combat Card Effect Multiplier = true`

其他普通卡暂时保持默认，不主动吃连招倍率，避免旧配置和特殊卡牌被连招数意外放大。

## BuffFlow 节点配置

### Apply Attribute Modifier

位置：节点 `Apply Attribute Modifier -> BuffFlow|Combat Card`

- `Use Combat Card Effect Multiplier`：开启后，节点的 `Value` 会乘以当前卡牌最终倍率。

适合攻击力、攻击倍率、护甲、治疗等数值型卡牌。

### Spawn Slash Wave Projectile

位置：节点 `Spawn Slash Wave Projectile -> Slash Wave|Combat Card`

- `Add Combo Stacks To Projectile Count`：开启后，根据连招层数增加投射物数量。
- `Projectiles Per Combo Stack`：每层连招增加几个投射物。
- `Max Bonus Projectiles`：最多额外增加几个投射物。
- `Spawn Projectiles Sequentially`：开启后，额外月光沿同一路径按时间间隔连续发射。
- `Sequential Projectile Spawn Interval`：连续发射间隔，推荐 `0.12`。

推荐月光 V1：

- `ProjectileCount = 1`
- `Add Combo Stacks To Projectile Count = true`
- `Projectiles Per Combo Stack = 1`
- `Max Bonus Projectiles = 2`
- `ProjectileConeAngleDegrees = 0`
- `Spawn Projectiles Sequentially = true`
- `Sequential Projectile Spawn Interval = 0.12`

结果：

- Combo1：1 个
- Combo2：2 个，同一路径连续发射
- Combo3 及以上：3 个，同一路径连续发射

### 已批量配置的 512 资产

- `/Game/Docs/BuffDocs/V2-RuneCard/512Generated/Flow/FA_Rune512_Moonlight_Base`
  - `ProjectileCount = 1`
  - `Add Combo Stacks To Projectile Count = true`
  - `Projectiles Per Combo Stack = 1`
  - `Max Bonus Projectiles = 2`
  - `ProjectileConeAngleDegrees = 0`
  - `Spawn Projectiles Sequentially = true`
  - `Sequential Projectile Spawn Interval = 0.12`

正向月光连携 Flow 也会吃连招次数加成，因为正向连携通常仍保留月光弹道形式。每一道连续发射的月光都会独立命中、独立发送命中事件，因此燃烧、中毒、穿透等正向连携效果会随多道月光多次触发。

不启用连招数量加成的情况：

- 反向月光连携：当前反向火/毒是地面路径，反向攻击是巨大慢速月光，不走连招额外月光数量。
- 分裂月光：分裂本身就是多方向弹道，继续使用 `ProjectileConeAngleDegrees` 和 Split 配置，不开启连续同路径月光。

## 注意

- `ComboIndex` 由战斗连招运行时提供，符文侧不要反查战斗状态。
- 普通旧卡默认不吃倍率。
- 如果某张卡只想改变月光数量，不需要开启卡牌的 `bUseComboEffectScaling`，只开启投射物节点的连招数量配置即可。
