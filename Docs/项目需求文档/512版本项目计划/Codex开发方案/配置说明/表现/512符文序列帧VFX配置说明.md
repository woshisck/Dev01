# 512 符文序列帧 VFX 配置说明

## 当前状态

序列帧 VFX 已保留为备用方案，当前 512 符文验收默认不再使用 `Play Flipbook VFX`。

原因：本轮已按验收反馈恢复 Niagara 表现。当前生成器会清空生成 FA 中的 Flipbook 节点资源引用，避免旧序列帧表现和 Niagara 重复出现。

## 材质备用配置

材质：

```text
/Game/Docs/BuffDocs/V2-RuneCard/VFX/Materials/M_Rune512_FlipbookSprite
```

`.ush` 路径：

```text
Shaders/Rune512FlipbookSprite.ush
```

Custom Node Include：

```text
/Project/Rune512FlipbookSprite.ush
```

Custom Node 输入：

| 输入 | 类型/来源 |
| --- | --- |
| `FlipbookTexture` | Texture Object 参数 |
| `UV` | Texture Coordinate |
| `Time` | Scalar 参数 |
| `Rows` | Scalar 参数 |
| `Columns` | Scalar 参数 |
| `Duration` | Scalar 参数 |
| `EmissiveColor` | Vector 参数 |
| `AlphaScale` | Scalar 参数 |

Custom Node 输出：

```text
float4 RGBA
```

材质设置：

| 项 | 值 |
| --- | --- |
| `Shading Model` | `Unlit` |
| `Blend Mode` | `Translucent` 或 `Additive` |
| `Two Sided` | 勾选 |
| 深度写入 | 关闭 |

## 如果后续重新启用

需要在 FA 中手动恢复 `Play Flipbook VFX` 节点的资源引用：

| 字段 | 推荐 |
| --- | --- |
| `Texture` | 对应 `T_Rune512_VFX_*` |
| `Material` | `M_Rune512_FlipbookSprite` |
| `Rows / Columns` | `4 / 4` |
| `Target` | `LastDamageTarget` |
| `bProjectToVisibleSurface` | 勾选 |

当前不建议和 `Play Niagara` 同时启用同一个命中特效，否则表现会重复。
