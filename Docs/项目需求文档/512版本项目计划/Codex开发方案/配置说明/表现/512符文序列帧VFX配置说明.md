# 512 符文序列帧 VFX 配置说明

本次先做核心验收用小尺寸序列帧，目的是替代过大的 Niagara 状态表现，方便观察符文命中和扩散。

## 1. 源文件

源图路径：

```text
SourceArt/512RuneVFX/PreviewFrames
```

已生成 6 张 4x4 透明 PNG 图集：

| 文件 | 用途 |
| --- | --- |
| `T_Rune512_VFX_MoonBlade_Flight.png` | 月刃飞行预览 |
| `T_Rune512_VFX_Moonlight_Hit.png` | 月光命中 |
| `T_Rune512_VFX_Poison_Hit.png` | 毒素命中 |
| `T_Rune512_VFX_Poison_Spread.png` | 毒素扩散 |
| `T_Rune512_VFX_Burn_Hit.png` | 燃烧命中 |
| `T_Rune512_VFX_ShieldPierce.png` | 护盾/穿透提示 |

导入目标：

```text
/Game/Docs/BuffDocs/V2-RuneCard/VFX/Textures
```

## 2. 材质

材质资产：

```text
/Game/Docs/BuffDocs/V2-RuneCard/VFX/Materials/M_Rune512_FlipbookSprite
```

材质由批生成器自动创建。需要手动检查时按下面参数确认。

| 项 | 值 |
| --- | --- |
| `Shading Model` | `Unlit` |
| `Blend Mode` | `Translucent`，需要更亮时可改 `Additive` |
| `Two Sided` | 勾选 |
| `Disable Depth Test` | 不勾选 |
| 深度写入 | 关闭，半透明材质默认不写深度 |

## 3. Custom Node

`.ush` 路径：

```text
Shaders/Rune512FlipbookSprite.ush
```

Custom Node Include：

```text
/Project/Rune512FlipbookSprite.ush
```

Custom Node 输入：

| 输入 | 类型/来源 |
| --- | --- |
| `FlipbookTexture` | Texture Object 参数 |
| `UV` | Texture Coordinate |
| `Time` | Scalar 参数 |
| `Rows` | Scalar 参数 |
| `Columns` | Scalar 参数 |
| `Duration` | Scalar 参数 |
| `EmissiveColor` | Vector 参数 |
| `AlphaScale` | Scalar 参数 |

Custom Node 输出：

```text
float4 RGBA
```

Custom Node 代码：

```hlsl
return Rune512FlipbookSpriteMain(FlipbookTexture, FlipbookTextureSampler, UV, Time, Rows, Columns, Duration, EmissiveColor, AlphaScale);
```

材质连接：

| Custom 输出 | 连接到 |
| --- | --- |
| `RGB` | `Emissive Color` |
| `A` | `Opacity` |

## 4. 批生成器

运行：

```text
UnrealEditor-Cmd.exe DevKit.uproject -run=RuneCardBatchGenerator -Apply -unattended -nop4 -nosplash
```

批生成器会自动：

1. 导入 6 张图集。
2. 创建 `M_Rune512_FlipbookSprite`。
3. 把毒素月光 LinkFlow 的 `Play Flipbook VFX` 节点指向对应贴图和材质。

## 5. 验收

打开材质后确认没有编译报错。打开毒素月光 LinkFlow，`Play Flipbook VFX` 节点的 `Texture` 和 `Material` 不应为空。
