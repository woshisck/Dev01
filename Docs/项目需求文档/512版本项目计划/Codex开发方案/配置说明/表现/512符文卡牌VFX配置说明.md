# 512 符文卡牌 VFX 配置说明

卡牌表现跟随 FA 配置：普通效果放 `BaseFlow`，连携效果放对应 `LinkFlow`。不要把毒、燃烧、命中表现继续写进一个投射物节点里；投射物节点只负责生成月刃、造成基础伤害、发送命中事件。

## 1. 新增节点

节点名称：`Play Flipbook VFX`

用途：在 FA 中播放小尺寸序列帧面片特效，用于命中、扩散、状态提示等短表现。

| 字段 | 推荐填写 | 说明 |
| --- | --- | --- |
| `Texture` | `T_Rune512_VFX_Poison_Hit` 等 | 4x4 序列帧贴图 |
| `Material` | `M_Rune512_FlipbookSprite` | 自动生成在 `/Game/Docs/BuffDocs/V2-RuneCard/VFX/Materials` |
| `Rows` | `4` | 图集行数 |
| `Columns` | `4` | 图集列数 |
| `Duration` | `0.35-0.50` | 播放时长 |
| `Size` | `72-180` | 面片尺寸，单位 cm；命中挂身特效优先 72-82，范围扩散再放大 |
| `Target` | `LastDamageTarget` | 命中特效通常挂命中的敌人 |
| `Socket` | `spine_03` 或留空 | 挂敌人身体时填骨骼，范围扩散可留空 |
| `Socket Fallback Names` | `spine_02, spine_01, pelvis, root` | 找不到 Socket 时自动回退 |
| `Offset` | `(0,0,8)` 或 `(0,0,18)` | 在目标位置上方微调 |
| `bFaceCamera` | `true` | 面片始终朝向相机 |
| `bDestroyWithFlow` | `false` | 一次性命中特效推荐 false |

## 3. 月刃节点职责

`Spawn Slash Wave Projectile` 仍负责月刃主体：

| 字段 | 说明 |
| --- | --- |
| `Projectile Visual Niagara System` | 512 当前验收版填 `None`；月刃主体使用 BP/投射物默认可视，避免旧 Niagara 过大和重复显示 |
| `Damage / Speed / Max Distance` | 月刃基础伤害和飞行参数 |
| `Hit Gameplay Event Tag` | 命中后发送给 FA 后续节点监听 |
| `Expire Gameplay Event Tag` | 月刃消失时发送事件 |

毒素、燃烧、命中特效不要再放到 `Hit Niagara System`、`Expire Niagara System`、`Launch Niagara System` 或 `Additional Hit Effect` 里。新配置应写成：

```text
Spawn Slash Wave Projectile
  -> Wait Gameplay Event
  -> Play Flipbook VFX
  -> ApplyEffect / ApplyGEInRadius
```

## 4. 验收

1. 连携失败时不播放 LinkFlow 里的序列帧特效。
2. 月光毒素命中时能看到小尺寸毒素命中面片。
3. 两个敌人靠近时，主目标中毒，次级目标收到扩散毒层/小伤害。
4. 毒素月光不再播放过大的烟雾 Niagara 命中特效。
5. 重新运行 `RuneCardBatchGenerator -Apply` 后，生成的月光投射物节点仍应保持 `Projectile/Hit/Expire/Launch Niagara=None`。
