# 512 符文卡牌 VFX 配置说明

## 1. 原则

VFX 跟随 FA 或 EffectProfile 配置，不写在 Combat Card 字段里，也不由 GE Tag 自动播放。

| 情况 | 表现配置位置 |
| --- | --- |
| 普通卡牌效果 | 当前卡牌 `BaseFlow` 或对应 Profile |
| 连携成功效果 | 命中的 `LinkFlow` 或对应 Profile |
| 地面区域表现 | `EffectProfile.Area.NiagaraSystem / DecalMaterial` |
| 目标身上持续表现 | `EffectProfile.VFX` 或独立 `Play Niagara` 节点 |

不要把多个连携表现塞进同一个节点。月光、燃烧、中毒、扩散等应拆成独立节点或独立 Profile，方便后续重新连线和调参。

## 2. 当前默认表现

| 用途 | 配置位置 | 资源 | 关键参数 |
| --- | --- | --- | --- |
| 燃烧身上表现 | `EffectProfile.VFX` 或 `Play Niagara` | `NS_Fire_Floor` | `Target=LastDamageTarget; Socket=spine_03; Scale=(0.28,0.28,0.28)` |
| 燃烧地面路径 | `EffectProfile.Area` | `NS_Fire_Floor` + 火焰 Decal | `Shape=Fan; NiagaraInstanceCount=7` |
| 中毒命中 | `Play Niagara` | `NS_Smoke_7_acid` | `Target=LastDamageTarget; Socket=spine_02` |
| 中毒扩散 | `Play Niagara` | `NS_Smoke_7_acid` | `Attach=false; Offset=(0,0,18)` |

## 3. 月光 + 燃烧

正向燃烧月光：

```text
Spawn Slash Wave Projectile
  -> Wait Gameplay Event(Action.Rune.MoonlightBurnHit)
  -> Play Niagara 或 Play Rune VFX Profile
  -> Apply Rune Effect Profile(UGE_RuneBurn)
```

反向燃烧月光：

```text
Calc Rune Ground Path Transform
  -> Spawn Rune Area Profile(EP_Rune512_Moonlight_Reversed_Burn)
```

`Spawn Rune Area Profile` 会同时读取：

| Profile 区块 | 作用 |
| --- | --- |
| `Area.NiagaraSystem / DecalMaterial` | 地面火焰和范围 |
| `Area.Effect` | 敌人进入区域后获得燃烧 |
| `VFX.NiagaraSystem` | 敌人身上挂燃烧特效 |

## 4. 验收

1. 卡牌必须在攻击 AN 命中阶段触发，不应在蒙太奇刚播放时触发。
2. 连携失败时不播放 LinkFlow 的表现节点。
3. 月光投射物内置 Niagara 字段保持 `None`，避免投射物和 FA 重复表现。
4. 燃烧身上特效只来自 FA/Profile，不来自 `Buff.Status.Burning` 自动监听。
5. 地面火焰只负责区域表现；持续掉血由 `UGE_RuneBurn` 和 `Data.Damage.Burn` 控制。
