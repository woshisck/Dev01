# 512 符文卡牌 VFX 配置说明

## 1. 核心原则

卡牌表现跟随 FA 配置：

| 卡牌情况 | 表现配置位置 |
| --- | --- |
| 普通卡牌 | `BaseFlow` |
| 连携卡牌成功 | 对应 `LinkFlow` |
| 连携失败但卡牌正常打出 | 当前卡牌自己的 `BaseFlow` |

不要把多个连携表现写进同一个节点。月光、燃烧、中毒、扩散都应拆成独立节点，方便后续策划在 FA 中重新连线。

## 2. 当前默认节点

当前默认使用 `Play Niagara`。

| 用途 | 节点 | Niagara | 关键参数 |
| --- | --- | --- | --- |
| 燃烧命中/持续表现 | `Play Niagara` | `NS_Fire_Floor` | `Target=LastDamageTarget; Socket=spine_03; Scale=(0.28,0.28,0.28); Lifetime=3.2` |
| 中毒命中 | `Play Niagara` | `NS_Smoke_7_acid` | `Target=LastDamageTarget; Socket=spine_02; Scale=(0.32,0.32,0.32); Lifetime=1.2` |
| 中毒扩散 | `Play Niagara` | `NS_Smoke_7_acid` | `Target=LastDamageTarget; bAttachToTarget=false; Offset=(0,0,18); Scale=(0.45,0.45,0.45); Lifetime=1.4` |

`Play Flipbook VFX` 节点保留为备用方案，但当前生成器会清空生成 FA 中的 Flipbook 节点资源引用，默认不再使用。

## 3. 月光 + 燃烧 FA

配置在：

```text
FA_Rune512_Moonlight_Forward_Burn
FA_Rune512_Moonlight_Reversed_Burn
```

推荐节点链：

```text
Spawn Slash Wave Projectile
  -> Wait Gameplay Event(Action.Rune.MoonlightBurnHit)
  -> Play Niagara(Rune.Moonlight.BurnHitNiagara)
  -> Apply Gameplay Effect Class(UGE_RuneBurn)
```

关键字段：

| 节点 | 字段 | 值 |
| --- | --- | --- |
| `Spawn Slash Wave Projectile` | `Hit Gameplay Event Tag` | `Action.Rune.MoonlightBurnHit` |
| `Spawn Slash Wave Projectile` | `Hit Niagara System` | `None` |
| `Wait Gameplay Event` | `Target` | `BuffOwner` |
| `Play Niagara` | `Niagara System` | `NS_Fire_Floor` |
| `Play Niagara` | `Attach Target` | `LastDamageTarget` |
| `Play Niagara` | `Lifetime` | `3.2` |
| `Apply Gameplay Effect Class` | `Effect` | `UGE_RuneBurn` |
| `Apply Gameplay Effect Class` | `Target` | `LastDamageTarget` |
| `Apply Gameplay Effect Class` | `bRemoveEffectOnCleanup` | 不勾选 |
| `Apply Gameplay Effect Class` | `SetByCallerTag1 / Value1` | `Data.Damage.Burn / 6` |

## 4. 月光 + 中毒 FA

配置在：

```text
FA_Rune512_Moonlight_Forward_Poison
FA_Rune512_Moonlight_Reversed_Poison
```

推荐节点链：

```text
Spawn Slash Wave Projectile
  -> Wait Gameplay Event(Action.Rune.MoonlightPoisonHit)
  -> Play Niagara(Rune.Moonlight.PoisonHitNiagara)
  -> Apply Gameplay Effect Class(GE_Poison)
  -> Play Niagara(Rune.Moonlight.PoisonSpreadNiagara)
  -> ApplyGEInRadius(GE_PoisonSplash/GE_Poison)
```

关键字段：

| 节点 | 字段 | 值 |
| --- | --- | --- |
| `Spawn Slash Wave Projectile` | `Hit Gameplay Event Tag` | `Action.Rune.MoonlightPoisonHit` |
| `Play Niagara` 命中 | `Niagara System` | `NS_Smoke_7_acid` |
| `Play Niagara` 命中 | `Attach Target` | `LastDamageTarget` |
| `Apply Gameplay Effect Class` | `Effect` | `GE_Poison` |
| `Apply Gameplay Effect Class` | `ApplicationCount` | `3` |
| `Play Niagara` 扩散 | `bAttachToTarget` | 不勾选 |
| `ApplyGEInRadius` | `Radius` | `300` |
| `ApplyGEInRadius` | `MaxTargets` | `3` |
| `ApplyGEInRadius` | `bExcludeLocationSourceActor` | 勾选 |
| `ApplyGEInRadius` | `SetByCallerTag1 / Value1` | `Data.Damage / 5` |

## 5. 验收

1. 卡牌必须在攻击 AN 命中阶段触发，不应在蒙太奇刚播放时触发。
2. 连携失败时不播放 LinkFlow 的 Niagara。
3. 月光投射物的内置 Niagara 字段保持 `None`。
4. 燃烧命中后应出现 Niagara 表现，并且后续持续掉血由 `UGE_RuneBurn` 负责。
5. 中毒月光命中后主目标叠毒，半径内次级敌人收到小额毒伤/毒层。
