# 512 符文图标导入配置说明

## 作用

记录 512 符文卡牌图标的源文件、导入位置和 DA 引用方式。

## 源文件

源 PNG 位于：
`X:\Project\Dev01\SourceArt\512RuneIcons`

| 文件 | 用途 |
| --- | --- |
| `T_Rune512_Burn.png` | 燃烧 |
| `T_Rune512_Poison.png` | 中毒 |
| `T_Rune512_Moonlight.png` | 月光 |
| `T_Rune512_Split.png` | 溅射/分裂 |
| `T_Rune512_Shield.png` | 护盾 |
| `T_Rune512_Pierce.png` | 穿透 |
| `T_Rune512_Attack.png` | 攻击 |
| `T_Rune512_ReduceDamage.png` | 减伤 |

## UE 导入位置

`/Game/Docs/BuffDocs/V2-RuneCard/Icons`

导入后资产名保持 `T_Rune512_*`。

## DA 引用位置

打开生成的 `DA_Rune512_*`：

`RuneInfo -> RuneConfig -> RuneIcon`

批量工具 `RuneCardBatchGenerator -Apply` 会自动写入图标引用。如果图标重新生成，重新运行 `-Apply` 即可覆盖导入并回写引用。

## 验收

1. `Icons` 文件夹下有 8 个 `T_Rune512_*.uasset`。
2. 每张 `DA_Rune512_*` 的 `RuneIcon` 不为空。
3. UI 中图标风格统一，缩略图不裁切关键符号。
