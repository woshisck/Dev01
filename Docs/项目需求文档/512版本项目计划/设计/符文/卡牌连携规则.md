# 卡牌连携规则

## 基础规则

连携默认只读取上一张卡输出的 `TraitContext`。

```text
TraitContext
- TraitType
- TraitPower
- Radius
- Duration
- MaxTargets
- MaxApplyCount
- SourceStatus
- SourceCardId
- bRestricted
```

## 结算顺序

1. 当前卡执行默认效果。
2. 如果动作匹配，激活当前卡 `SignatureTrait`。
3. 如果有 `ReadPreviousTrait`，读取上一张过滤后的 `TraitContext`。
4. 当前 `EffectForm` 尝试解释上一张特征。
5. 可解释时执行特征转译。
6. 不可解释时降级为数值增强或默认效果。
7. 如果有 `PassCurrentTrait`，输出过滤后的新 `TraitContext`。
8. 如果是终结技，交给 `FinisherRule` 单独处理。

## 连携词条

| 词条 | 作用 | 准绳 |
| --- | --- | --- |
| `ReadPreviousTrait` | 读取上一张输出的特征 | 默认连携词条 |
| `PassCurrentTrait` | 把当前激活的特征传给下一张 | 仅传过滤后的 `TraitContext` |
| `TranslateTrait` | 将上一张特征转译到当前形态 | 需要当前形态支持 |
| `AmplifyTrait` | 放大当前特征强度 | 有上限 |
| `FinisherConsumesTrait` | 终结技消耗特征并单独设计效果 | 适合稀有卡 |

## 推荐转译

| 上一张特征 | 当前效果 | 结果 |
| --- | --- | --- |
| `Spread` | 燃烧 | 扩大燃烧周围伤害范围或影响目标数 |
| `AuraDamage` | 中毒 | 毒传播落点附加小范围持续伤害 |
| `MovementPunish` | 击退 / 恐惧 | 位移过程额外结算撕裂收益 |
| `ForceImpulse` | 剑气 | 剑气命中造成一次方向击退 |
| `OnHitPunish` | 多段剑气 | 每目标按上限触发伤口追加 |
| `ArmorPressure` | 重击 / 扇形斩 | 增加护甲伤害 |
| `AreaScale` | 燃烧 / 毒池 / 爆炸 | 扩大半径或持续区域 |
| `EscapeCheck` | 冻结 / 恐惧 | 强化逃离失败惩罚 |

## 失败降级

以下情况只保留默认效果，不携带特征：

- 动作不匹配
- 无上一张特征
- 上一张未匹配
- 当前形态无法解释上一张特征
- 目标免疫该特征
- 洗牌期间

## 示例

### 毒先火后

```text
中毒匹配输出 Spread。
燃烧匹配读取 Spread。
结果：燃烧仍是燃烧，但周围伤害范围或目标数扩大。
```

### 火先毒后

```text
燃烧匹配输出 AuraDamage。
中毒匹配读取 AuraDamage。
结果：中毒仍是中毒，但传播落点附加小范围持续伤害。
```

## 特殊化学反应

特殊化学反应不进入默认规则，只能来自：

- 稀有卡
- 特殊卡
- 特殊武器

它必须单独写清触发条件、消耗特征、目标数、持续时间、失败降级、VFX 和测试边界。
