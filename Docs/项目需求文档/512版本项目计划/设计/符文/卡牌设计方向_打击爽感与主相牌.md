# 卡牌设计方向：打击爽感与主相牌

## 定位

512 版本卡牌系统的主轴不是无限法术拼装，而是打击升级。

玩家的第一感受应该是：

```text
我在打连招，卡牌让这一击变得更重、更大、更狠、更有变化。
```

卡牌系统不替代动作系统。动作系统提供稳定的攻击、轻重连招、DashSave 和终结击；卡牌系统负责把这些攻击结果改写成更有层次的命中反馈、状态连锁和终结爆发。

```text
动作连招 = 手感骨架
卡牌序列 = 打击修饰
状态连锁 = 结果扩散
主相牌 = 力量性质改写
终结爆发 = 节奏兑现
```

本方向参考 Noita 的可预测因果链，但不追求无限拼装。玩家应该能预测大方向，制作侧必须能控制复杂度。

## 核心优先级

1. 打击爽感优先
   卡牌首先服务命中、击退、破甲、停顿、多段、范围、震荡和终结反馈。

2. 状态连锁为辅
   燃烧、中毒、撕裂、伤口、恐惧、冻结等状态用于给打击结果留下可解释材料。

3. 主相牌提升性质变化
   主相牌负责把攻击、状态或终结改写为某个力量体系的表现。

4. 终结爆发作为打击升级
   终结爆发必须像一段连招和一轮卡牌节奏的结果，而不是突然插入的无关法术。

## 卡牌分层

### 基础打击牌

基础打击牌是卡组里的稳定骨架，数量最多。

它们不明显改变动作动画，主要改变本次命中的结果。

常见效果：

```text
攻击伤害提高
命中停顿增强
小范围震荡
轻微击退
额外破甲
追加短距离剑气
攻击中减伤
```

制作准绳：

```text
基础打击牌不让玩家换脑子。
玩家看到它，只需要知道下一击更强。
```

### 状态打击牌

状态打击牌是打击和连锁之间的桥。

它们仍以命中为核心，但会给敌人留下可被后续牌解释的状态材料。

常见状态方向：

| 状态 | 战斗用途 | 可作为材料 |
| --- | --- | --- |
| 燃烧 | 范围压制、清场 | 爆燃、范围扩大、持续伤害 |
| 中毒 | 反精英、反护甲、扩散 | 传染、毒池、寄生 |
| 撕裂 | 位移惩罚 | 击退结算、恐惧惩罚 |
| 伤口 | 多段攻击收益 | 追击、处决、血肉增殖 |
| 击退 | 空间控制 | 撞击、破甲、位移爆发 |

制作准绳：

```text
状态必须先表现为一次打击结果，再成为后续连锁材料。
```

### 位移与破甲牌

位移和破甲牌负责制造空间变化和防御崩坏，是打击爽感的重要中层。

常见效果：

```text
强击退
拉扯
震荡
破甲
霸体压制
位移过程追加伤害
撞墙爆发
```

制作准绳：

```text
位移和破甲必须看起来像打击力量的结果，而不是凭空施法。
```

这类牌适合绑定重攻击、DashSave 衔接或终结击。

### 主相牌

主相牌是原“相性牌”的设定升级。

主相牌代表高阶力量相性。它不只是造成效果，而是改变攻击、状态、连携或终结的性质。

普通牌回答：

```text
这一击产生什么效果？
```

主相牌回答：

```text
这一击被哪种高阶力量改写？
```

主相牌可以参考塔罗牌中“大阿卡那”的设计方法：每张主牌都有象征主题、战斗动词、构筑倾向和限制，但不建议直接照搬塔罗名称。

### 终结爆发牌

终结爆发牌只在明确的终结条件下兑现高强度反馈。

可读取的信息应该有限：

```text
当前卡特征
上一张 TraitContext
目标身上的状态
当前连击段数
是否破甲
是否击退或位移中
```

不读取完整历史，不做无限追溯。

制作准绳：

```text
终结爆发必须像这一段连招的结果，不能像突然插入一个无关技能。
```

## 主相牌设定

游戏世界观中存在两个主要力量谱系：

```text
灵魂谱系：灵魂、暗夜、梦境，以灵魂为主
血肉谱系：血肉、力量、寄生、收割，以血肉为主
```

主相牌不应设计成普通元素牌，而应该是这两个谱系中的高阶象征牌。

### 灵魂谱系

关键词：

```text
灵魂、暗夜、梦境、月光、回声、影子、幻觉、牵引、错位、延迟、裂隙
```

战斗气质：

```text
优雅、诡异、精准、延迟爆发、空间错位、弱控制、精神干扰
```

典型设计语言：

```text
不是把敌人打碎，而是让敌人的存在被错开、牵走、折返、回响。
```

推荐首批主相：

| 主相牌 | 核心动词 | 适合连携 | 战斗定位 | 限制 |
| --- | --- | --- | --- | --- |
| 月 | 折射 / 留痕 / 斩波 | 剑气、路径、燃烧、中毒、护盾 | 招式变体 | 直接爆发不应总是最高 |
| 影 | 残响 / 追击 / 错位 | 伤口、多段、背击、恐惧 | 追加打击 | 需要命中或标记作为来源 |
| 梦 | 延迟 / 梦魇 / 失控 | 恐惧、冻结、标记、逃离判定 | 延迟爆发 | 需要明显提示和等待时间 |
| 魂 | 抽取 / 回流 / 护佑 | 减伤、护盾、击杀、精英目标 | 攻防循环 | 回复和护盾必须有上限 |
| 夜 | 遮蔽 / 压制 / 扩域 | 范围、减速、恐惧、暗场 | 场域控制 | 强控制不默认传播 |
| 镜 | 复制 / 反射 / 反相 | 投射物、剑气、反击、月光 | 高级连携 | 必须限制复制次数 |

示例：

```text
因为燃烧被【月】改写，
所以普通燃烧不再只是灼烧，而是沿月光路径爆燃。

因为伤口被【影】改写，
所以普通伤口不再只是受击追加，而是产生一次暗影追击。

因为恐惧被【梦】改写，
所以敌人逃离失败时触发梦魇惩罚。
```

### 血肉谱系

关键词：

```text
血肉、力量、寄生、收割、伤口、骨刺、血藤、脉动、吞噬、处决
```

战斗气质：

```text
粗暴、压迫、近身、持续撕扯、生命代价、高收益收割
```

典型设计语言：

```text
不是优雅地施法，而是把每次命中变成生长、撕裂、吞噬和收割。
```

推荐首批主相：

| 主相牌 | 核心动词 | 适合连携 | 战斗定位 | 限制 |
| --- | --- | --- | --- | --- |
| 肉 | 增殖 / 硬化 / 吞噬 | 近战、重击、护甲、减伤 | 近身压制 | 不能无限叠加肉芽或护甲 |
| 牙 | 撕咬 / 穿刺 / 破甲 | 穿透、伤口、破甲、精英目标 | 单体爆发 | 需要命中目标，范围较弱 |
| 巢 | 寄生 / 扩散 / 孵化 | 中毒、敌群、死亡、持续伤害 | 群体连锁 | 单体爆发弱，扩散次数有限 |
| 骨 | 突刺 / 震裂 / 支撑 | 击退、扇形、重击、破甲 | 重击控制 | 不默认造成强控 |
| 镰 | 收割 / 处决 / 爆血 | 低血、终结击、击杀、撕裂 | 终结爆发 | 依赖收割线或终结条件 |
| 脉 | 脉动 / 反噬 / 代价 | 吸血、减伤、生命代价、多段 | 高风险增益 | 必须有代价、冷却或上限 |

示例：

```text
因为中毒被【巢】改写，
所以普通中毒不再只是掉血，而是变成寄生扩散。

因为伤口被【牙】改写，
所以普通伤口不再只是受击追加，而是变成破甲穿刺后的撕咬爆发。

因为终结击被【镰】改写，
所以低血目标被击杀时触发范围收割反馈。
```

## 主相牌制作规则

### 不做全能牌

每张主相牌只掌握 2 到 3 个核心动词。

例如：

```text
月：折射、留痕、斩波
影：残响、追击、错位
梦：延迟、梦魇、失控
魂：抽取、回流、护佑

肉：增殖、硬化、吞噬
牙：穿刺、撕咬、破甲
巢：寄生、扩散、孵化
镰：处决、收割、爆血
```

后续设计时先判断：

```text
这张主相牌的核心动词能不能解释当前牌？
能解释，产生主相变体。
不能解释，降级为普通主相效果或默认效果。
```

### 用语法控制数量

不要为每个“主相牌 × 状态牌”单独设计完整效果。应使用固定语法控制扩展：

```text
打击形态决定怎么打出去
状态材料决定用什么燃料
主相主题决定表现成什么性质
```

组合句式：

```text
【打击形态】 + 【状态材料】 + 【主相主题】
= 【可解释的战斗变化】
```

示例：

```text
重击 + 燃烧 + 月
= 月光重斩命中后产生蓝白爆燃

剑气 + 中毒 + 巢
= 毒性寄生体沿命中目标扩散

扇形斩 + 伤口 + 牙
= 扇形穿刺造成伤口增殖和破甲撕咬
```

### 限制读取范围

主相牌默认只读取上一张牌输出的 `TraitContext`，或作为反向连携等待下一张牌。

不读取完整历史，不跨洗牌保存，不无限递归。

### 限制变招权限

普通卡主要改命中结果。

允许明显改变招式形态的卡牌类型：

```text
主相牌
稀有连携牌
终结爆发牌
特殊武器牌
```

### 爆炸必须有来源

爆炸效果必须有燃料或触发来源。

允许来源：

```text
燃烧
引信
护甲碎裂
位移撞击
寄生成熟
灵魂回响
终结击
低血收割
```

不允许普通状态无条件变成强爆炸。

## 推荐战斗闭环

```text
1. 玩家进行轻重攻击连招
2. 每次有效攻击消耗当前卡
3. 普通卡直接强化这次命中
4. 状态卡留下可解释材料
5. 主相牌读取材料并改写性质
6. 终结击把当前积累兑现成爆发
7. 卡组打空进入短暂洗牌空窗
8. 玩家继续用基础攻击维持战斗节奏
9. 洗牌结束，进入下一轮爆发
```

玩家体验目标：

```text
平砍不空
卡牌有料
连锁合理
变招克制
终结很爽
空窗不罚站
```

## 制作红线

- 普通卡不大幅改动画，只改命中结果。
- 主相牌、稀有牌、终结牌才允许明显改招式形态。
- 连携默认只读上一张或等待下一张，不读完整历史。
- 强控制、诅咒、强爆炸不默认传播。
- 爆炸必须有燃料或明确触发来源。
- 每张主相牌只保留 2 到 3 个核心动词。
- 每个主相牌只解释少数适配材料，不解释所有状态。
- 洗牌期间不触发卡牌，但攻击永远不被禁止。
- 状态连锁必须能用一句话解释。

## 验收句式

单张卡验收：

```text
因为玩家打出了【某种攻击】，
当前卡让这次命中产生【打击升级】。
```

连携验收：

```text
因为上一张卡提供了【特征】，
所以当前卡的【效果形态】发生了【可见变化】。
```

主相牌验收：

```text
因为当前攻击或状态被【主相牌】改写，
所以它从【原本效果】变成了【该谱系能解释的战斗表现】。
```

终结爆发验收：

```text
因为这一段连招已经积累了【状态 / 位移 / 破甲 / 主相】，
所以终结击将其兑现为【爆发反馈】。
```

如果以上句式说不清楚，这张卡或这组连携就需要收窄。
