# 卡牌设计文档索引

本目录用于存放 512 版本卡牌、符文、状态和连携的设计资料。

## 文档列表

| 文档 | 用途 |
| --- | --- |
| `卡牌设计总纲.md` | 本批设计文档的背景、目标和阅读入口 |
| `卡牌设计准绳.md` | 制作原则、红线和整体方向 |
| `卡牌字段规范.md` | 单张卡必须填写的有限字段 |
| `卡牌特征库.md` | 状态的默认效果、标志性特征和护甲交互 |
| `卡牌连携规则.md` | 特征读取、转译、失败降级和终结技边界 |
| `卡牌设计填写表.md` | 单张卡和多卡组合的填写模板 |

## 核心句

```text
玩家自由组合卡牌；制作只允许组合有限字段、有限效果形态和有限特征。
```
