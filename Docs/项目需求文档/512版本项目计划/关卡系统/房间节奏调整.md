w# 房间节奏调整

## 目标

关卡节奏要让玩家频繁验证卡组构筑，而不是长时间脱离战斗整理背包。

每个普通房都应让玩家至少体验一次完整卡牌序列；精英房应让玩家体验洗牌空窗、连招终结、热度强化。

## 普通房节奏

| 项目 | 建议 |
| --- | --- |
| 目标时长 | 45-90 秒 |
| 敌人波次 | 1-3 波 |
| 主要目的 | 快速触发卡牌效果，验证新奖励 |
| 敌人强度 | 普通敌人为主，少量压力点 |

## 精英房节奏

| 项目 | 建议 |
| --- | --- |
| 目标时长 | 90-150 秒 |
| 敌人配置 | 精英敌人 + 小怪压力 |
| 主要目的 | 测试连招、闪避、终结技、洗牌空窗 |
| 体验要求 | 至少让玩家经历一次卡组打空和回填 |

## 关卡体验目标

玩家进入房间后的体验应为：

1. 开局看到当前 1D 卡组。
2. 通过轻重攻击按顺序消耗卡牌。
3. 卡牌打空后进入短暂洗牌。
4. 洗牌期间继续靠基础攻击和走位战斗。
5. 卡组回填后再次进入爆发节奏。
6. 清场后选择新卡或符文。
7. 进入下一房间立刻验证新构筑。

## 验收标准

- 普通房能让玩家至少打完一轮 4 张卡。
- 精英房能让玩家体验至少一次洗牌空窗。
- 房间结束后不会强制进入背包整理流程。
- 奖励选择后下一房间能立刻感受到卡组变化。
