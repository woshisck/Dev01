# 小怪掉落与主奖励

## 目标

小怪掉落负责即时反馈，清场主奖励负责构筑成长。两者分工明确，避免玩家错过核心成长，也避免战斗中频繁停下来处理复杂奖励。

## 小怪掉落

| 掉落类型 | 说明 |
| --- | --- |
| 金币 | 可自动吸附或简化拾取 |
| 小资源 | 用于升级、商店、事件 |
| 临时补给 | 可选，用于生命、热度、短时增益 |

小怪掉落不建议直接掉复杂卡牌，避免玩家在战斗中分心判断。

## 清场主奖励

| 奖励类型 | 说明 |
| --- | --- |
| 攻击卡 | 改变轻重攻击触发效果 |
| Link 卡 | 提供进阶连携 |
| Finisher 卡 | 强化连招末端爆发 |
| Passive 卡 | 可作为后续扩展，第一版谨慎使用 |

## 跳过奖励

清场后传送门可以开启。玩家可以不拾取奖励直接进门，但这等价于主动放弃成长。

需要 UI 明确表达：

- 奖励仍在房间内。
- 进门不会自动领取该奖励。
- 放弃奖励不阻塞流程。

## 验收标准

- 小怪可以掉落金币或小资源。
- 小怪掉落不打断战斗节奏。
- 清场后出现主奖励。
- 主奖励选择后进入后台背包和卡组。
- 玩家可以不拾取奖励直接进门。
