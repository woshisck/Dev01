## 使用规则

1. 本页只放“编辑器配置/制作事项”，不放纯代码任务。
2. 配置完成后，在 [TASKS](PM/TASKS.md) 对应行标记，并在必要时补 [验收报告](验收与报告/)。
3. 如果配置步骤稳定可复用，应沉淀到 [03_策划配置与制作手册](../03_策划配置与制作手册/README.md)。
4. 如果任务已经不符合当前新战斗方案，应迁入 [99_归档](../99_归档/README.md) 或 [98_废弃](../98_废弃/README.md)。
