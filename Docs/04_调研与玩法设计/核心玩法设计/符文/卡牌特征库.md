# 卡牌特征库

## 特征列表

| 特征 | 中文名 | 典型来源 | 作用方向 | 传递准绳 |
| --- | --- | --- | --- | --- |
| `Spread` | 传染/扩散 | 中毒 | 影响周边目标或下一个目标 | 限制目标数 |
| `AuraDamage` | 周围持续伤害 | 燃烧 | 目标周围产生持续伤害圈 | 限制半径 |
| `MovementPunish` | 位移惩罚 | 撕裂 | 按目标移动距离结算伤害 | 适合击退/恐惧 |
| `OnHitPunish` | 受击追加 | 伤口 | 目标受击时额外扣血 | 限制同次触发 |
| `ArmorPressure` | 护甲压制 | 中毒、击退 | 额外护甲伤害或削弱护甲 | 不能直接破甲 |
| `ForceImpulse` | 冲击位移 | 击退 | 当前形态获得一次方向击退 | 默认只结算一次 |
| `EscapeCheck` | 逃离判定 | 恐惧、冻结 | 离不开触发点则受罚 | 受限传递 |
| `ControlLock` | 行动锁定 | 眩晕、冻结 | 禁止移动、攻击或主动行为 | 不默认传递 |
| `CurseAmplify` | 负面放大 | 诅咒 | 根据负面状态数量削弱最大生命值 | 不默认传递 |
| `AreaScale` | 范围增幅 | 范围强化词条 | 扩大半径、长度或角度 | 有数值上限 |
| `ChainBounce` | 弹射 | 特殊能量、碰撞 | 在目标间跳转 | 必须有次数上限 |
| `Detonate` | 引爆 | 引信、终结技 | 消耗状态并爆发 | 通常由终结技使用 |

## 状态表

| 状态 | 默认效果 | 匹配后特征 | 设计方向 |
| --- | --- | --- | --- |
| 流血 | 每秒固定伤害，有护甲时无法触发 | 无主特征 | 破甲后收割 |
| 中毒 | 每秒按最大生命值掉血，不至死 | `Spread`、`ArmorPressure` | 反精英、反护甲、传染 |
| 撕裂 | 按移动距离掉血，原地 2 秒后消失 | `MovementPunish` | 配合击退、恐惧、冻结 |
| 伤口 | 目标受击时额外扣血 | `OnHitPunish` | 配合多段攻击、剑气 |
| 击退 | 受击时被推开 | `ForceImpulse`、`ArmorPressure` | 物理流核心 |
| 燃烧 | 每秒固定伤害 | `AuraDamage` | 范围压制、清场 |
| 眩晕 | 禁止移动和攻击 | `ControlLock` | 强控制，受限 |
| 恐惧 | 逃离附近敌人，失败受伤 | `EscapeCheck` | 逼位、服务撕裂 |
| 冻结 | 逃不出范围则冻结眩晕并受伤 | `EscapeCheck`、`ControlLock` | 延迟控制，受限 |
| 诅咒 | 每个负面效果降低最大生命值 | `CurseAmplify` | 稀有高收益 |

## 护甲全局表

| 效果 | 有护甲时 | 无护甲时 |
| --- | --- | --- |
| 流血 | 无法触发 | 正常触发 |
| 中毒 | 可以触发，额外扣 25% 护甲，只扣护甲 | 正常按最大生命值掉血，不至死 |
| 撕裂 | 触发概率下降 25% | 正常触发 |
| 伤口 | 触发概率下降 25% | 正常触发 |
| 击退 | 距离减少，额外造成 15% 护甲伤害 | 正常击退 |
| 燃烧 | 正常触发 | 伤害提高 15% |
| 眩晕 | 概率提升，持续时间增加；霸体免疫 | 正常触发；霸体免疫 |
| 恐惧 | 正常触发 | 正常触发 |
| 冻结 | 正常判定 | 正常判定 |
| 诅咒 | 正常计入负面效果 | 正常计入负面效果 |
