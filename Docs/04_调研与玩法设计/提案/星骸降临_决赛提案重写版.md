## 决赛高亮重点

- 「好玩有乐趣」：玩家不需要停下来出牌。攻击、冲刺、连招和终结技会自然推进卡组，卡牌把每一次命中改写成击退、传播、破甲、引爆、追击和终结爆发。
- 「提案有创意」：不是简单的“动作 + 卡牌”，而是把卡牌做成连招语法。玩家真正构筑的不是数值 Buff，而是一套“这一击之后，下一击会变成什么”的战斗句子。
- 「作品独特性」：卡组固定顺序、可预测连携、主相牌改写攻击性质、终结技兑现一轮卡组节奏，形成区别于 Hades、Dead Cells 和传统卡牌肉鸽的前台体验。
- 「作品完成度」：已有 PV、体验包和三轮玩家集市反馈；核心系统已经从 2D 背包迭代为 1D 线性卡组，证明项目不是停留在概念阶段，而是经历过可玩验证后的收束。
- 「最想问评委的一个问题」：如果一款动作肉鸽把“构筑”从战前选择变成战斗中的连招语法，您认为它最应该优先证明的是操作爽感、构筑深度，还是长期内容扩展性？

## 3. 作品简介

### 3.1 产品核心概念

《星骸降临》是一款俯视角动作卡牌 Roguelite。

它的核心不是“玩家在动作游戏里手动出牌”，而是：

```text
玩家不出牌。
玩家打连招。
卡牌改写连招的结果。
```

玩家在实时战斗中通过轻攻击、重攻击、冲刺、连段和终结输入推进一条固定顺序的 1D 战斗卡组。每一次有效攻击都会尝试消耗当前卡牌；卡牌不会打断动作，而是把这次命中改写成更强、更大、更狠、更有变化的战斗结果。

一张卡可能让这一击附带伤口；下一张卡可能读取这个伤口，把普通剑气改写成暗影追击；再下一张主相牌可能把这次攻击改写成月光折射；最后一张终结技牌会要求玩家完成一段指定输入，把前面一轮卡组积累的状态、击退、破甲或主相力量兑现为终结爆发。

### 3.2 核心体验样例

玩家带着一组固定顺序卡组进入房间：

```text
伤口牌 -> 月相牌 -> 击退剑气 -> 怒锋入铠
```

第一击命中后，敌人身上留下“伤口”。  
第二击触发“月相牌”，读取上一张牌的伤口特征，让普通攻击留下月光残痕。  
玩家在敌人反击前冲刺，但 DashSave 保住了连招上下文，下一击不会从头开始。  
第三击打出击退剑气，把敌人推向墙面并制造破甲机会。  
第四张牌是终结技牌“怒锋入铠”。玩家用 Heavy 命中后，卡牌先释放“破阵斩”；如果玩家继续在短输入窗口内完成 `Light -> Heavy`，就触发“半剑凿铠”终结蒙太奇：半剑突刺、错身拨开、倒持剑柄砸碎护甲。

卡组打空后进入短暂洗牌空窗。这个空窗不会禁止玩家攻击，玩家仍然可以用基础动作维持战斗，只是不触发卡牌。洗牌结束后，卡组按固定顺序回填，玩家开始计划下一轮爆发。

这就是《星骸降临》的核心乐趣：

```text
平砍不空。
卡牌有料。
连锁合理。
变招克制。
终结很爽。
空窗不罚站。
```

### 3.3 设计支柱

#### Pillar 1：卡牌是连招语法

传统动作肉鸽里的构筑往往是后台数值：伤害更高、范围更大、冷却更短。  
《星骸降临》希望让构筑进入战斗前台，但不抢玩家注意力。

玩家不需要在战中读满手牌，也不需要暂停思考“现在出哪张”。卡牌按固定顺序进入战斗，玩家通过攻击节奏、连招延续、冲刺衔接和终结输入，让卡牌自然改写这次命中。

玩家构筑的不是一堆被动词条，而是一套可以被自己打出来的战斗语法。

#### Pillar 2：可预测的卡牌连携

卡组不随机洗牌，玩家可以决定顺序。  
连携默认只读取上一张牌输出的特征，不读取完整历史，不做无限递归。

这让玩家可以形成清晰预期：

```text
因为上一张卡提供了“传染”，
所以当前燃烧牌扩大了燃烧范围。

因为上一张卡提供了“击退”，
所以当前剑气额外造成方向冲击。

因为上一张卡提供了“伤口”，
所以当前影相攻击触发追击。
```

组合有惊喜，但不是黑箱。玩家会逐步学会：哪张牌适合做材料，哪张牌适合做承接，哪张牌适合做爆发。

#### Pillar 3：主相牌改写力量性质

游戏中的高阶卡牌不是普通元素牌，而是“主相牌”。  
主相牌代表世界观中的高阶力量谱系，它回答的问题不是“这一击加多少伤害”，而是“这一击被哪种力量改写”。

灵魂谱系偏向月光、梦境、暗夜、影子、回声、牵引和错位。  
血肉谱系偏向寄生、撕裂、伤口、骨刺、脉动、吞噬和收割。

例如：

```text
燃烧 + 月 = 沿月光路径爆燃。
伤口 + 影 = 产生一次暗影追击。
中毒 + 巢 = 毒性寄生体沿敌群扩散。
低血 + 镰 = 终结击触发范围收割。
```

主相牌把世界观、战斗表现和构筑深度绑在一起，让玩家觉得自己不是在堆数值，而是在学习并驾驭这个世界的力量语法。

#### Pillar 4：终结技兑现一轮卡组节奏

终结技不是突然插入的技能按钮，而是一轮连招和卡组节奏的兑现。

部分终结技卡会提供额外输入窗口。玩家如果完成指定轻重攻击输入，就能触发专属终结动作；如果失败，卡牌仍有默认效果，不会让玩家觉得完全浪费。

以德式双手剑终结技“怒锋入铠”为例：

```text
Heavy 命中 -> 破阵斩
短窗口输入 Light -> Heavy
成功 -> 半剑凿铠终结技
失败 -> 保留破阵斩，不播放终结蒙太奇
```

这样，终结技既有动作玩家喜欢的输入兑现，也有构筑玩家喜欢的顺序计划。

## 4. 产品形态

### 4.1 题材

俯视角动作卡牌 Roguelite。  
世界观为架空维多利亚时期的黑暗奇幻，融合克苏鲁式宇宙恐怖、哥特宗教、炼金术、碎片叙事和血肉诅咒。

玩家扮演一位在监牢深处复活的教宗残魂，反复死亡、重生、战斗和探索，在失控的圣城中找回记忆，并逐渐理解绯红之星、白昼教会、炼金术、血肉诅咒和自身存在之间的关系。

### 4.2 美术

美术方向为风格化写实的俯视角动作表现。

- 角色：3D 卡通渲染，强调轮廓、动作识别和光影二分。
- 场景：半写实风格化，保留维多利亚城市、监牢、炼金工坊、教堂废墟和仪式遗迹的材质细节。
- 战斗反馈：命中停顿、闪白、敌人前摇红光、热度升阶发光、武器边缘光、终结技镜头震动共同服务“打中了、打重了、爆发了”的体感。
- 卡牌表现：HUD 不展示复杂手牌，而展示下一张和当前序列，让玩家能瞥一眼理解节奏。

### 4.3 世界观

《星骸降临》的世界不是单纯被怪物占领的城市，而是一个信仰、技术、阶级、血肉与世界线同时失控的圣城。

这个世界的现实由三层神秘结构维持：

| 结构 | 含义 | 对应能力 |
| --- | --- | --- |
| 名 | 事物被世界承认的名字 | 真言 |
| 相 | 事物呈现给观察者的形态 | 灵视 |
| 响 | 事物在时间、死亡和记忆中留下的回声 | 灵听 |

黑夜不是邪恶，白昼也不是纯粹正义。黑夜代表星空、梦境、死亡、静默和隐藏知识；白昼教会负责把不可见启示制度化为律法、档案、审判、仪式和社会秩序。

当无星之夜切断神谕，玫瑰教派通过迎星仪式召来绯红之星，血肉诅咒以白昼教堂为中心爆发。玩家进入的圣城，是制度长期失控后的结果。

游戏的深层主题不是“战胜邪神”，而是：

```text
在被制造、被命令、被注视、被记录、被反复死亡重置的世界中，
玩家是否还能建立一个属于自己的“我”。
```

### 4.4 游戏循环

局内循环：

```text
进入房间
-> 动作战斗
-> 攻击推进卡组
-> 卡牌改写命中
-> 连携产生爆发
-> 卡组打空洗牌
-> 清场获得奖励
-> 奖励进入卡组
-> 进入下一房间验证新构筑
```

局外循环：

```text
解锁武器
-> 解锁卡牌和主相
-> 解锁终结技模板
-> 解锁路线和剧情碎片
-> 形成新的开局策略
-> 继续挑战更深层圣城
```

长期乐趣不是单纯叠数值，而是“理解机制 -> 设计卡组 -> 在动作战斗中打出可预期爆发 -> 再次优化构筑”。

### 4.5 玩法机制

#### 4.5.1 1D 战斗卡组

战斗卡组是一条固定顺序的卡牌序列。  
玩家每次有效攻击都会尝试消耗当前卡。卡牌被消耗后触发效果，卡组指针推进。卡组打空后进入短暂洗牌空窗，洗牌期间攻击仍可正常进行，但不会触发卡牌。

这个规则解决两个问题：

1. 战斗不中断：玩家不用停下来出牌。
2. 构筑有预期：玩家能记住自己的下一张、下一轮和爆发节点。

#### 4.5.2 普通牌

普通牌是卡组的稳定骨架。它们不大幅改变动作动画，而是改变本次命中的结果。

常见效果包括：

- 额外命中停顿
- 轻微击退
- 小范围震荡
- 追加短距离剑气
- 破甲
- 状态附着
- 攻击中减伤

普通牌的设计目标是：玩家不用换脑子，但能感到这一击更有料。

#### 4.5.3 状态牌与特征转译

状态牌不仅造成状态，还会留下可被下一张牌读取的特征。

例如：

| 状态 | 默认效果 | 可输出特征 |
| --- | --- | --- |
| 中毒 | 持续掉血，不至死 | 传染、护甲压制 |
| 燃烧 | 持续伤害 | 周围持续伤害 |
| 撕裂 | 位移时受罚 | 位移惩罚 |
| 伤口 | 再次受击追加伤害 | 受击追加 |
| 击退 | 推开敌人 | 冲击位移、护甲压制 |

下一张卡如果能解释这个特征，就产生连携；如果不能解释，就降级为普通效果或数值增强。  
这样既保留组合惊喜，也避免每两张卡都要写独立配方。

#### 4.5.4 连携牌

连携牌是少数机制卡，不是所有卡都无限连携。

它们分为三类：

| 类型 | 说明 |
| --- | --- |
| 正向连携 | 读取前一张牌的特征，改变当前牌效果 |
| 反向赋能 | 当前牌先释放，把自身特征挂给后一张牌 |
| 断链释放 | 玩家退出连招或下一张不满足条件时，释放基础效果或断链效果 |

这个设计来自玩测后的取舍：旧方案中，所有卡都能复杂连携，玩家理解负担过重；新方案收束为少数关键连携牌，让构筑亮点更集中，也更适合快节奏动作战斗。

#### 4.5.5 主相牌

主相牌是高阶力量牌。它们不只是给敌人上状态，而是改写攻击、状态、连携或终结的性质。

主相牌必须遵守三条规则：

1. 每张主相牌只掌握 2 到 3 个核心动词。
2. 它只解释少数适配材料，不解释所有状态。
3. 它必须让玩家看见攻击性质发生了变化。

例如：

| 主相 | 核心动词 | 战斗定位 |
| --- | --- | --- |
| 月 | 折射、留痕、斩波 | 招式变体 |
| 影 | 残响、追击、错位 | 追加打击 |
| 梦 | 延迟、梦魇、失控 | 延迟爆发 |
| 巢 | 寄生、扩散、孵化 | 群体连锁 |
| 骨 | 突刺、震裂、支撑 | 重击控制 |
| 镰 | 处决、收割、爆血 | 终结爆发 |

#### 4.5.6 终结技牌

终结技牌负责兑现一轮卡组和连招节奏。  
它可以读取当前特征、上一张特征、目标状态、连击段数、是否破甲、是否击退等有限信息。

终结技不读取完整历史，不做无限追溯。  
它必须像这一段连招的结果，而不是突然插入一个无关技能。

#### 4.5.7 武器与战斗风格

武器决定玩家的基础战斗节奏，也是构筑入口。

- 大剑：重击、破甲、击退、动作承诺和终结爆发。
- 匕首：高速位移、连击、状态播种和追击。
- 火绳枪：蓄力、装填、弹药管理、远程爆发和距离控制。

武器不是锁死职业，而是提供清晰下限。玩家即使还没完全理解卡组，也能通过武器动作获得基础爽感；理解卡组后，武器会成为连携和主相牌的放大器。

#### 4.5.8 路线、净化、献祭与商店

非战斗节点用于调整本局构筑：

- 净化：移除低收益或干扰构筑的卡牌/符文。
- 商店：稳定补强当前流派缺口。
- 献祭：用生命、卡牌或负面代价换取高风险强力恩赐。
- 路线选择：玩家根据当前卡组状态，选择稳妥房间、高价值事件或高风险奖励。

这些节点让玩家不是被随机奖励推着走，而是在每局中持续修正自己的战斗语法。

## 5. 玩家长期参与方式

### 5.1 玩家知识型成长

玩家会越来越理解：

- 卡组顺序如何影响爆发节点。
- 哪些牌适合做材料，哪些牌适合承接。
- 哪些主相牌能解释当前状态。
- 哪些敌人适合用击退、破甲、毒、伤口或收割处理。
- 洗牌空窗应该主动进攻、拉开距离，还是准备下一轮爆发。

这种成长不是单纯属性变高，而是玩家真的更懂系统。

### 5.2 内容解锁型成长

局外成长以提供更多选择为主：

- 解锁新武器，获得不同动作节奏。
- 解锁新普通牌、状态牌、连携牌、主相牌和终结技牌。
- 解锁新路线、隐藏地图、剧情碎片和结局。
- 解锁起始卡组模板和终结技选择。

### 5.3 长期游玩目标

玩家持续游玩的动力来自三句话：

```text
我发现这几张牌可以组成新的连携闭环。
我知道这个敌人该怎么被我的卡组处理。
我想试试另一把武器能不能打出完全不同的终结节奏。
```

## 6. 市场前景

### 6.1 目标用户

核心目标用户：

- 喜欢 Hades、暖雪、Dead Cells 等动作 Roguelite 的玩家。
- 喜欢卡牌构筑、组合流派和策略涌现，但不喜欢战斗频繁停顿的玩家。
- 希望体验动作爽感、成长构筑和反复挑战，但不希望一开始被硬核操作门槛劝退的中度玩家。

项目入口是动作爽感，深度来自后台卡组。  
玩家先爽起来，再逐步理解系统。

### 6.2 竞品差异

| 竞品方向 | 代表作品 | 本作差异 |
| --- | --- | --- |
| 动作 Roguelite | Hades、Dead Cells、暖雪 | 本作让玩家直接编辑战斗节奏和卡牌序列，而不只是选择后台强化 |
| 卡牌构筑 Roguelike | Slay the Spire、Balatro | 本作不打断战斗，不要求回合制读牌，构筑反馈在实时动作中兑现 |
| 背包构筑 | Backpack Hero、Backpack Battles | 本作吸收构筑组合乐趣，但把复杂整理退到后台，让战斗前台保持流畅 |
| 类魂 / 血源向动作 | Bloodborne、类魂作品 | 本作保留读招、闪避和动作承诺，但用卡牌降低纯操作门槛，提高构筑表达 |

### 6.3 玩测反馈与关键迭代

项目已经经历三轮玩家集市。  
核心验证点不是“玩家是否喜欢卡牌”，而是“动作和卡牌是否会互相抢注意力”。

关键迭代：

#### 迭代 1：从 2D 背包改为 1D 线性卡组

旧 2D 背包提供更高构筑自由度，但玩家会花大量时间整理背包，战斗心流被打断。  
新 1D 卡组牺牲一部分构筑广度，换取战斗体验深度。

结果是：

- 玩家更愿意在战斗中测试卡牌组合。
- “组合发现的惊喜感”成为正向反馈。
- 核心循环从“整理系统”变成“打出来验证系统”。

#### 迭代 2：DashSave 与连段奖励

硬核动作玩家反馈冲刺会打断连招，动作循环不完整。  
因此加入 DashSave：玩家在允许窗口冲刺后，可以保留连招进度和卡牌上下文。

这个改动让动作系统和卡组系统真正连接起来：  
冲刺不只是保命动作，也可以成为维持战斗句子的标点。

### 6.4 核心卖点

1. **卡牌构筑不打断动作心流**  
   战前思考卡组，战中只管打。卡牌在后台放大战斗结果。

2. **固定顺序卡组带来可预测爆发**  
   玩家不是被随机手牌支配，而是能规划这一轮怎么起手、承接和终结。

3. **主相牌让世界观进入战斗系统**  
   月、影、梦、巢、骨、镰等力量不是背景词，而是改写攻击性质的机制牌。

4. **终结技把构筑和动作一起兑现**  
   玩家既能通过卡组计划爆发，也需要用输入完成终结动作。

5. **从试玩反馈中完成过方向收束**  
   项目已经从复杂 2D 背包转向更聚焦的 1D 战斗卡组，证明团队能够根据玩家体验做取舍。

### 6.5 商业模式

建议采用 PC / Steam 买断制 + Demo + 抢先体验 + 长线更新。

阶段规划：

| 阶段 | 内容目标 | 商业目标 |
| --- | --- | --- |
| Demo / Steam 新品节 | 1 把核心武器、基础卡组、少量连携和一个完整小循环 | 验证愿望单和核心机制理解 |
| EA | 2-3 把武器、完整单次通关流程、基础主相牌和终结技体系 | 收集卡组生态和平衡反馈 |
| 1.0 | 完整剧情线、更多武器、主相牌、隐藏地图、结局和长线挑战 | 形成完整单机动作肉鸽产品 |

定价建议：

- EA：美区 14.99 美元，国区 68 元左右。
- 1.0：美区 19.99 美元，国区 88 元左右。

## 7. 开发规划

### 7.1 当前基础

当前项目已经具备以下基础：

- 近战攻击、多段连击、冲刺和 DashSave。
- 命中反馈、HitStop、敌人前摇预警、韧性和霸体。
- 1D 战斗卡组、洗牌空窗、奖励入卡组。
- 武器系统、火绳枪框架、弹药 HUD 框架。
- 关卡循环、三选一奖励、传送门切关、跨关存档。
- 世界观、主相牌、卡牌特征、终结技示例和核心文档。

### 7.2 近期 Demo 目标

Demo 不追求一次性做完所有系统，而是优先证明核心体验：

```text
一把武器
一条 1D 卡组
三类普通牌
两类状态牌
两张连携牌
一张主相牌
一张终结技牌
一个完整房间循环
一次奖励入组
一次洗牌空窗
一次终结爆发
```

Demo 的验收标准：

- 玩家能在 10 分钟内理解“攻击推进卡组”。
- 玩家能在 20 分钟内主动调整卡组顺序。
- 玩家能说出“我这套是某种流派”。
- 玩家能体验至少一次可预测的连携爆发。
- 玩家能体验至少一次终结技兑现。

### 7.3 中期 EA 目标

EA 阶段扩展到：

- 2-3 把核心武器。
- 3 个主相方向。
- 多个状态和连携池。
- 2-3 个区域主题。
- 基础剧情推进和隐藏路线。
- 更完整的局外成长和卡组模板选择。

### 7.4 主要风险与应对

#### 风险 1：动作爽感与卡牌策略互抢注意力

应对：  
战中不手动出牌。卡组思考发生在战前和奖励阶段；战斗中只展示下一张和关键原因提示。普通卡攻击即触发，少数连携卡承担策略深度。

#### 风险 2：卡牌组合复杂度失控

应对：  
不做所有卡牌无限连携。连携默认只读取上一张 TraitContext；主相牌只解释少数适配材料；强控制、强爆炸和诅咒只来自稀有卡、特殊卡、终结技或特殊武器。

#### 风险 3：终结技制作成本高

应对：  
终结技优先绑定少数代表武器和少数高价值卡牌。每张终结技必须有默认效果，输入失败也不浪费；终结蒙太奇只在成功输入时播放。

#### 风险 4：世界观容易显得泛克苏鲁

应对：  
提案表达不只强调“维多利亚 + 克苏鲁”，而强调“名 / 相 / 响”的现实结构，以及主相牌如何把世界观力量转化成战斗机制。

#### 风险 5：内容产能压力

应对：  
优先做有限字段、有限特征、有限主相的组合系统。卡牌通过规则组合产生变化，而不是每张卡都从零制作独立效果。

## 8. 附录：可用于答辩的短表达

### 8.1 一句话版本

《星骸降临》是一款把卡牌变成连招语法的俯视角动作 Roguelite：玩家不手动出牌，而是通过攻击顺序、连招延续和终结输入，让卡牌改写每一次命中的性质。

### 8.2 三句话版本

玩家在战前决定卡组顺序，战中只需要专注攻击、冲刺和连招。  
每次攻击都会推进卡组，卡牌把命中改写成状态、击退、破甲、传播、主相变体和终结爆发。  
我们已经通过试玩把旧 2D 背包收束成 1D 线性卡组，让构筑深度服务动作爽感，而不是打断它。

### 8.3 和竞品的区别

Hades 的构筑更像后台祝福，Slay the Spire 的卡牌需要回合制出牌，Backpack Hero 的核心是整理背包。  
《星骸降临》把三者中最有价值的部分重新组合：动作游戏的即时爽感、卡牌游戏的可预测组合、背包构筑的作者感，但最终都落到实时连招里。

### 8.4 最应该展示的 30 秒内容

展示一套固定卡组如何被玩家打出来：

```text
伤口 -> 月相 -> 击退剑气 -> 怒锋入铠
```

让评委看到：

1. 玩家没有停下来出牌。
2. 卡牌顺序确实改变了攻击结果。
3. DashSave 让冲刺延续连招。
4. 终结技把这一轮卡组节奏兑现成爆发。
