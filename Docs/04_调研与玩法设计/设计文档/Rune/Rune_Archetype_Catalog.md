## 一、物理引擎流

**核心幻想：** 把战场变成台球桌  
**装备后行为转变：** 不再原地输出，开始主动把敌人打向墙壁和人群

---

### 人体炮弹

| 字段 | 内容 |
|------|------|
| **触发** | 重攻命中 |
| **效果** | 击退力 ×3，被弹飞的敌人碰撞到同伴时，同伴受到等量碰撞伤害并同样被弹开 |
| **感知** | 看到敌人飞出去撞翻另一个敌人，后者又飞走——台球效果立即成立 |
| **代价** | 重攻后摇更长，站桩环境收益低 |
| **FA 实现** | OnHeavyHit → Apply 3x KnockbackForce GE → Listen KnockbackCollision → Apply SameForce to Collided Target |

---

### 弹射链

| 字段 | 内容 |
|------|------|
| **触发** | 敌人撞到同伴（碰撞事件） |
| **效果** | 碰撞伤害在两个目标间弹射，每次弹射 +30%，最多弹射 3 次 |
| **感知** | 能量弧从 A 弹到 B 再弹到 C，每次数字更大，连锁反应一目了然 |
| **代价** | 开阔地带完全无效，必须制造碰撞 |
| **FA 实现** | OnKnockbackCollision(A, B) → Apply GE(damage × 1.3) to B → 若 B 与 C 相邻，再弹 → 计数器 ≤ 3 |

---

### 碎墙震爆

| 字段 | 内容 |
|------|------|
| **触发** | 敌人撞墙（墙壁碰撞） |
| **效果** | 撞击点产生 120° 扇形冲击波，范围 600cm 内所有敌人被额外击退 |
| **感知** | 撞墙瞬间墙面爆发扇形冲击波粒子，周围敌人同时被推开，连锁清场效果强烈 |
| **代价** | 需要专门在墙边战斗，移动到位有成本 |
| **FA 实现** | OnKnockbackHitWall(Location, Direction) → Overlap Sphere 600 → Apply Knockback GE to all enemies in arc |

---

### 重力磁场

| 字段 | 内容 |
|------|------|
| **触发** | 击退命中敌人落地瞬间 |
| **效果** | 着地点产生 2 秒磁场，缓慢将周围 400cm 内敌人拉向中心 |
| **感知** | 能看到敌人缓慢被吸向一点，聚拢后可以一击打飞全部——"布局"的快感 |
| **代价** | 需要等待 2 秒磁场聚拢过程，主动进攻期被动 |
| **FA 实现** | OnKnockbackLand(Location) → Spawn 磁场 Actor，Tick 施加 PullForce 到周围敌人 CharacterMovementComponent，2s 后 Destroy |

## 二、动能流

**核心幻想：** 移动是武器，停下来就是浪费  
**装备后行为转变：** 攻击前必须先移动/冲刺，站桩输出主动放弃

---

### 动能积蓄

| 字段 | 内容 |
|------|------|
| **触发** | 连续移动积累层数（静止 1.5s 清零） |
| **效果** | 最多 5 层，每层 +20% 攻击伤害；攻击时消耗所有层，同时产生范围冲击波 |
| **感知** | 角色周围能量环随层数增大变亮，消耗时爆发效果明显，可以清楚看到"蓄满"和"释放"两个阶段 |
| **代价** | 静止清零，攻击前必须保持移动，拉开了与普通站桩的差异 |
| **FA 实现** | Tick(移动中) → Increment Stack GE (HasDuration) → OnAnyHit → Consume all stacks → Apply Burst GE(damage ×(1 + 0.2×stacks)) + 范围冲击波 |

---

### 冲刺融合

| 字段 | 内容 |
|------|------|
| **触发** | 冲刺状态中攻击（替代普通轻攻） |
| **效果** | 触发专属冲刺斩：玩家瞬间向目标加速，冲入目标造成 150% 伤害，结束后保持速度 |
| **感知** | 专属攻击动作完全不同于普通连击，冲入时有明显速度线效果，打击感强烈 |
| **代价** | 消耗冲刺的 i 帧窗口，冲刺融合中无无敌帧 |
| **FA 实现** | 在 GA_MeleeAttack 检测 HasTag(Character.State.Dashing) → 分支选 Montage_SprintSlash → Apply 1.5x Damage GE；或在 GA_PlayerDash 内注入攻击监听 |

---

### 急停冲击

| 字段 | 内容 |
|------|------|
| **触发** | 速度 >800 的移动后急停（冲刺结束或方向骤转） |
| **效果** | 脚下产生 360° 冲击波，击退半径 400cm 内所有敌人 |
| **感知** | 停下瞬间脚下环形爆炸扩散，非常直观；玩家会主动用"刹车"来创造距离 |
| **代价** | 急停后有短暂位移惯性，容易撞进人群 |
| **FA 实现** | 监听 CharacterMovement 速度从 >800 变为 <100（同一帧或连续帧）→ Overlap Sphere 400 → Apply Knockback GE |

---

### 疾风割裂

| 字段 | 内容 |
|------|------|
| **触发** | 移动速度超过阈值（>600）时发动普通攻击 |
| **效果** | 攻击附带速度方向的风刃投射物，射程 600cm，独立命中 |
| **感知** | 高速移动时挥击可以看到风刃飞出击中远处敌人，攻击范围视觉上扩展了两三倍 |
| **代价** | 慢速或静止时没有风刃，必须持续保持高速才有附加效果 |
| **FA 实现** | OnLightAttackHit AND GetPlayerSpeed > 600 → Spawn ProjectileActor(WindBlade, Direction=PlayerVelocity, Range=600, Damage=基础 ×0.5) |

## 三、武道流

**核心幻想：** 精准是力量，ACT 高手的主场  
**装备后行为转变：** 从连打变为"等待时机再出手"，读怪取代无脑压制

---

### 节奏共鸣

| 字段 | 内容 |
|------|------|
| **触发** | 连续命中不断链 |
| **效果** | 每击额外伤害递增 +8%（最多 8 层），断链或被打断立即清零 |
| **感知** | 角色发光强度随层数增加，8 层满状态时金光灿烂，攻击特效强化，高低对比明显 |
| **代价** | 被打断/打空立即清零，奖励连续稳定输出，惩罚激进冒打 |
| **FA 实现** | OnHit → AddStack(ComboStack GE, HasDuration 2s) → 攻击 GE 读取 ComboStack 计数作为 Damage 乘数；OnMissed/OnDamageTaken → Remove GE |

---

### 精准决裂

| 字段 | 内容 |
|------|------|
| **触发** | 在敌人出招前摇 120ms 内命中（对应 ANS_PreAttackFlash 时间窗） |
| **效果** | 该击伤害 ×2.5，并造成额外硬直（让玩家主动读怪然后"抢先一步"） |
| **感知** | 命中瞬间有白色闪光 + 0.15s 慢动作，明确强调"你做对了"；普通命中无此效果，对比强烈 |
| **代价** | 必须读怪才能触发，保守等待时节奏比普通慢 |
| **FA 实现** | ANS_PreAttackFlash 播放期间，敌人保持 Tag `Enemy.State.PreAttack`；OnPlayerHit AND TargetHasTag(Enemy.State.PreAttack) → Apply 2.5x GE + 触发 GameplayCue.PrecisionStrike |

---

### 完美回响

| 字段 | 内容 |
|------|------|
| **触发** | 完美回避（i 帧内成功闪过攻击） |
| **效果** | 3 秒内下一击同时触发"影回击"：在目标上产生第二个独立伤害（相当于一次额外的普攻） |
| **感知** | 完美回避后有影子跟随玩家，命中时影子同步攻击，视觉上"我在攻击，影子也在攻击" |
| **代价** | 依赖完美回避触发，激进硬吃打法完全无效 |
| **FA 实现** | OnPerfectDodge → Apply EchoReady GE(HasDuration 3s)；OnNextHit AND HasTag(EchoReady) → Remove EchoReady → Apply Echo Damage GE + Spawn ShadowStrike GameplayCue |

---

### 武道印

| 字段 | 内容 |
|------|------|
| **触发** | 连招终结击（连击链最后一击命中） |
| **效果** | 积累"武道印"（最多 3 印）；3 印满时下一次终结击变为超级终结击：伤害 ×4，附带范围击退所有周围敌人 |
| **感知** | HUD 上有 3 个印记槽，满槽时发光；超级终结击有完全不同的特效，是本符文的"啊哈时刻" |
| **代价** | 积累需要完成多次完整连招，中途被打断不清零但积累慢 |
| **FA 实现** | OnComboFinisher → AddStack(MartialSeal GE, Infinite)；Stack ≥ 3 AND OnNextFinisher → Clear Stack → Apply 4x Damage GE + Knockback All Nearby |

---

## 四、炼金状态流

**核心幻想：** 先埋地雷，再踩地雷  
**装备后行为转变：** 两阶段思维——"种"和"收"的节奏取代无脑叠伤害

---

### 活引信

| 字段 | 内容 |
|------|------|
| **触发** | 任意命中施加引信状态（2 秒计时） |
| **效果** | 引信期间若敌人受到击退，在其位置引爆大范围爆炸（半径 400cm，伤害 80）；若 2 秒内未被击退，引信哑火 |
| **感知** | 敌人身上有明显燃烧引信视觉（Niagara 附着粒子）；被打飞瞬间引爆，冲击波明显；没有击退就白种，风险感真实 |
| **代价** | 需要引信期内打出击退，单纯 DoT 流无法触发 |
| **FA 实现** | OnHit → Apply FuseMarker GE(HasDuration 2s)；OnKnockbackApplied AND TargetHasTag(Buff.Status.FuseMarker) → Remove FuseMarker → Spawn ExplosionActor at TargetLocation |

---

### 流血猎手

| 字段 | 内容 |
|------|------|
| **触发** | 命中施加流血 |
| **效果** | 流血目标移动时，每秒额外失去最大 HP 的 3%；完全静止不扣血（玩家需要驱使敌人移动，或等待 AI 自然移动） |
| **感知** | 移动中的敌人有明显血液飞溅粒子，静止时粒子停止；玩家会观察 AI 移动节奏而非单纯打伤害 |
| **代价** | 种入阶段伤害低，适合中期再收割 |
| **FA 实现** | OnHit → Apply Bleed GE(Infinite)；Tick(Target 移动中，Speed >50) AND HasTag(Buff.Status.Bleeding) → Apply Damage GE(MaxHP × 0.03 Instant) |

---

### 死亡感染

| 字段 | 内容 |
|------|------|
| **触发** | 带有任意状态的敌人死亡 |
| **效果** | 死亡瞬间，所有状态传播给周围 400cm 内所有敌人，传播的状态计时重置为满 |
| **感知** | 死亡时状态扩散特效从死亡敌人向外辐射（橙红/紫色圆环）；与流血/引信等搭配后连锁场面壮观 |
| **代价** | 需要先种入状态，死亡时才触发；时序依赖 |
| **FA 实现** | OnEnemyDeath(Target) AND HasAnyStatusTag(Target) → 获取 Target 所有 Status GEs → Overlap Sphere 400 → Apply same GEs to all enemies |

---

### 诅咒倒计时

| 字段 | 内容 |
|------|------|
| **触发** | 命中施加诅咒标记 |
| **效果** | 诅咒期间（5 秒）对目标的所有伤害先记录为"待结算伤害"；5 秒到期，一次性集中结算所有待结算伤害的 60%（追加爆发） |
| **感知** | 敌人头顶有倒计时数字（5→0），到期时爆炸式数字喷发；玩家会主动"先种诅咒再疯狂输出" |
| **代价** | 诅咒期间追加伤害延迟结算，敌人若提前死亡则诅咒白费；必须活到引爆 |
| **FA 实现** | OnHit → Apply CurseMarker GE(HasDuration 5s)；每次 OnHit AND HasCurse → 记录 Damage 到 CurseAccumulator；OnCurseExpired → Apply AccumulatedDamage × 0.6 as Burst |

## 五、转化符文（跨流派）

> 每个转化符文必须让两个流派的玩家都"值得装"，且产生两个流派单独无法实现的玩法。

---

### 横冲直撞

| 连接 | 动能 × 物理引擎 |
|------|------|
| **效果** | 玩家当前移速越高，击退力越大（线性缩放，600速 = 1x，1200速 = 2x） |
| **新玩法** | 冲刺即是发动攻击的最优时机——冲刺进入攻击的同时带走最大击退力，引发连锁碰撞 |
| **感知** | 高速攻击时敌人飞出距离明显更远，数值反馈和视觉效果都强调"速度" |

---

### 引信爆裂

| 连接 | 炼金状态 × 物理引擎 |
|------|------|
| **效果** | 对有引信状态的敌人造成击退时，引爆伤害额外 +50%，并对范围内敌人传播引信状态 |
| **新玩法** | "种引信 → 打飞 → 落点感染同伴 → 连环引爆"形成 3 步清场循环 |
| **感知** | 连环引爆视觉冲击强，可以清楚追踪每次传播和爆炸 |

---

### 冲刺播种

| 连接 | 动能 × 炼金状态 |
|------|------|
| **效果** | 冲刺穿过或经过敌人身边 150cm 时，自动施加流血状态 |
| **新玩法** | 冲刺不再只是位移工具，变成状态播种手段；动能流的"快速移动"直接转化为炼金流的"状态覆盖" |
| **感知** | 冲刺时路过的敌人依次出现血液粒子，视觉上能跟踪播种路径 |

---

### 节奏引爆

| 连接 | 武道 × 炼金状态 |
|------|------|
| **效果** | 节奏积累满层（8 层）时，下次攻击引爆范围 500cm 内所有已有状态的敌人 |
| **新玩法** | 先用小状态覆盖全场，再靠精准连击把节奏打满，最后一击清场 |
| **感知** | 节奏满层发光 → 范围引爆时所有状态同时爆炸，场面壮观，策略感最高 |

---

### 精准击飞

| 连接 | 武道 × 物理引擎 |
|------|------|
| **效果** | 精准时机出击（出招前窗口命中）时，击退力 ×3，且保证击退方向朝向最近的墙壁 |
| **新玩法** | 读怪成功的奖励直接变成"精确控制物理引擎"，一次精准出手 = 碰撞必然发生 |
| **感知** | 精准命中时敌人直接飞向墙壁（方向自动锁定），配合 ANS 闪光，因果链一目了然 |

---

## 六、流派档案速查

| 流派 | 代表符文 | 核心操作变化 | 最强协同 |
|------|----------|-------------|----------|
| 物理引擎 | 人体炮弹、碎墙震爆 | 关注敌人位置和墙距，主动"打台球" | 弹射链（多敌放大） |
| 动能 | 动能积蓄、急停冲击 | 永远在移动，冲刺即攻击 | 冲刺融合（冲刺武器化） |
| 武道 | 节奏共鸣、精准决裂 | 读怪等待，精准出手，不急 | 武道印（终结击强化） |
| 炼金状态 | 活引信、死亡感染 | "种"和"收"两段节奏，管理状态时序 | 诅咒倒计时（爆发收割） |
