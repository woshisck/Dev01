# 局外成长开发方案

## 组定位

本方案负责教程完成后的局外成长最小闭环。第一版目标是让玩家知道“下一局可以追什么”，不是一次做完整天赋树。

## 负责范围

| 模块 | 说明 |
| --- | --- |
| 局外资源 | 从资源房、Boss、挑战房带回 |
| 解锁展示 | 结算页展示资源和下一步目标 |
| 后续扩展 | 武器制作、玩家道具、据点设施 |

## 一期实现

| 功能 | 做法 |
| --- | --- |
| 基础卡组模板 | 先做 3 个预设模板 |
| 局外资源 | 先用中性资源名，避免过早绑定美术设定 |
| 第二局入口 | 完成教程后开放选择界面 |

## 二期扩展

| 功能 | 做法 |
| --- | --- |
| 武器制作 | 解锁新武器，改变初始卡组和奖励倾向 |
| 据点设施 | 铁匠、档案馆、祭坛、商人 |

## 验收标准

- 教程结束后能获得局外资源。
- 局外资源能被保存和读取。
