# 游戏开局教程方案

## 目标

第一局看起来像正式游戏，但内容被强控，用最少信息让玩家理解核心循环：

```text
拾取武器 -> 看到卡组 -> 攻击消耗卡 -> 洗牌回填 -> 获得奖励 -> 奖励入卡组 -> 触发连携 -> 触发终结技 -> 第一次路线选择
```

本教程优先解决信息过载。第一局不讲商店、祭坛、完整局外成长、完整卡池和复杂房间 Buff。

## 教程房间顺序

| 阶段 | 房间 | 教学点 | 验收 |
| --- | --- | --- | --- |
| T0 | 局外默认选择 | 默认基础卡组 + 默认终结技 | 新玩家不用理解全部系统也能开始 |
| T1 | 武器房 | 拾取武器后显示 1D 卡组 | 玩家知道攻击与卡组相关 |
| T2 | 基础卡房 | 攻击会按顺序消耗卡牌 | 玩家能看到当前卡被消耗 |
| T3 | 洗牌房 | 卡组打空后进入洗牌，回填后继续循环 | 玩家理解卡组不是弹药上限，而是一轮循环 |
| T4 | 奖励房 | 清场奖励进入卡组 | 新卡在下一房立刻可见和可触发 |
| T5 | 连携房 | 第一张连携卡自动摆位并触发 | 玩家理解卡牌顺序会改变效果 |
| T6 | 终结技验证房 | 用默认终结技完成一次爆发 | 玩家理解终结技是本局目标 |
| T7 | 路线选择 | 战斗奖励房 vs 局外资源房 | 玩家第一次理解“当前变强”和“投资下一局” |

## 信息释放规则

| 教学点 | 允许出现 | 不允许出现 |
| --- | --- | --- |
| 武器房 | 武器拾取、卡组条 | 奖励、连携、终结技、背包 |
| 基础卡房 | 当前卡高亮、消耗动画 | 卡池说明、稀有度 |
| 洗牌房 | 洗牌进度、回填反馈 | 热度系统、复杂 Buff |
| 奖励房 | 固定奖励或二选一 | 完整三选一复杂卡池 |
| 连携房 | 一条连携原因提示 | 正逆位全规则 |
| 终结技房 | 终结技条件与强反馈 | 多终结技切换 |
| 路线选择 | 奖励倾向预览 | 完整大地图 |

## 路线选择教学

第一局末尾加入一次二选一传送门。

| 传送门 | 奖励倾向 | 教学目的 |
| --- | --- | --- |
| 关卡 A：战斗奖励房 | 普通卡、连携卡、升级卡 | 选择后本局立刻变强 |
| 关卡 B：资源房 | 武器材料、研究碎片、遗物图纸 | 选择后推进局外成长 |

传送门预览必须明确显示奖励倾向，避免玩家盲选。

## 验收标准

- 新存档玩家能在 8-12 分钟内完成第一局。
- 玩家至少完成一次卡牌消耗、一次洗牌、一次奖励入卡组、一次连携、一次终结技。
- 教程不依赖长篇说明，主要通过 HUD 高亮和短提示完成。
- 完成教程后，玩家知道下一局有局外选择和资源目标。
