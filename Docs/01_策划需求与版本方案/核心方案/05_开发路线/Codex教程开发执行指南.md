# Codex 教程开发执行指南

## 目标

本文件用于指导后续如何使用 Codex 按“游戏开局教程方案”推进开发。

当前优先级是：

```text
先跑通游戏开局教程 -> 再做局外成长最小闭环 -> 最后扩展武器制作、玩家道具和据点设施
```

不要一次性让 Codex 实现完整局外成长。每次只推进一个可验收的小阶段。

## 必读文档

每次开始教程相关开发前，先让 Codex 阅读：

| 文档 | 用途 |
| --- | --- |
| `04_系统需求/关卡/游戏开局教程方案.md` | 教程体验目标和房间顺序 |
| `01_开发方案/关卡/游戏开局教程开发方案.md` | 主城首局教程的实现范围 |
| `05_开发路线/教程优先开发路线.md` | 优先级和时间节点 |
| `04_系统需求/关卡/房间奖励分流与局外资源.md` | 战力房 / 资源房奖励分流 |

## 使用 Codex 的开发方式

每个任务分两步：

1. 先审计：读文档、查代码、列出已具备和缺口。
2. 再实现：只做当前阶段范围内的最小闭环，并编译或给出验收方式。

不要跳过审计直接写代码。

## 第一条推荐任务

```text
请根据游戏开局教程方案，先做“第 1 周：教程脚本与数据冻结”的代码路径审计和配置清单。

要求：
1. 阅读以下文档：
Docs/01_策划需求与版本方案/核心方案/04_系统需求/关卡/游戏开局教程方案.md
Docs/01_策划需求与版本方案/核心方案/01_开发方案/关卡/游戏开局教程开发方案.md
Docs/01_策划需求与版本方案/核心方案/05_开发路线/教程优先开发路线.md

2. 检查现有代码和资产路径：
TutorialManager、LENode_ShowTutorial、YogGameMode、RoomDataAsset、RewardPickup、LootSelectionWidget、CombatDeckComponent。

3. 输出一份“教程第一局配置清单”，包括：
- 需要几个房间
- 每个房间要配置什么 RoomData
- 每个房间给什么敌人和奖励
- 哪些可以复用现有系统
- 哪些需要新增代码
- 第一阶段最小实现任务列表

4. 暂时不要写代码。
```

## P0 实现任务模板

### 任务：主城首局教程入口

```text
目标：实现新存档进入主城后触发首局教程。

范围：
- TutorialManager
- YogGameMode
- RoomDataAsset / CampaignData 入口
- 存档中的教程完成状态

必须阅读：
- 游戏开局教程方案.md
- 游戏开局教程开发方案.md
- 教程优先开发路线.md

暂时不做：
- 局外成长 UI
- 武器制作
- 玩家道具
- 完整随机房间池

验收：
- 新存档进入主城后触发首次武器 / 卡组提示。
- 教程完成后不会重复触发首局提示。
- 编译通过。
```

### 任务：奖励房

```text
目标：实现游戏开局教程的奖励房，玩家清场后获得固定普通卡，并加入 DeckList。

范围：
- RewardPickup
- LootSelectionWidget
- YogGameMode 奖励生成
- CombatDeckComponent 奖励入组路径

必须阅读：
- 游戏开局教程方案.md
- 奖励接入卡组.md
- 房间奖励分流与局外资源.md

暂时不做：
- 完整随机奖励池
- 局外资源
- 商店 / 祭坛
- UI 美术重做

验收：
- 清场后出现固定奖励。
- 选择后加入 DeckList。
- 下一房间能看到新卡。
- 编译通过。
```

### 任务：局外成长最小闭环

```text

范围：
- SaveGame / RunState / MetaProgression 状态
- 起始卡组模板
- 教程结算入口

必须阅读：
- 局外开局选择.md
- 局外成长开发方案.md
- 教程优先开发路线.md

暂时不做：
- 武器制作
- 玩家道具池
- 据点设施
- 复杂天赋树

验收：
- 教程结束后获得局外资源。
- 选择结果能进入新 Run。
- 编译通过。
```

## 每次实现后的文档同步

每完成一个阶段，需要让 Codex 更新对应文档的“当前已实现 / 需要配置 / 验收步骤”。

不要重写整篇文档，只补充实际落地状态。

推荐提示：

```text
刚才实现后，请同步更新对应开发方案文档。
只补“当前已实现 / 需要配置 / 验收步骤”，不要重写整篇。
```

## 开发边界

第一轮不做：

- 完整天赋树。
- 大量永久攻击力 / 生命上限成长。
- 十几个货币。
- 完整据点场景。
- 全量道具池。
- 完整随机地图。

第一轮只追求：

```text
新存档 -> 主城 -> 首局教程提示 -> 学会卡组循环 -> 获得局外资源 -> 回到主城选择 -> 第二局自由进入
```
