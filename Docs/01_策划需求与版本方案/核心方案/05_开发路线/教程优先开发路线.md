# 教程优先开发路线

## 总原则

当前优先级是先解决玩家首局信息过载。所有局外成长、武器制作、道具池和设施系统都应服务教程完成后的第二局回流。

## 优先级

| 优先级 | 目标 | 原因 |
| --- | --- | --- |
| P0 | 游戏开局教程可玩 | 先让玩家理解核心循环 |
| P0 | 教程 UI 与反馈 | 教程靠 HUD 高亮和短提示，不靠长文 |
| P1 | 局外成长最小闭环 | 教程结束后给玩家下一局目标 |
| P1 | 路线奖励分流 | 教会玩家本局战力和局外资源的取舍 |
| P2 | 武器制作、道具、据点设施 | 长期内容，等首局和回流稳定后扩展 |

## 时间节点

| 时间 | 里程碑 | 主要任务 | 验收 |
| --- | --- | --- | --- |
| 第 1 周 | 教程脚本与数据冻结 | 固定房间、敌人、奖励、文案、HUD 高亮点 | 有一份可配置清单 |
| 第 2 周 | 教程第一局可玩纵切 | 接入强控教程 Run 和固定奖励池 | 新存档能完整完成第一局 |
| 第 5 周 | 第二局自由度接入 | 奖励池过滤、资源房、战力房、传送门预览 | 玩家能选择战力或资源路线 |
| 第 6 周 | 武器与道具第一批 | 铁匠入口、2 把武器、6-8 个简单道具 | 解锁内容能带入下一局 |

## 本轮不做

- 完整天赋树。
- 大量永久数值成长。
- 十几个货币。
- 完整据点场景。
- 全量道具池。

## 验收主线

```text
新存档 -> 游戏开局教程 -> 学会卡组循环 -> 获得局外资源 -> 回到局外选择 -> 第二局自由进入
```
