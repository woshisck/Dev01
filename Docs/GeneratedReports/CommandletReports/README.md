# Commandlet Reports

This folder mirrors root-level Markdown reports generated by editor commandlets.

- Local working copies are still written to `Saved/*.md`.
- Shared copies are written here so they can be committed and pulled by other team members.
- Do not store logs, autosaves, screenshots, CSV snapshots, or editor cache files here.
- Re-run the relevant commandlet to refresh a report instead of editing generated report content by hand.
