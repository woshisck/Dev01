# 局外成长开发方案

## 组定位

本方案负责教程完成后的局外成长最小闭环。第一版目标是让玩家知道“下一局可以追什么”，不是一次做完整天赋树。

## 负责范围

| 模块 | 说明 |
| --- | --- |
| 局外状态 | 已解锁模板、终结技、武器、道具、设施、资源 |
| 开局选择 | 基础卡组模板 + 终结技卡 |
| 局外资源 | 从资源房、Boss、挑战房带回 |
| 解锁展示 | 结算页展示资源和下一步目标 |
| 后续扩展 | 武器制作、玩家道具、据点设施 |

## 一期实现

| 功能 | 做法 |
| --- | --- |
| 基础卡组模板 | 先做 3 个预设模板 |
| 终结技选择 | 每局选择 1 张终结技，独立槽位 |
| 局外资源 | 先用中性资源名，避免过早绑定美术设定 |
| 第二局入口 | 完成教程后开放选择界面 |
| 奖励池过滤 | 根据已解锁内容和终结技路线控制掉落 |

## 二期扩展

| 功能 | 做法 |
| --- | --- |
| 武器制作 | 解锁新武器，改变初始卡组和奖励倾向 |
| 武器升级 | 改变洗牌时间、初始卡、终结技适配 |
| 玩家道具 / 遗物 | 改写规则，例如洗牌、奖励、终结技、商店 |
| 据点设施 | 铁匠、档案馆、祭坛、商人 |
| 研究任务 | 通过行为解锁卡池、道具或终结技变体 |

## 验收标准

- 教程结束后能获得局外资源。
- 第二局开始前能选择基础模板和终结技。
- 局外资源能被保存和读取。
- 选择不同终结技后，局内奖励倾向可被调整。
