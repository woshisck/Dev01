# 局外开局选择

## 目标

局外选择只决定“本局从什么方向开始”，不让玩家在开局前处理完整构筑。

第一版建议：

```text
选择基础卡组模板 + 选择 1 张终结技卡
```

完成教程后再逐步开放武器制作、武器升级和玩家道具。

## 开局选择内容

| 类型 | 第一版做法 | 后续扩展 |
| --- | --- | --- |
| 基础卡组 | 使用预设模板，不自由拼单卡 | 解锁更多模板或允许少量替换 |
| 终结技卡 | 每局携带 1 张，独立槽位 | 解锁变体和支持包 |
| 武器 | 初期跟随模板 | 后续由武器制作和升级决定 |
| 玩家道具 | 教程阶段不开放或默认携带 | 后续可携带 1 个已解锁道具 |
| 连携卡 | 不在局外选择 | 主要在局内奖励中获得 |

## 基础模板示例

| 模板 | 初始卡组 | 定位 |
| --- | --- | --- |
| 稳定剑士 | 基础斩击 x2、重击 x1、护盾 x1 | 新玩家默认，容错高 |
| 燃烧剑士 | 基础斩击 x2、燃烧 x1、轻击强化 x1 | 持续伤害，反馈明显 |
| 月光剑士 | 基础斩击 x2、月光斩 x1、减伤 x1 | 展示项目特色，数值保守 |

## 终结技示例

| 终结技 | 触发方向 | 用途 |
| --- | --- | --- |
| 断月斩 | 本轮消耗 3 张卡后，连招终结击触发 | 单体爆发，适合教程 |
| 银焰处刑 | 本轮触发燃烧或火焰连携后触发 | 引导燃烧路线 |
| 圣盾反击 | 本轮获得护盾或减伤后触发 | 引导防御路线 |

## 验收标准

- 玩家开始正式局前必须能看到基础模板和终结技目标。
- 默认推荐项能让新玩家不思考也能开始。
- 选择界面不展示完整卡池长说明。
- 终结技不进入普通 DeckList，避免卡组阅读压力过高。
