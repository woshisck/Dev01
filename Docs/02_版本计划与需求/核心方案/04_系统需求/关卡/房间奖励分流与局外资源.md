# 房间奖励分流与局外资源

## 目标

房间奖励分成两条线：

1. 本局战力奖励：让当前这局马上变强。
2. 局外成长资源：带回据点，推动下一局开局选择和长期解锁。

## 奖励类型

| 收益线 | 例子 | 玩家心智 | 适合房间 |
| --- | --- | --- | --- |
| 本局战力奖励 | 普通卡、连携卡、删卡、升级卡、回血、金币、局内道具 | 让我这一局更容易打通 | 战斗房、精英房、商店房、锻造房 |
| 局外成长资源 | 武器材料、研究碎片、遗物图纸、设施材料、终结技残页 | 这次不一定马上变强，但下一局有更多选择 | 资源房、事件房、挑战房、Boss 房、祭坛房 |

## 传送门预览要求

传送门需要提前告诉玩家奖励倾向。

| 预览字段 | 示例 |
| --- | --- |
| 房间类型 | 战斗奖励房 / 资源房 / 商店房 / 祭坛房 |
| 主要收益 | 本局卡牌奖励 / 武器材料 / 遗物图纸 |
| 风险说明 | 普通战斗 / 精英压力 / 限时挑战 |
| 是否影响局外 | 是 / 否 |

## 第一版建议

- 教程第一局只做一次 A/B 路线选择。
- 第二局开始再开放更多资源房和服务房。
- 小怪掉落只放金币或小资源，不打断战斗节奏。
- 局外资源不应替代清场主奖励，而是成为路线选择的一部分。

## 验收标准

- 玩家能从传送门预览区分“当前变强”和“投资下一局”。
- 资源房完成后，结算页能显示获得的局外资源。
- 本局战力房奖励进入 DeckList 后，下一房能立即验证。
